# Классическое Монте-Карло для Ze-модели

Моделирование H_Ze на 3D решётке (2 пространства + 1 время) методом Метрополиса.

## Параметры
- `Lx, Ly` — пространственные размеры
- `Lt` — временной размер
- `J_t` — антиферромагнитная связь вдоль времени (антипараллелизм S=−T)
- `J_s` — ферромагнитная связь вдоль пространства (локальность)
- `h` — внешнее поле
- `T` — температура

## Запуск
```bash
pip install -r requirements.txt
python ze_mc.py              # полная симуляция
python ze_mc.py --quick      # быстрый тест
python ze_mc.py --scan-js    # сканирование по J_s
```

## Измеряемые величины
- `v = ⟨z⟩` — Ze-скорость (намагниченность)
- `v* = 1−ln 2` — теоретическая критическая точка
- `ξ_t, ξ_x` — корреляционные длины
- `⟨E⟩/N` — энергия на узел
