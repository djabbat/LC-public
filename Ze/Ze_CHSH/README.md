# Ze_CHSH — CHSH(энтропия) как фальсифицируемое предсказание Ze

**Родительский проект:** Ze → LC
**Статус:** 🟢 Submitted — Quantum Information Processing (18 июн 2026)

## Что это

Теория Ze предсказывает: параметр CHSH линейно падает с ростом инжектируемой энтропии:

$$S(H) = 2\sqrt{2} - \frac{2}{\ln 2} \cdot H$$

Это предсказание можно проверить на существующей квантово-оптической установке за 24 часа.
