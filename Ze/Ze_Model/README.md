# Ze Model

A foundational model in which spacetime and quantum correlations emerge from global knowledge — an active predictive resource, not passive memory.

## Five Axioms

1. **Time** as a fundamental parameter (t ∈ ℝ)
2. **Discrete local dynamics** within the light cone
3. **Knowledge as code of the past** (full encoding)
4. **Prediction error** η(t) as a physical quantity
5. **Nonlocal knowledge** (global connectivity)

## Key Results

- Born rule postulated from norm conservation
- Tsirelson bound S = 2√2 from Hilbert projector structure
- Barrett's memory loophole avoided (nonlocal, not local memory)
- Yang's trichotomy: violates ODL, preserves MI, rejects classical ontology
- Falsifiable predictions: non-equilibrium Born rule deviations (γ/√N)
