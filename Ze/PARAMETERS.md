# Ze: Параметры Теории

В таблице перечислены все ключевые количественные и качественные параметры Ze Vector Theory. Для каждого указано происхождение, единицы измерения, статус (Теоретический / Эмпирический / Гипотетический) и текущее значение с обоснованием.

| Символ | Название | Описание | Происхождение (Источник) | Единицы / Тип | Статус | Текущее Значение / Диапазон | Обоснование и Примечания |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **d** | Размерность гильбертова пространства | Количество ортогональных базисных состояний, доступных Ze-агенту. | Постулат 1. Задаётся моделью системы. | Безразмерное целое (d ≥ 2). | Теоретический (модельный) | Зависит от применения. Для бинарной модели ЭЭГ: **d = 2**. | Выбор d=2 для ЭЭГ — феноменологическое упрощение. В полном квантово-физическом формализме d может быть произвольным конечным. |
| **ρ_Z** | Матрица плотности (состояние убеждений) | Описывает знания/ожидания Ze о своей среде. | Постулат 1. | Эрмитова матрица размера d×d, ρ_Z ≥ 0, Tr(ρ_Z)=1. | Теоретический | Нет фиксированного значения. Инициализируется, затем эволюционирует. | В классическом пределе (например, для EEG-анализа) сводится к вектору вероятностей **P_Z = (p₀, p₁)**. |
| **{M_i}** | Набор POVM-операторов | Описывает взаимодействие Ze со средой, каждому исходу i сопоставлен тип события (T/S). | Постулат 1. | Набор операторов в H, удовлетворяющих Σ_i M_i† M_i = I. | Теоретический | Задаётся моделью измерения. Для проективных измерений в базисе { \|0⟩, \|1⟩ }: M₀=\|0⟩⟨0\|, M₁=\|1⟩⟨1\|. | |
| **τ_Z** | Счётчик собственного времени | Дискретный ресурс существования Ze. Уменьшается при T-событиях. | Постулаты 1 и 2. | Целое неотрицательное число (τ_Z ∈ ℕ). | Теоретический | Задаётся при инициализации (например, τ₀=200 в симуляциях). | Прямое измерение в физических/биологических системах отсутствует. Требует определения прокси. |
| **θ_Z** | Порог предсказания | Внутренний параметр, определяющий, какая частота T-событий побуждает Ze делать предсказание. | Постулаты 1 и 3. | Безразмерное, θ_Z ≥ 0. Интерпретируется в битах при сравнении с log₂(d). | Теоретический / Гипотетический | В симуляциях: **θ_Z = 0.3**. В теореме 5.1: значение сравнивается с **log₂(d)**. | Выбор 0.3 эмпирически мотивирован. Критический параметр для применимости Теоремы 5.1. |
| **N** | Длина последовательности измерений | Общее количество интервалов/измерений в наблюдаемом временном ряду. | Операциональное определение v. | Целое число (N ≥ 2). | Операциональный | Зависит от эксперимента (например, 120 для 2-минутной ЭЭГ с эпохой 1с). | |
| **N_S** | Количество S-событий (Stretch) | Число интервалов, где исход был классифицирован как S. | Операциональное определение v. | Целое число (0 ≤ N_S ≤ N-1). | Операциональный (вычисляемый) | Вычисляется из данных. | Событие классифицируется как S, если исход совпал с наиболее вероятным прогнозом, основанным на текущем ρ_Z/P_Z. |
| **v** | Наблюдаемая скорость | Доля синхронных (S) интервалов в последовательности. | Каноническое определение (CORRECTIONS_2026-04-22). | Безразмерное, v ∈ [0, 1]. **v = N_S / (N - 1)**. | Операциональный (вычисляемый) | Вычисляется из данных. | **ВАЖНО:** Это единственное каноническое определение. Старые формулы (`v = N_T/(N_T+N_S)`) более не используются. |
| **v*_passive** | Оптимальная скорость для пассивного наблюдателя | Теоретический оптимум для агента, который только фильтрует, но не изменяет среду. | Аналитический вывод в рамках ZVT. | Безразмерное. **v*_passive = 1 - ln(2) ≈ 0.30685281944**. | Теоретический (аналитически выведен) | 0.3069 | Соответствует максимуму информационной энтропии последовательности при заданном среднем числе T-событий. **Статус верификации:** Требует независимой проверки математического вывода. |
| **v*_active** | Эмпирическая оценка оптимальной скорости для активного агента | Предполагаемый оптимум для агента, способного к нишестроительству. | Эмпирическая оценка на ограниченных данных. | Безразмерное. | Гипотетический (предварительная оценка) | **~0.456** (но см. Примечание) | Получена как weighted average из гетерогенных датасетов (Cuban, Dortmund, MPI-LEMON, N=196). **Из-за высокой гетерогенности (I²=90.3%) это значение не может считаться надёжной универсальной оценкой.** Требует dataset-specific анализа. |
| **χ_Ze** | Индекс синхронизации Ze | Мера близости наблюдаемой v к оптимальной v*. | Определение в теории: χ_Ze = 1 - \|v - v*\| / max(v*, 1-v*). | Безразмерное, χ_Ze ∈ [0, 1]. | Теоретический конструкт / Гипотетический биомаркер | Вычисляется из v и v*. | **Ключевое разграничение:** В рамках ZVT — теоретический конструкт. При применении к биоданным — гипотетический, **невалидированный** биомаркер. Все клинические пороги отозваны. |
| **θ_Q** | Операциональный порог (в битах) для Теоремы 5.1 | Порог, с которым сравнивается log₂(d) в условии теоремы. | Адаптация θ_Z для сравнения. Предполагается θ_Q = θ_Z (в битах). | Безразмерное (биты). | Гипотетический (для приложений) | В EEG-контексте предполагается **θ_Q = 1.5**. | Если θ_Q ≥ log₂(d) (для d=2, θ_Q ≥ 1), Теорема 5.1 **не гарантирует** оптимальности правила Борна. Это ослабляет теоретическую базу для использования χ_Ze, основанного на Born-rule, в бинарных моделях. |
| **γ_S** | Коэффициент связи в MCOA | Влияние других счётчиков MCOA на состояние D_S Ze-счётчика. | Общая формула MCOA: D_S(n, t) = D_S,0 + α_S·(n/n_S*) + β_S·(t/τ_S) + **γ_S·I(others)**. | Безразмерное. | Теоретический (в рамках MCOA) | По умолчанию: **γ_S = 0** (гипотеза независимости). | Согласно CORRECTIONS_2026-04-22, ненулевые γ_i возникают только если post-hoc статистический анализ отвергает гипотезу независимости на реальных данных. Нельзя ссылаться на «MCOA Test 2» как на априорный источник. |
