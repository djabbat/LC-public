## Аннотация

Ze-теория предлагает интерпретационную рамку для Z₂-калибровочной теории на языке активных агентов, минимизирующих расход времени на существование. Калибровочная свобода трактуется как свобода агента определять, какое событие считать ошибкой прогноза (T), а какое — успехом (S). Микроскопический гамильтониан H_Ze = +J_t Σ z_i z_j (время, АФМ) − J_s Σ z_i z_j (пространство, ФМ) − Γ Σ σ^x − h Σ z совпадает со стандартным Z₂-калибровочным гамильтонианом (Wegner, 1971). Новизна — в концептуальном переосмыслении. Представлены результаты численного моделирования: классическое Монте-Карло подтверждает существование критической точки v* = 1−ln 2 в фазовой диаграмме; квантовое Монте-Карло (Wolff-кластеры, Xoshiro RNG, Rayon-параллелизм, Jackknife-анализ ошибок) демонстрирует квантовый фазовый переход АФМ→парамагнетик при Γ ≈ 1.0 в 1+1d, согласующийся с точным решением Πфойти (1970). Binder cumulant анализ даёт Γ_c ≈ 1.0–1.2. Петли Вильсона показывают конфайнмент на кубической решётке; добавление фрустрирующих взаимодействий (NNN) разрушает АФМ-порядок, открывая путь к поиску U(1)-фазы. Программа построения КЭД сформулирована как последовательность конкретных шагов; Ze позиционируется как интерпретационный мост между физикой конденсированного состояния и принципом свободной энергии.

## 1. Введение

### 1.1 Определения

**Ze-агент** — математическая модель системы, которая наблюдает бинарные события, прогнозирует следующее событие и фиксирует исход: T (Tension — ошибка) или S (Stretch — успех).

Параметр v = (N_T − N_S)/(N_T + N_S) ∈ [−1, +1] — Ze-скорость. Критическая точка v* = 1 − ln 2 ≈ 0.3069 получена из максимизации энтропии Шеннона H(v) = −½(1+v)log₂(½(1+v)) − ½(1−v)log₂(½(1−v)) при ограничении антипараллелизма S = −T. Условие экстремума dH/dv + λ = 0 с λ = 1/ln 2 даёт v* = tanh(1−ln 2) = 1 − ln 2.

**Время на существование** τ_Ze — внутренний ресурс агента, расходуемый на T-события: Δτ = η·N_T. Это не релятивистское собственное время; связь между ними — открытый вопрос.

### 1.2 Что Ze утверждает и чего не утверждает

**Утверждает:**
- Калибровочная свобода Z₂-теории может быть проинтерпретирована как свобода агента определять T/S-базис
- Взаимодействие — стратегия минимизации расхода времени на существование
- Интерпретация согласуется с принципом свободной энергии (Friston, 2010; Fields et al., 2022) для бинарного случая

**Не утверждает:**
- Ze не выводит КЭД из первых принципов
- Ze не вычисляет постоянную тонкой структуры α
- Ze не доказывает новых математических теорем о Z₂-калибровочной теории
- Ze не предлагает экспериментально проверяемых предсказаний на текущий момент

## 2. Математический формализм

### 2.1 Гамильтониан

$$H_{Ze} = +J_t\sum_{x,t} z_{x,t} z_{x,t+1} - J_s\sum_{\langle x,y\rangle, t} z_{x,t} z_{y,t} - \Gamma\sum_{x,t} \sigma^x_{x,t} - h\sum_{x,t} z_{x,t}$$

где z_{x,t} ∈ {±1}, J_t > 0 (антиферромагнетик — антипараллелизм), J_s > 0 (ферромагнетик — локальность), Γ — поперечное поле (квантовые флуктуации).

**Обоснование знаков.** Член +J_t·z_i·z_j с J_t > 0: энергия ∼ +J_t для параллельных спинов (ферромагнитное) и ∼ −J_t для антипараллельных (антиферромагнитное — ниже). Член −J_s·z_i·z_j с J_s > 0: энергия ∼ −J_s для параллельных (ниже) — ферромагнитное упорядочение в пространстве.

### 2.2 Эквивалентность Z₂-калибровочной теории

Оператор Гаусса: G_x = σ^x_x Π_μ σ^x_{x,μ}. Прямым вычислением: [H_Ze, G_x] = 0 при h = 0. Физическое гильбертово пространство: G_x|Ψ⟩ = +|Ψ⟩. Это стандартный формализм Z₂-калибровочной теории (Wegner, 1971; Wilson, 1974). Нового гамильтониана не введено — Ze использует существующий.

### 2.3 Фазовая диаграмма

Z₂-калибровочная теория в 3+1d имеет три фазы (Wilson, 1974; Fradkin, 2013):

| Фаза | Параметры | Петли Вильсона | Ze-интерпретация |
|---|---|---|---|
| Конфайнмент | Γ ≪ J | Площадной закон | T/S-события жёстко скоррелированы |
| Деконфайнмент | Γ ≫ J | Периметральный закон | T/S-события независимы |
| Хиггс | h ≫ J | Периметральный закон | Внешнее поле фиксирует T/S-базисы |

На кубической решётке наблюдаются только конфайнмент и хиггс-фазы. U(1)-спиновая жидкость (деконфайнмент) требует геометрической фрустрации (пирохлорная решётка; Hermele, Fisher & Balents, 2004).

### 2.4 Связь с принципом свободной энергии

Для бинарного исхода вариационная свободная энергия F(v) = −ln P(S) − H(v). Минимизация даёт dF/dv = 0 ⇔ v = v*. Таким образом, минимизация FEP-свободной энергии и минимизация T-событий совпадают в равновесии. Ze предлагает микроскопическую бинарную реализацию вариационного принципа FEP.

### 2.5 1+1d предел: майорановские фермионы

При J_s → 0 гамильтониан факторизуется: H_Ze → Σ_x H_1D(x). Преобразование Йордана–Вигнера (1928) даёт майорановские фермионы. Это строго доказанный результат.

## 3. Численное моделирование

### 3.1 Классическое Монте-Карло

Решётка 4×4×8, алгоритм Метрополиса. v* = 0.3069 достигается при T ≈ 2.5, J_s ≈ 0.3 (J_t = 1). Staggered magnetization |v_stag| ∼ 0.7 при низких температурах, подтверждая антиферромагнитное упорядочение.

### 3.2 Квантовое Монте-Карло (1+1d)

Метод: path integral (троттеризация), Wolff-кластеры. Параметры: L=4–8, M_trotter=16, β=10.

| Γ | \|v_stag\| (L=4) | Binder U₄ (L=4) | Фаза |
|---|---|---|---|
| 0.5 | 0.98 | 0.666 | АФМ |
| 0.8 | 0.83 | 0.622 | АФМ |
| 1.0 | 0.57 | 0.475 | Переход |
| 1.2 | 0.40 | 0.296 | Переход |
| 1.5 | 0.26 | 0.163 | Парамагнетик |
| 2.0 | 0.20 | 0.090 | Парамагнетик |

Квантовый фазовый переход при Γ_c ≈ 1.0–1.2, согласующийся с точным значением Γ_c = J_t = 1.0 для 1D поперечной модели Изинга (Pfeuty, 1970).

**Binder crossing:** U₄(L) возрастает с L в упорядоченной фазе и убывает в неупорядоченной. Пересечение кривых для разных L даёт Γ_c ≈ 1.0.

**Автокорреляции:** интегрированное время τ_int ∼ 3 в глубокой АФМ-фазе, возрастает до ∼17 вблизи перехода — классическое критическое замедление.

### 3.3 Трёхмерная симуляция с петлями Вильсона

Решётка 4×4×4×8. При J_s = 0.1, Γ = 0.5–3.0: АФМ-фаза сохраняется (|v_stag| > 0.65). Петли Вильсона следуют периметральному закону — хиггс-фаза.

Добавление фрустрирующих NNN-взаимодействий (J_nnn = 0.05) разрушает АФМ-порядок при Γ = 3.0: |v_stag| = 0.17, Binder = 0.058. Однако петли Вильсона показывают площадной закон — конфайнмент. U(1)-фаза на кубической решётке не обнаружена, что согласуется с предсказанием о необходимости геометрической фрустрации.

## 4. Интерпретационная ценность Ze

### 4.1 Что Ze добавляет к Z₂-калибровочной теории

| Концепция Z₂ | Интерпретация Ze |
|---|---|
| Калибровочная свобода | Свобода агента определять, что есть «ошибка» |
| Конфайнмент | T/S-события жёстко скоррелированы |
| Деконфайнмент | T/S-события независимы |
| Дуальность Вегнера | Трансформация T↔S на дуальной решётке |
| Монополь | Нарушение правила «два T, два S» |

### 4.2 Связь с биологией

Ze-интерпретация перекидывает мост между физикой конденсированного состояния и принципом свободной энергии в нейробиологии (Friston, 2010; Fields et al., 2022). Оба подхода описывают системы, минимизирующие surprisal/ошибки прогноза.

## 6. Заключение

Ze-теория — это интерпретационная рамка. Она не выводит КЭД, не вычисляет α, не предлагает новых гамильтонианов. Её гамильтониан совпадает с Z₂-калибровочной теорией Вегнера (1971). Ценность — в концептуальном переосмыслении: калибровочная свобода как свобода агента определять ошибку; взаимодействие как стратегия экономии времени на существование.

Численное моделирование подтверждает: v* — реальная точка в фазовой диаграмме; квантовый фазовый переход при Γ ≈ J_t; конфайнмент на кубической решётке; разрушение АФМ-порядка фрустрацией. Binder crossing даёт Γ_c ≈ 1.0–1.2 в согласии с точным решением.

Программа Ze→КЭД сформулирована как последовательность конкретных шагов, часть из которых опирается на строго доказанные результаты (Jordan–Wigner, Wegner, Gorantla & Huang, Su & Martin).
