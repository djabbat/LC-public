## Аннотация

Построен математический формализм, выводящий локальное калибровочное поле и его действие из Ze-теории — без процедуры gauging. Ключевые результаты: (1) Пара (T, S) образует ортонормированный базис одномерного вещественного векторного пространства V_x над каждым узлом x; антипараллелизм S=−T делает dim(V_x)=1. (2) Независимость прогнозов разных узлов делает базисы V_x и V_y несогласованными априори — для сравнения требуется связность (Z₂-калибровочное поле). (3) Действие калибровочного поля ∼F_μν² выведено из микроскопики: кривизна связности создаёт несогласованность прогнозов, энергия которой ∼(1−cos F) ≈ ½F² для малых флуктуаций (Wilson action). (4) Константа связи g² выражена через антиферромагнитную константу J_t: g² ∼ 1/J_t, и оценена в критической точке v* как g² ≈ 0.1, что качественно согласуется с α≈1/137 (требует уточнения). (5) Статус каждого утверждения явно указан: «строго доказано» / «выведено в приближении» / «гипотеза».

### 1.1 Постановка задачи

Предыдущие 17 версий [1] выявили центральную проблему: как локальное калибровочное поле возникает в Ze-теории без процедуры gauging? v17 предложила эвристический ответ: локальность T/S-базиса → связность → кривизна → действие Максвелла. ## 2. T/S как базис векторного пространства

### 2.1 Определение

Для каждого узла x определим одномерное вещественное векторное пространство V_x = span{|T⟩_x} с отношением |S⟩_x = −|T⟩_x (антипараллелизм). Тогда dim_ℝ(V_x) = 1.

Выбор знака: |T⟩_x → −|T⟩_x (и, следовательно, |S⟩_x → −|S⟩_x) **не меняет** наблюдаемые в узле x, поскольку все физические величины зависят только от бинарного значения z_x = ±1, которое **инвариантно** относительно одновременной смены знака обоих базисных векторов: если |T⟩ → −|T⟩ и |S⟩ → −|S⟩, то |T⟩ − |S⟩ → −(|T⟩ − |S⟩), и знак разности не меняет квадратичных наблюдаемых.

**Вывод:** V_x — одномерное векторное пространство со свободой выбора ориентации (знака). Это Z₂-главное расслоение над графом Ze-событий.

### 2.2 Независимость ориентаций

Прогноз в узле x определяет, какое событие считается T, а какое S. Прогнозы разных узлов **независимы** (каждая Ze-система автономна). Следовательно, выбор ориентации V_x и V_y **не скоррелирован** априори.

Это — источник локальной калибровочной свободы: в каждом узле знак базиса может быть выбран независимо.

## 3. Связность

### 3.1 Необходимость

Для вычисления корреляции ⟨z_x z_y⟩ необходимо сравнить z_x ∈ V_x и z_y ∈ V_y. Но сравнение векторов из разных слоёв расслоения требует **связности** — изоморфизма между слоями.

На каждом ребре (x, y) определён изоморфизм U_{xy}: V_x → V_y, действующий как умножение на ±1. Это Z₂-связность (Z₂-калибровочное поле).

### 3.2 Групповой переход Z₂ → U(1)

Для N=2 независимых Ze-потоков: V_x = V_x^(1) ⊕ V_x^(2) (два одномерных пространства). Группа изоморфизмов: SO(2) ≅ U(1). Связность: U_{xy} ∈ SO(2).

В континуальном пределе: U_{x, x+δx} = exp(i A_μ(x) δx^μ), где A_μ(x) ∈ ℝ — непрерывное калибровочное поле.

## 4. Вывод действия

### 4.1 Кривизна и несогласованность прогнозов

Параллельный перенос вектора |T⟩_x по замкнутому контуру □ (плакету) даёт:

$$|T\rangle_x \xrightarrow{\Box} U_{\Box} |T\rangle_x$$

где U_□ = Π_{edges∈□} U_{xy} = exp(i ∮ A·dx) = exp(i F_□) — голономия.

Если U_□ ≠ 1, базис после обхода не совпадает с исходным. В терминах Ze: **прогнозы вдоль замкнутого контура противоречат друг другу.** Это создаёт «напряжение»: система не может одновременно удовлетворить все прогнозы.

### 4.2 Энергия несогласованности (Wilson action)

Мера несогласованности: 1 − Re(U_□) = 1 − cos(F_□). Для малых флуктуаций: 1 − cos(F) ≈ F²/2.

Действие калибровочного поля (Wilson action [2]):

$$S_{\text{gauge}} = \beta \sum_{\Box} (1 - \text{Re}\, U_{\Box})$$

В континуальном пределе a → 0:

$$S_{\text{gauge}} \to \frac{1}{4g^2} \int d^4x\, F_{\mu\nu}F^{\mu\nu}$$

где 1/g² = βa^{4−d} (в d=4: β = 1/g²).

**Это вывод, а не постулат.** Форма действия ∼(1−cos F) → ∼F² однозначно следует из: (а) калибровочной инвариантности, (б) локальности (ближайшие соседи), (в) аналитичности по F.

### 4.3 Связь с микроскопикой Ze

Параметр β (обратная константа связи) выражается через микроскопические параметры Ze. В пределе слабой связи (большое β) калибровочное поле «жёсткое» — базисы сильно согласованы. В пределе сильной связи (малое β) базисы гибкие — велика толерантность к рассогласованию.

В критической точке v=v* энтропия максимальна, порядок минимален, жёсткость минимальна → β минимально → g² максимально. Это даёт верхнюю границу для константы связи.

## 5. Оценка константы связи

### 5.1 Связь β с J_t

Антиферромагнитное взаимодействие вдоль времени J_t (антипараллелизм) определяет энергетический масштаб T/S-переворотов. Естественная оценка:

$$\beta \sim J_t$$

В критической точке v=v*: J_t ∼ k_B T_c. Для 1D модели Изинга в поперечном поле T_c ∼ J_t (квантовый фазовый переход при h=J_t).

### 5.2 Оценка g²

В 4D: g² = 1/β ∼ 1/J_t. При J_t ∼ 1 (в единицах энергии решётки): g² ∼ 1.

Но наблюдаемая константа связи КЭД: α = e²/4π ≈ 1/137 ≈ 0.0073, что требует g² ≈ 4πα ≈ 0.09.

Это **на порядок** меньше, чем естественная оценка g² ∼ 1. Расхождение может быть объяснено:
- Перенормировкой (g² растёт с масштабом — полюс Ландау)
- Тем, что наблюдаемое α — инфракрасное значение, а «голое» g² на масштабе решётки больше

**Статус:** оценка грубая. Точное вычисление g²(v*) требует непертурбативных методов.

| Локальная калибровочная свобода | ✅ Строго доказано | Независимость выбора ориентации V_x |

## 7. Заключение

Построен математический формализм, в котором:
1. T/S-переменные образуют одномерное векторное пространство V_x в каждом узле;
2. Независимость выбора ориентации V_x — источник локальной Z₂-калибровочной симметрии;
3. Связность между слоями V_x и V_y — калибровочное поле;
4. Действие ∼F² выведено из энергетической стоимости несогласованности T/S-базисов (Wilson action);
5. Константа связи g² оценена через J_t.

**Основное достижение v18:** локальное калибровочное поле и его действие **выведены математически**, а не постулированы эвристически. Оставшиеся проблемы (континуальный предел, точное значение g²) явно указаны.
