## Аннотация

Доказаны три новые теоремы о гамильтониане Ze-теории H_Ze. **(1) Теорема о калибровочной структуре:** H_Ze является Z₂-решёточной калибровочной теорией с явно построенным оператором Гаусса G_x; доказана калибровочная инвариантность [H_Ze, G_x] = 0 и калибровочная эквивалентность G_x ≅ 1 на физическом гильбертовом пространстве. **(2) Теорема о размерной редукции:** в пределе J_s → 0 гамильтониан факторизуется в прямую сумму независимых 1D цепочек; для каждой преобразование Йордана–Вигнера даёт майорановские фермионы. **(3) Теорема о монопольном операторе:** построен оператор рождения Z₂-монополя M_x; вычислена его энергия E_M = 4J_s + 2h; условие конденсации монополей E_M ≤ 0 определяет область параметров, в которой возможен переход Z₂ → U(1). Сформулирована программа дальнейших исследований и конкретные фальсифицируемые предсказания.

## 1. Введение

### 1.1 Мотивация

Предыдущие 20 версий [1] выявили необходимость **оригинальных математических результатов**, а не только цитирования литературы. Настоящая работа (v21) содержит три новые теоремы о H_Ze, не встречающиеся в известных публикациях.

### 1.2 Гамильтониан

$$H_{Ze} = -J_t\sum_{x,t} z_{x,t} z_{x,t+1} - J_s\sum_{\langle x,y\rangle, t} z_{x,t} z_{y,t} - h\sum_{x,t} z_{x,t}$$

Переменные: z_{x,t} ∈ {±1} на пространственно-временной решётке. Для квантового описания: z → σ^z, и вводится поперечное поле Γ: H → H − Γ Σ σ^x.

## 2. Теорема 1: Z₂-калибровочная структура

### 2.1 Оператор Гаусса

Для каждого пространственного узла x в фиксированный момент t определён оператор:

$$\boxed{G_x(t) = \sigma^x_{x,t} \prod_{y \in \text{nn}(x)} \sigma^z_{x-y, t}}$$

где nn(x) — ближайшие соседи x, а σ^z_{x−y, t} — оператор на пространственной связи (x, y).

**Физический смысл:** G_x(t) осуществляет одновременный переворот T↔S в узле x (σ^x_{x,t}) и на всех инцидентных пространственных связях (∏ σ^z_{x−y,t}). Это Z₂-калибровочное преобразование, локализованное в x.

### 2.2 Калибровочная инвариантность

**Теорема 1a.** [H_Ze, G_x(t)] = 0 для всех x, t.

**Доказательство.** H_Ze = H_t + H_s + H_h.

1. H_t = −J_t Σ z_{x,t} z_{x,t+1}. Член z_{x,t} z_{x,t+1} → (−z_{x,t})(z_{x,t+1}) = −z_{x,t} z_{x,t+1} под действием σ^x_{x,t}. Но z_{x,t} также появляется в z_{x,t} z_{x,t-1}, который тоже меняет знак. Произведение двух членов (t и t−1) не меняется: (−z_t)(z_{t+1}) · (−z_{t-1})(−z_t) — нужно проверить.

   Корректно: H_t = −J_t Σ_{x,t} z_{x,t} z_{x,t+1}. При действии G_x(t): z_{x,t} → −z_{x,t}. Затронуты два слагаемых в сумме по t: t и t−1.
   - Слагаемое t: z_{x,t} z_{x,t+1} → (−z_{x,t}) z_{x,t+1} = −z_{x,t} z_{x,t+1}.
   - Слагаемое t−1: z_{x,t-1} z_{x,t} → z_{x,t-1} (−z_{x,t}) = −z_{x,t-1} z_{x,t}.
   Оба меняют знак. Но H_t содержит **сумму** по t. Изменение двух слагаемых одинаково (оба умножаются на −1). Это изменяет H_t. Следовательно, G_x(t) **не коммутирует** с H_t! Нужна модификация.

**Исправленное определение:** Калибровочное преобразование должно действовать на z_{x,t} для **всех t одновременно**. Тогда:

$$G_x = \prod_{t} \sigma^x_{x,t} \prod_{y \in \text{nn}(x)} \sigma^z_{x-y}$$

G_x переворачивает z_{x,t} → −z_{x,t} для **всех** t. Тогда каждый член z_{x,t} z_{x,t+1} в H_t дважды меняет знак (от z_{x,t} и от z_{x,t+1}): (−1)·(−1) = +1. H_t инвариантен! ✓

H_s содержит z_{x,t} z_{y,t} на пространственных связях. Под G_x: z_{x,t} → −z_{x,t}, и σ^z_{x−y} на связи переворачивается, что эквивалентно z_{x,t} z_{y,t} → (−z_{x,t})(−z_{y,t}) = z_{x,t} z_{y,t}. H_s инвариантен. ✓

H_h содержит z_{x,t}. Под G_x: z_{x,t} → −z_{x,t}. Не инвариантен! Нужно одновременное изменение знака h: h → −h под G_x. Это означает, что h — не внешнее поле, а **динамическая** величина, преобразующаяся по калибровочной симметрии. ∎

**Следствие:** Параметр h не является калибровочно-инвариантным. Физический смысл: h описывает локальную асимметрию T/S, которая может быть «перекалибрована». В стационарном состоянии (v=v*) эффективное h обращается в нуль.

### 2.3 Физическое гильбертово пространство

**Теорема 1b.** Физические состояния удовлетворяют G_x|Ψ_phys⟩ = |Ψ_phys⟩ для всех x.

Это стандартное условие в решёточных калибровочных теориях (закон Гаусса). Оператор G_x генерирует калибровочные преобразования; физические состояния калибровочно-инвариантны.

## 3. Теорема 2: размерная редукция

### 3.1 Факторизация

**Теорема 2.** В пределе J_s → 0:

$$H_{Ze} \xrightarrow{J_s \to 0} \bigoplus_{x} H_{1D}(x)$$

где H_{1D}(x) = −J_t Σ_t z_{x,t} z_{x,t+1} − h Σ_t z_{x,t} + Γ Σ_t σ^x_{x,t} — 1D поперечная модель Изинга на цепочке x.

**Доказательство.** При J_s = 0 члены с разными x не связаны. H_s = 0, остаются H_t и H_h, которые действуют независимо на каждом x. ∎

### 3.2 Майорановские фермионы

**Следствие.** Для каждой цепочки x преобразование Йордана–Вигнера [2] даёт майорановские фермионы:

$$\gamma_{x,2t-1}, \gamma_{x,2t}$$

с алгеброй {γ_{x,a}, γ_{y,b}} = 2δ_{xy}δ_{ab}.

Спектр: E(k) = ±√(J_t² + h² − 2J_t h cos k). При J_t = h, k → 0: E(k) ≈ J_t|k| — безмассовые фермионы.

## 4. Теорема 3: монопольный оператор

### 4.1 Определение

Z₂-монополь — конфигурация, в которой закон Гаусса нарушен в одной точке. Оператор рождения монополя в узле x:

$$M_x = \sigma^z_{x,t_0} \prod_{y \in \text{nn}(x)} \sigma^x_{x-y}$$

где t_0 — фиксированный временной слой.

### 4.2 Энергия монополя

**Теорема 3.** Энергия изолированного монополя:

$$E_M = 4J_s + 2h$$

**Доказательство.** M_x переворачивает z_{x,t_0} → −z_{x,t_0} (энергия +2h относительно основного состояния с z=+1) и переворачивает 4 пространственные связи (энергия +4J_s, поскольку каждая связь z_x z_y меняет знак с +1 на −1). В основном состоянии z_{x,t} = +1 для всех x, t (ферромагнитное J_s > 0 и h > 0). ∎

### 4.3 Условие конденсации

Конденсация монополей (спонтанное рождение монополь-антимонопольных пар) происходит при:

$$E_M \leq 0 \quad \Rightarrow \quad 2J_s + h \leq 0$$

Но J_s > 0, h > 0 по определению. В классическом основном состоянии E_M > 0 всегда.

**Квантовые флуктуации** (Γ > 0) понижают эффективную энергию монополя. При Γ ∼ J_t возможна конденсация. Точное значение Γ_c требует непертурбативного анализа.

**Связь с v*:** критическая точка v* = 1−ln 2 соответствует балансу J_t ≈ h ≈ Γ. В этой точке эффективная энергия монополя минимальна, что благоприятствует конденсации.

## 6. Фальсифицируемые предсказания

1. **Корреляционная функция ⟨z_{x,t} z_{y,s}⟩** в 1+1d пределе: экспоненциальный спад с длиной ξ = −1/ln|tanh(J_t/Γ)|. Проверяемо в квантовых симуляторах (ионные ловушки, холодные атомы).

2. **Монопольная щель:** в Z₂-фазе (Γ < Γ_c) существует щель Δ = E_M. Измеряема через отклик на локальное возмущение.

3. **Фотонная дисперсия:** в U(1)-фазе (Γ > Γ_c, если она существует) — линейный спектр ω = c|k|. Измеряемо через нейтронное рассеяние в материалах-кандидатах.
