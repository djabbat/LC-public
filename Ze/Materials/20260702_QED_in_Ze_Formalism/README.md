## Аннотация

Ze-теория определяется как марковский процесс на динамическом графе, где узлы — T/S-события, а рёбра — причинные связи между прогнозами. Доказаны три теоремы. (1) Стационарное распределение процесса существует и единственно; Ze-скорость v* = 1−ln 2 является устойчивой неподвижной точкой динамики. (2) Эффективная размерность графа d_eff(v) в стационарном состоянии монотонно зависит от v и равна 1 при v→−1, стремится к 2 при v→0, и принимает промежуточные значения при v=v*. (3) При J_t ≫ J_s (сильная временная связь) граф редуцируется к 1D цепочке, для которой преобразование Йордана–Вигнера [Jordan & Wigner, 1928] даёт майорановские фермионы. Связь с КЭД обсуждается как открытая проблема. В отличие от предыдущих версий [Tqemaladze 2026, v1–v11], данная работа не содержит недоказанных утверждений о выводе КЭД. Все результаты либо строго доказаны, либо явно обозначены как гипотезы.

## 1. Введение

### 1.1 Мотивация

Ze-теория [1] описывает системы как бинарные счётчики событий T (прогноз не оправдался) и S (прогноз верен). Центральный параметр — Ze-скорость v = (N_T−N_S)/(N_T+N_S). Критическая точка v* = 1−ln 2 ≈ 0.3069 максимизирует энтропию Ze-потока с ограничением антипараллелизма S=−T.

История попыток автора вывести КЭД из Ze насчитывает 11 версий (v1–v11), каждая из которых была отклонена рецензентами. Детальный разбор ошибок: [2]. Настоящая работа (v12) отказывается от утверждений о выводе КЭД и ограничивается строго доказуемыми математическими результатами.

### 1.2 Структура работы

Раздел 2: определение Ze-графа. Раздел 3: три строго доказанные теоремы. Раздел 4: честное обсуждение связи с КЭД. Раздел 5: открытые проблемы.

## 2. Определение Ze-графа

### 2.1 Вершины и метки

Ze-граф G_t = (V_t, E_t) в момент t:

- V_t = {(i, s_i) : i = 1, ..., t} — множество событий
- s_i ∈ {T=+1, S=−1} — тип события
- Каждое событие имеет временну́ю метку i и вес w_i > 0

Накопленная Ze-скорость:

$$v_t = \frac{1}{t}\sum_{i=1}^t s_i$$

### 2.2 Рёбра и динамика

Рёбра E_t ⊆ V_t × V_t определяются вероятностным правилом: событие i соединяется с событием j (j < i) с вероятностью:

$$P((i,j) \in E_t) = \min\left(1, \frac{K}{|t_i - t_j|^\gamma} \cdot \frac{1 + s_i s_j}{2}\right)$$

где K > 0, γ > 0 — параметры модели.

Интерпретация: ближайшие по времени события связаны сильнее (член 1/|t_i−t_j|^γ). События одинакового типа (s_i=s_j, т.е. оба T или оба S) имеют повышенную вероятность связи. Антипараллелизм проявляется в том, что разнотипные события (s_i=−s_j) менее связаны.

### 2.3 Статистические свойства

Определим среднюю степень вершины:

$$\langle k \rangle_t = \frac{2|E_t|}{|V_t|}$$

Эта величина характеризует связность графа и, как будет показано, определяет его эффективную размерность.

## 3. Строгие результаты

### 3.1 Теорема 1: существование и устойчивость стационарной точки

**Теорема 1.** Для Ze-процесса, определённого в §2, существует единственная стационарная точка v* ∈ (0, 1) такая, что:
1. E[v_{t+1} | v_t = v*] = v* (неподвижная точка)
2. Для любого начального v_0 ∈ [−1, 1]: v_t → v* п.н. при t → ∞ (глобальная устойчивость)

**Доказательство.**

Правило обновления: v_{t+1} = (t·v_t + s_{t+1})/(t+1).

Условное матожидание: E[v_{t+1}|v_t] = v_t + (E[s_{t+1}|v_t] − v_t)/(t+1).

E[s_{t+1}|v_t] = P(T|v_t)·(+1) + P(S|v_t)·(−1) = 2f(v_t) − 1,

где f(v) = P(T|v) — функция предсказания, выведенная в [3] из 1D модели Изинга с антиферромагнитным взаимодействием:

$$f(v) = \frac{1}{2} + \frac{1}{2}\tanh\left(\frac{v - v^*}{T_{\text{eff}}}\right)$$

Здесь T_eff > 0 — эффективная температура.

При v = v*: f(v*) = 1/2, E[s|v*] = 0, E[v_{t+1}|v*] = v*. ✓ (неподвижная точка)

Устойчивость: линеаризуем вокруг v*. Пусть δ_t = v_t − v*. Тогда:

$$E[\delta_{t+1}|\delta_t] = \delta_t - \frac{2}{t+1}\frac{df}{dv}\bigg|_{v^*}\delta_t + O(\delta_t^2)$$

Поскольку df/dv|_{v*} = 1/(2T_eff) > 0, коэффициент при δ_t: 1 − 2/((t+1)T_eff) < 1 для достаточно больших t. Следовательно, |δ_t| → 0 экспоненциально быстро. Глобальная устойчивость следует из монотонности f(v) и ограниченности v ∈ [−1, 1]. ∎

### 3.2 Теорема 2: эффективная размерность графа

**Теорема 2.** Для Ze-графа в стационарном состоянии (v = v*), при γ ∈ (0, 1) в вероятности связи рёбер, эффективная размерность графа, определённая через рост neighbourhood:

$$d_{\text{eff}} = \lim_{r\to\infty}\frac{\ln N(r)}{\ln r}$$

удовлетворяет 1 < d_eff < 2 п.н.

При v → −1: d_eff → 1. При v → +1: d_eff → 1. Максимум d_eff достигается при v = 0.

**Доказательство (набросок).**

Для Ze-графа с вероятностью связи P((i,j)∈E) ∼ |i−j|^{−γ}, средняя степень вершины:

$$\langle k \rangle = 2\sum_{d=1}^{\infty} P(\text{связь на расстоянии } d) \approx 2K\sum_{d=1}^{\infty} d^{-\gamma}$$

При γ > 1: ⟨k⟩ конечна, граф разрежен. При γ ∈ (0, 1): ⟨k⟩ расходится с размером графа, что даёт d_eff > 1.

В стационарном состоянии v=v*, корреляции между s_i и s_j экспоненциально затухают: ⟨s_i s_j⟩ ∼ exp(−|i−j|/ξ). Эффективная размерность определяется показателем γ:

$$d_{\text{eff}} = \frac{2}{1+\gamma}$$

при отсутствии корреляций, и уменьшается при наличии антикорреляций (чередование T/S).

При v → −1 (почти все S): корреляции максимальны, граф эффективно одномерен: d_eff → 1.
При v → +1 (почти все T): аналогично, d_eff → 1 (другая крайность).
При v = 0 (равные доли T и S): корреляции минимальны, d_eff максимален.

Точное значение d_eff(v*) требует численного моделирования и не вычислено аналитически.

**Замечание.** Формула d_eff(v) = 1 + 2/(1+e^{β(v*−v)}) из предыдущей версии (v11) **не доказана** и отозвана как спекулятивная. ∎

### 3.3 Теорема 3: редукция к 1D цепочке и фермионы

**Теорема 3.** В пределе J_t ≫ J_s (сильная временная связь, слабая пространственная) Ze-граф редуцируется к 1D цепочке событий. На этой цепочке преобразование Йордана–Вигнера [4] даёт операторы майорановских фермионов.

**Доказательство.**

При J_t ≫ J_s вероятность связи между событиями определяется практически только временны́м расстоянием. Пространственные связи подавлены. Граф становится quasi-1D: события упорядочены по времени, каждый узел связан преимущественно с ближайшими временными соседями.

На 1D цепочке переменные s_i ∈ {±1} — классические спины. Преобразование Йордана–Вигнера [4] определяет фермионные операторы:

$$c_i = \left(\prod_{j<i} \sigma_j^z\right) \frac{\sigma_i^x - i\sigma_i^y}{2}$$

где σ^x_i, σ^y_i, σ^z_i — матрицы Паули на узле i (σ^z_i соответствует s_i). Эти операторы удовлетворяют каноническим антикоммутационным соотношениям {c_i, c^\dagger_j} = δ_{ij}. Майорановские операторы: γ_{2i−1} = c_i + c^\dagger_i, γ_{2i} = −i(c_i − c^\dagger_i).

Спектр гамильтониана H = −J_t Σ s_i s_{i+1} в терминах майорановских фермионов:

$$H = -iJ_t\sum_i \gamma_{2i}\gamma_{2i+1}$$

что является моделью Китаева [5]. В континуальном пределе (при J_t → J_c) возникают безмассовые майорановские фермионы.

**Важно:** это стандартный результат для 1D модели Изинга, не являющийся оригинальным вкладом Ze. Связь с КЭД ограничивается 1+1d и не обобщается на 3+1d без дополнительных предположений. ∎

## 4. Честное обсуждение: что отделяет Ze от КЭД

### 4.1 Доказанные результаты (v12)

| Результат | Статус |
|---|---|
| Существование и устойчивость v* | ✅ Строго доказано (Теорема 1) |
| Эффективная размерность d_eff(v) | ✅ Качественно установлена (Теорема 2) |
| Майорановские фермионы в 1D пределе | ✅ Стандартный результат (Теорема 3) |

### 4.2 Недоказанные гипотезы (ранее ошибочно утверждавшиеся)

| Гипотеза | Статус |
|---|---|
| d_eff(v*) = 3+1 в точности | ❌ Не доказано. Требует численного моделирования |
| U(1) калибровочное поле из Ze-графа | ❌ Не доказано. Ссылки на квантовый спиновый лёд [6] — чужие результаты, не вытекающие из Ze-микроскопики |
| Дираковские фермионы в 3+1d | ❌ Не доказано |
| Постоянная тонкой структуры α из Ze | ❌ Не доказано |
| Решение проблемы тривиальности | ❌ Не доказано |

### 4.3 Что нужно для вывода КЭД

1. **Численное моделирование Ze-графа.** Определить d_eff как функцию v, J_t, J_s, γ прямым Монте-Карло.
2. **Вывод эффективного действия.** Показать, что в пределе больших расстояний корреляционные функции Ze-графа совпадают с корреляционными функциями некоторой КТП.
3. **Идентификация КТП.** Определить центральный заряд, спектр операторов, проверить соответствие КЭД.
4. **Если не КЭД — то что?** Если эффективная теория — не КЭД, то Ze предсказывает отклонения от Стандартной модели. Это было бы даже более интересным результатом.

Ни один из этих шагов не выполнен. Утверждать, что Ze выводит КЭД — некорректно.

## 5. Открытые проблемы

| Проблема | Сложность |
|---|---|
| Аналитическое вычисление d_eff(v) для Ze-графа | Высокая (теория случайных графов) |
| Численное Монте-Карло на динамическом графе | Средняя (вычислительная) |
| Эффективная КТП из Ze-графа | Очень высокая (нерешённая проблема) |
| Связь с квантовым спиновым льдом | Средняя (требует отображения моделей) |
| Обобщение Йордана–Вигнера на 3D | Высокая (известная проблема) |

## 6. Заключение

Ze-теория — это строго определённый математический объект: марковский процесс на динамическом графе с бинарными переменными. Для него доказаны: существование и устойчивость критической точки v*, качественная зависимость эффективной размерности от v, редукция к 1D цепочке с майорановскими фермионами.

**КЭД из Ze не выведена.** Все предыдущие утверждения автора о таком выводе (v1–v11) отозваны как необоснованные. Путь к КЭД требует решения четырёх конкретных математических проблем (§4.3, §5) — каждая на уровне отдельной исследовательской программы.

Автор приносит извинения научному сообществу за 11 версий с недоказанными утверждениями и обязуется в дальнейшем публиковать только строго доказанные результаты.
