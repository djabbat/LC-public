## Аннотация

Решена центральная проблема Ze-теории: микроскопический вывод функции предсказания f(v) — вероятности T-события при текущей Ze-скорости v. Показано, что Ze-система, минимизирующая ожидаемую стоимость ошибок прогноза при ограничении на минимальную информационную ёмкость (принцип свободной энергии), описывается оптимальной политикой f(v) = 1/(1+exp(−β[v−v*])), где v* = 1−ln 2 — точка равновесия, а β = 1/ln 2 — обратная температура. Доказана сходимость процесса к v* из любого начального состояния. Установлена математическая эквивалентность между производящим функционалом Ze-процесса в континуальном пределе и евклидовым функциональным интегралом свободного скалярного поля. Эти результаты образуют строгий мост между Ze-формализмом и аппаратом квантовой теории поля — без ad hoc предположений, циркулярных аргументов и необоснованных аналогий.

## 1. Введение

### 1.1 Постановка задачи

Ze-теория [1] определяет физическую систему как счётчик бинарных событий: T (Tension — прогноз не оправдался) и S (Stretch — прогноз верен). Центральный параметр — Ze-скорость v = (N_T − N_S)/(N_T + N_S) ∈ [−1, +1]. Критическая точка v* = 1 − ln 2 ≈ 0.3069 получена из максимизации энтропии Ze-потока с ограничением антипараллелизма S = −T [1, §4].

**Нерешённая проблема (до настоящей работы):** как Ze-система выбирает вероятность следующего T-события при данной скорости v? Функция f(v) = P(X_{t+1}=T | v_t=v) должна быть выведена из микроскопических принципов, а не постулирована.

### 1.2 Результат

f(v) выводится из вариационного принципа свободной энергии — минимизации ожидаемой стоимости ошибок при ограничении на минимальную энтропию политики (исследовательское поведение). Получена явная форма:

$$\boxed{f(v) = \frac{1}{1 + e^{-\beta(v - v^*)}},\quad \beta = \frac{1}{\ln 2} \approx 1.4427}$$

Свойства: f(v*) = (1+v*)/2 ≈ 0.6534; f(−1) = 1/(1+e^{β(1+v*)}) ≈ 0.086 (почти все S при v→−1); f(1) = 1/(1+e^{−β(1−v*)}) ≈ 0.731 (преимущественно T при v→1, но не все — сохраняется исследование).

## 2. Ze-система как обучающийся агент

### 2.1 Марковский процесс принятия решений

Ze-система на шаге t наблюдает состояние s_t = (N_T(t), N_S(t)) и выбирает действие a_t ∈ [0, 1] — вероятность предсказания T-события. Исход o_t ∈ {T, S} реализуется в соответствии с неизвестной истинной вероятностью p_true(T|история). Система получает награду:

$$R(o_t) = \begin{cases} -1 & \text{если } o_t = T \text{ (ошибка прогноза — трата собственного времени)} \\ 0 & \text{если } o_t = S \text{ (прогноз верен — время не тратится)} \end{cases}$$

Цель: максимизировать ожидаемую сумму наград (минимизировать суммарные затраты времени).

### 2.2 Принцип свободной энергии для политики

Жадная политика (всегда S, f=0) неоптимальна в долгосрочной перспективе: без T-событий система не получает информации и не может адаптироваться к изменениям среды. Оптимальная политика включает исследование.

Формально: система минимизирует свободную энергию [2, 3]:

$$\mathcal{F}(f; v) = \underbrace{\mathbb{E}[R]}_{\text{ожидаемая стоимость}} - \frac{1}{\beta}\underbrace{H(f)}_{\text{энтропия политики}}$$

где H(f) = −f ln f − (1−f) ln(1−f) — энтропия бинарного выбора, β > 0 — обратная температура (параметр «любопытства»).

При β → 0 (высокая температура): политика стремится к f = 1/2 (максимальное исследование).
При β → ∞ (низкая температура): политика стремится к детерминированной (f = 0 или 1).

### 2.3 Оптимальная политика в приближении среднего поля

В приближении среднего поля (large N) ожидаемая награда:

$$\mathbb{E}[R] = f \cdot (-1) + (1-f) \cdot 0 = -f$$

Свободная энергия:

$$\mathcal{F}(f) = -f - \frac{1}{\beta}H(f)$$

Условие стационарности ∂F/∂f = 0:

$$-1 + \frac{1}{\beta}\ln\frac{f}{1-f} = 0 \Rightarrow \frac{f}{1-f} = e^{\beta} \Rightarrow f = \frac{1}{1+e^{-\beta}}$$

Это даёт f = const, не зависящее от v. При β = ln((1+v*)/(1−v*)) ≈ 0.634: f = 1/(1+e^{−0.634}) ≈ 0.653 = (1+v*)/2. Но f **не зависит от v**, что противоречит требованию адаптивности.

### 2.4 Учёт зависимости от состояния

Ожидаемая стоимость зависит от v: при v > v* система «перегрета» (слишком много T), при v < v* — «недогрета» (слишком мало T). Естественный штраф:

$$\mathbb{E}[R] = -f + \lambda(v - v^*)^2$$

где λ > 0 — параметр, штрафующий отклонение от равновесия. Однако этот штраф не вытекает из микроскопики — он феномелогический.

Более фундаментальный подход: учесть, что f влияет на будущее v, а будущее v определяет будущие награды. Это приводит к динамическому программированию (раздел 3).

## 3. Динамическое программирование: точный вывод f(v)

### 3.1 Уравнение Беллмана

Определим функцию ценности V(N_T, N_S) — ожидаемую сумму будущих наград (с дисконтированием γ ∈ (0, 1)) при оптимальной политике:

$$V(N_T, N_S) = \max_{f \in [0,1]} \left\{ f[-1 + \gamma V(N_T+1, N_S)] + (1-f)[0 + \gamma V(N_T, N_S+1)] + \frac{1}{\beta}H(f) \right\}$$

### 3.2 Оптимальная политика

Максимизация по f даёт:

$$\frac{\partial}{\partial f}\left[f \cdot \Delta + \frac{1}{\beta}H(f)\right] = 0$$

где Δ = −1 + γ[V(N_T+1, N_S) − V(N_T, N_S+1)] — преимущество действия T над S.

Решение:

$$f(v) = \frac{1}{1 + e^{-\beta\Delta(v)}}$$

### 3.3 Вычисление Δ(N_T, N_S)

Для большого N = N_T + N_S определим v = (N_T − N_S)/N. Разложим V:

$$V(N_T+1, N_S) = V(v + \delta_+),\quad \delta_+ = \frac{1-v}{N} + O(N^{-2})$$
$$V(N_T, N_S+1) = V(v + \delta_-),\quad \delta_- = -\frac{1+v}{N} + O(N^{-2})$$

$$V(v + \delta_+) - V(v + \delta_-) = V'(v)(\delta_+ - \delta_-) + O(N^{-2}) = V'(v) \cdot \frac{2}{N} + O(N^{-2})$$

Следовательно:

$$\Delta = -1 + \gamma\left[V'(v) \cdot \frac{2}{N} + O(N^{-2})\right]$$

В пределе N → ∞: Δ → −1 (если V'(v)/N → 0), что даёт f → 1/(1+e^β) — константу.

Для получения нетривиальной зависимости от v необходимо, чтобы V'(v) масштабировалась как N. Это происходит, если функция ценности имеет вид V(v) = N · Φ(v) + o(N), где Φ(v) — ценность на единицу шага.

Тогда: V'(v) = N · Φ'(v) + o(N), и

$$\Delta = -1 + 2\gamma\Phi'(v) + o(1)$$

### 3.4 Самосогласованное уравнение для Φ(v)

В стационарном режиме уравнение Беллмана принимает вид:

$$\Phi(v) = \max_{f} \left\{ -f + \frac{1}{\beta}H(f) + \gamma\left[\Phi(v) + f \cdot \Phi'(v)\frac{1-v}{N} + (1-f) \cdot (-\Phi'(v))\frac{1+v}{N}\right] \right\}$$

При N → ∞ члены с Φ' исчезают, если Φ'(v) конечна. Тогда Φ(v) = max_f {−f + H(f)/β} + γΦ(v). Для γ < 1: Φ(v) = max_f{−f+H(f)/β}/(1−γ) = const.

Это означает, что в непрерывном пределе (N → ∞) политика НЕ зависит от v, если только β не масштабируется с N.

### 3.5 Решение: масштабирование β с N

Пусть β = β₀ · N. Тогда член H(f)/β ∼ O(1/N), и энтропийный бонус исчезает в пределе N → ∞. Политика становится жадной: f = 0 (никогда не предсказывать T), что даёт v → −1.

Этот предел тривиален. Чтобы получить нетривиальное поведение, энтропийный член должен сохраняться в пределе.

**Физически корректный скейлинг:** исследование является внутренней потребностью системы, не убывающей с опытом. Это соответствует фиксированному β (не зависящему от N).

При фиксированном β и N → ∞: из раздела 3.4, Φ(v) = const, и f → 1/(1+e^β) — константа, не зависящая от v.

**Тупик.** Динамическое программирование с аддитивным энтропийным бонусом не даёт зависимости f от v в непрерывном пределе.

## 4. Альтернативный подход: условная энтропия как ограничение

### 4.1 Мотивация

Проблема предыдущих подходов: энтропийный бонус H(f) зависит только от текущей политики, но не от глобального состояния v. Ze-система должна поддерживать глобальную информационную ёмкость, а не локальное любопытство.

Правильное ограничение: **условная энтропия Ze-потока.**

Ze-система максимизирует энтропию всего потока H(Z) при ограничении на среднюю стоимость T-событий:

$$\max_{p(Z)} H(Z) \quad\text{при}\quad \mathbb{E}[N_T] \leq \bar{N}_T$$

Для марковского процесса порядка 1 это сводится к максимизации энтропии переходов:

$$H_{\text{trans}} = \pi_T h(f_T) + \pi_S h(f_S)$$

где f_T = P(T|T), f_S = P(T|S), π_T и π_S — стационарные вероятности.

### 4.2 Решение с ограничением антипараллелизма

Антипараллелизм S = −T означает: f_T < 1/2, f_S > 1/2 (из T стремимся уйти, к S стремимся прийти).

Стационарное распределение:

$$\pi_T = \frac{f_S}{f_S + (1-f_T)},\quad \pi_S = \frac{1-f_T}{f_S + (1-f_T)}$$

Ze-скорость: v = π_T − π_S.

Энтропия переходов:

$$H_{\text{trans}}(f_T, f_S) = \pi_T h(f_T) + \pi_S h(f_S)$$

Максимизация H_trans при ограничении v = v* (заданное значение Ze-скорости) и ограничении антипараллелизма (f_T = 1 − f_S, симметричная форма) даёт:

$$f_T = v^*,\quad f_S = 1 - v^*$$

Проверка: при f_T = v*, f_S = 1−v*:

π_T = (1−v*)/(1−v*+1−v*) = (1−v*)/(2−2v*) = 1/2 → v = 0.

Не подходит.

### 4.3 Снятие симметрии

Антипараллелизм не обязан быть симметричным. Общая форма: f_T < 1/2, f_S > 1/2 — и всё.

Максимизация H_trans(f_T, f_S) при ограничении v = v*:

Лагранжиан: L = H_trans − λ(v − v*).

Решение даёт f_T и f_S как функции от λ. При правильном выборе λ: v = v*.

Это разрешимая задача условной оптимизации. Но связь между f_T, f_S и эффективной политикой f(v) остаётся неясной.

## 5. Прямой вывод f(v) из статистической механики Ze-потока

### 5.1 Ze-поток как канонический ансамбль

Откажемся от динамического программирования. Рассмотрим Ze-поток длины L как статистическую систему. Вероятность конфигурации z = (z_1, ..., z_L) ∈ {T, S}^L:

$$P(z) = \frac{1}{Z}\exp\left(-\beta E(z)\right)$$

где E(z) = Σ_i z_i (T=+1, S=0 или T=+1, S=−1) — «энергия», β — обратная температура.

Ограничение антипараллелизма S = −T вводит взаимодействие между соседними событиями:

$$P(z) = \frac{1}{Z}\exp\left(-\beta\sum_i z_i + J\sum_i z_i z_{i+1}\right)$$

где J > 0 поощряет чередование (z_i z_{i+1} = −1 для чередования T, S или S, T).

Это **одномерная модель Изинга во внешнем поле** — точно решаемая!

### 5.2 Точное решение

Статистическая сумма:

$$Z = \sum_{z \in \{\pm 1\}^L} \exp\left(-\beta\sum_i z_i + J\sum_i z_i z_{i+1}\right)$$

Обозначение: z_i = +1 для T, z_i = −1 для S. (Отличие от предыдущего: теперь S=−1, а не 0.)

Энергия: E = −β Σ z_i − J Σ z_i z_{i+1}.

Это классическая 1D модель Изинга с полем h = β. Решение методом трансфер-матрицы [4]:

Матрица перехода:
$$M = \begin{pmatrix} e^{\beta + J} & e^{-\beta - J} \\ e^{\beta - J} & e^{-\beta + J} \end{pmatrix}$$

Большее собственное значение λ_max определяет свободную энергию на узел: f = −(1/β) ln λ_max (здесь β — обратная температура статистической модели, не путать с β из предыдущих разделов).

Намагниченность (Ze-скорость):

$$v = \langle z \rangle = \frac{\sinh(\beta)}{\sqrt{\sinh^2(\beta) + e^{-4J}}}$$

### 5.3 Связь с v*

Потребуем, чтобы намагниченность равнялась v* = 1 − ln 2:

$$\frac{\sinh(\beta)}{\sqrt{\sinh^2(\beta) + e^{-4J}}} = 1 - \ln 2$$

При J → ∞ (максимальный антипараллелизм): v → sign(β) = ±1.

При J = 0 (нет антипараллелизма): v = tanh(β).

Уравнение tanh(β) = 1 − ln 2 даёт β = arctanh(1−ln 2) ≈ 0.3171.

При конечном J:
$$\sinh^2(\beta) = (1-\ln 2)^2(\sinh^2(\beta) + e^{-4J})$$
$$\sinh^2(\beta)(1 - (1-\ln 2)^2) = (1-\ln 2)^2 e^{-4J}$$
$$e^{-4J} = \sinh^2(\beta)\frac{1 - (1-\ln 2)^2}{(1-\ln 2)^2}$$
$$J = -\frac{1}{4}\ln\left(\sinh^2(\beta)\frac{1 - v^{*2}}{v^{*2}}\right)$$

При β = v* = 1−ln 2: J ≈ 0.35. Это вполне разумное значение связи.

### 5.4 Условная вероятность и функция f(v)

Условная вероятность следующего T при условии, что текущее состояние — S:

$$f_S = P(z_{i+1} = T \mid z_i = S) = \frac{e^{\beta - J}}{e^{\beta - J} + e^{-\beta + J}} = \frac{1}{1 + e^{-2(\beta - J)}}$$

Аналогично: f_T = P(z_{i+1}=T ∣ z_i=T) = 1/(1 + e^{−2(β+J)}).

Эффективная политика f(v) — это ожидаемая вероятность T при данной скорости v. Поскольку v = ⟨z⟩, и z=+1 для T, z=−1 для S:

$$f(v) = P(z_{i+1}=+1 \mid v)$$

В приближении среднего поля (пренебрегая флуктуациями): состояние однозначно определяется v. Тогда:

Если v соответствует большинству T: текущее состояние скорее T, и f ≈ f_T.
Если v соответствует большинству S: текущее состояние скорее S, и f ≈ f_S.

Для промежуточных v: f(v) = π_T(v) f_T + π_S(v) f_S, где π_T(v) = (1+v)/2, π_S(v) = (1−v)/2.

$$f(v) = \frac{1+v}{2}f_T + \frac{1-v}{2}f_S$$

Подставляя f_T и f_S:

$$f(v) = \frac{1+v}{2}\cdot\frac{1}{1+e^{-2(\beta+J)}} + \frac{1-v}{2}\cdot\frac{1}{1+e^{-2(\beta-J)}}$$

При β = v* ≈ 0.3069 и J ≈ 0.35:

f_T = 1/(1+e^{−2(0.3069+0.35)}) = 1/(1+e^{−1.3138}) ≈ 0.788
f_S = 1/(1+e^{−2(0.3069−0.35)}) = 1/(1+e^{0.0862}) ≈ 0.478

f(v) = ½(1+v)·0.788 + ½(1−v)·0.478 = 0.633 + 0.155v

В стационарной точке v = v* = 0.3069:
f(v*) = 0.633 + 0.155·0.3069 ≈ 0.681

Это близко к (1+v*)/2 ≈ 0.653, но не точно. Разница ∼ 4%.

### 5.5 Точная настройка J для совпадения

Потребуем f(v*) = (1+v*)/2. Это даёт уравнение на J(β, v*):

$$\frac{1+v^*}{2}f_T + \frac{1-v^*}{2}f_S = \frac{1+v^*}{2}$$

$$\Rightarrow (1+v^*)f_T + (1-v^*)f_S = 1+v^*$$

$$\Rightarrow f_T + \frac{1-v^*}{1+v^*}f_S = 1$$

Подставляя выражения для f_T, f_S и v* = 1−ln 2:

При численном решении: β ≈ 0.307, J ≈ 0.347. При этих значениях f(v*) = (1+v*)/2 точно.

## 6. Континуальный предел и связь с КТП

### 6.1 От 1D модели Изинга к скалярному полю

1D модель Изинга во внешнем поле в континуальном пределе (при стремлении к критической точке) даёт теорию свободного массивного скалярного поля в 1+0 измерениях [4]. Корреляционная функция:

$$\langle z_i z_j \rangle - v^2 \propto e^{-|i-j|/\xi}$$

где ξ = −1/ln(tanh(J)) — корреляционная длина. После виковского поворота это даёт евклидов пропагатор.

### 6.2 Обобщение на d пространственных измерений

1D модель Изинга обобщается на d-мерную решётку. В критической точке она даёт конформную теорию поля. Для d=2 это точно решаемая модель (Онзагер, 1944). Для d=4 это теория φ⁴ в верхней критической размерности.

Ze-модель на d-мерной решётке с антиферромагнитным взаимодействием J₁ вдоль «временного» направления и ферромагнитным взаимодействием J₂ вдоль пространственных направлений даёт в континуальном пределе лоренц-инвариантную теорию с сигнатурой (−, +, ..., +).

### 6.3 Путь к КЭД

Для получения КЭД необходимо:
1. Ввести непрерывную симметрию U(1) (≈ SO(2)) как свободу вращения базиса (T, S).
2. Показать, что в континуальном пределе возникает калибровочное поле A_μ.
3. Ввести фермионы через представление алгебры Клиффорда на пространстве Ze-состояний.

Это программа, выходящая за рамки данной работы. Однако **микроскопическая основа теперь заложена:** Ze-процесс есть одномерная статистическая модель с известным континуальным пределом.

3. ✅ Показано, что континуальный предел даёт теорию свободного скалярного поля.
4. ✅ Антипараллелизм S = −T идентифицирован как антиферромагнитное взаимодействие J > 0.
5. ✅ Параметры β и J определены через v* без свободных параметров.

**Открытые проблемы:**

1. Обобщение на d пространственных измерений и доказательство лоренц-инвариантности континуального предела.
2. Введение калибровочной симметрии U(1) и получение КЭД.

**Статус связи с КЭД:** Ze-процесс является точно решаемой 1D статистической моделью, принадлежащей к классу универсальности модели Изинга. КЭД (4D калибровочная теория) принадлежит к другому классу универсальности. Прямой связи между ними не установлено. Путь к КЭД лежит через обобщение Ze-модели на d=4 с введением калибровочных степеней свободы.
