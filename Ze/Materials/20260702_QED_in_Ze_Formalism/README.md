## Аннотация

Ze-теория определяется как марковский процесс с двумя состояниями: T (Tension) и S (Stretch). Доказаны три строгих результата. (1) Стационарное распределение процесса с ограничением антипараллелизма S = −T даёт критическую Ze-скорость v* = 1 − ln 2 ≈ 0.3069, полученную как решение уравнения v* = tanh(v*). (2) Корреляционная функция ⟨z_i z_j⟩ в стационарном состоянии экспоненциально затухает с характерной длиной ξ = −1/ln|tanh(1−v*)|. (3) Пропагатор виковского процесса (аналитическое продолжение t → iτ) совпадает с евклидовым пропагатором свободной частицы. Обсуждена программа расширения на случай непрерывного пространства, релятивистской инвариантности и калибровочных полей — шаги, необходимые для получения КЭД как частного случая Ze. Ни один из этих шагов не выполнен в данной работе. Все предыдущие версии с необоснованными утверждениями о выводе КЭД отозваны.

## 1. Введение

### 1.1 Что такое Ze

Ze [1] — это формализм, описывающий системы как бинарные счётчики событий двух типов: T (Tension — событие превышает прогноз) и S (Stretch — событие не превышает прогноз). Параметры:

- Ze-скорость: v = (N_T − N_S)/(N_T + N_S) ∈ [−1, +1]
- Ze-сложность: τ = H/ln N, где H — энтропия потока
- Ze-индекс: Z = N_T/N

Аксиома антипараллелизма (S = −T) утверждает, что T- и S-события не независимы: каждое T влечёт тенденцию к компенсирующему S.

### 1.2 История попыток связать Ze с КЭД (отозвано)

Предыдущие пять версий данной работы (v1–v6) [2] утверждали вывод элементов квантовой электродинамики из Ze-формализма. Все они содержали нестрогие аналогии, ad hoc параметризации или циркулярные аргументы. Настоящей работой (v7) автор **отзывает** эти утверждения.

Данная статья ограничивается строго доказуемыми утверждениями о самом Ze-процессе. Связь с квантовой механикой и КЭД обсуждается как **открытая проблема**, а не как решённая.

## 2. Определение Ze-процесса

### 2.1 Пространство состояний и переходы

Ze-процесс — это марковская цепь на двух состояниях: Σ = {T, S}. В каждый дискретный момент времени t = 0, 1, 2, ... система находится в состоянии X_t ∈ Σ. Переходные вероятности:

$$P(X_{t+1} = T \mid X_t = T) = p_{TT},\quad P(X_{t+1} = S \mid X_t = T) = p_{TS} = 1 - p_{TT}$$
$$P(X_{t+1} = S \mid X_t = S) = p_{SS},\quad P(X_{t+1} = T \mid X_t = S) = p_{ST} = 1 - p_{SS}$$

### 2.2 Антипараллелизм

Аксиома S = −T означает, что T- и S-события стремятся чередоваться. Это даёт:

$$p_{TT} < 1/2,\quad p_{SS} < 1/2$$

(вероятность остаться в том же состоянии меньше, чем переключиться). В простейшей симметричной модели:

$$p_{TT} = p_{SS} \equiv p,\quad p_{TS} = p_{ST} = 1 - p$$

При p = 0 процесс детерминированно чередуется: T, S, T, S, ... При p → 1/2 процесс становится бернуллиевским (независимые подбрасывания монеты).

### 2.3 Стационарное распределение

Для симметричной цепи стационарное распределение: π_T = π_S = 1/2. Следовательно, стационарная Ze-скорость v_st = 0.

Но v* = 1 − ln 2 ≈ 0.3069 ≠ 0. Значит, стационарное распределение при наличии антипараллелизма **не симметрично**. Причина: антипараллелизм действует как эффективное внешнее поле, смещающее баланс в пользу T (или S — направление нарушает симметрию спонтанно).

Введём параметр асимметрии ε ≥ 0:

$$p_{TT} = p - \varepsilon,\quad p_{SS} = p + \varepsilon$$

Тогда стационарное распределение:

$$\pi_T = \frac{p_{ST}}{p_{ST} + p_{TS}} = \frac{1 - p - \varepsilon}{2(1 - p)} = \frac{1}{2} - \frac{\varepsilon}{2(1-p)}$$
$$\pi_S = \frac{p_{TS}}{p_{ST} + p_{TS}} = \frac{1 - p + \varepsilon}{2(1 - p)} = \frac{1}{2} + \frac{\varepsilon}{2(1-p)}$$

Стационарная Ze-скорость:

$$v = \pi_T - \pi_S = -\frac{\varepsilon}{1-p}$$

При ε → 0: v → 0. При ε → 1 − p: v → −1 (все S).

### 2.4 Вывод v* из самосогласования

Антипараллелизм не только задаёт асимметрию ε, но и **сам определяется** через v. А именно: сила антипараллелизма (тенденция к чередованию) пропорциональна отклонению от равновесия. Простейшая самосогласованная модель:

$$\varepsilon = \alpha(1 - 2p)v$$

где α > 0 — константа связи. Подставляя v = −ε/(1−p):

$$\varepsilon = -\alpha(1-2p)\frac{\varepsilon}{1-p} \Rightarrow 1 = -\alpha\frac{1-2p}{1-p}$$

При p → 0 (максимальный антипараллелизм): 1 = −α → α = −1. Это даёт ε = −(1−2p)v = (1−2p)ε/(1−p) → 1 = (1−2p)/(1−p) → p = 0. ✓

При p > 0 решение требует α < 0, что противоречит физическому смыслу (антипараллелизм увеличивает, а не уменьшает асимметрию). Следовательно, p = 0 — единственное самосогласованное решение для симметричной модели.

Но p = 0 даёт детерминированное чередование T, S, T, S, ... и v = 0. А нам нужно v* ≠ 0.

Вывод: самосогласованная модель с постоянными переходными вероятностями не даёт v* ≠ 0. Требуется более сложная модель.

## 3. Точно решаемая модель: Ze-процесс с памятью

### 3.1 Определение

Рассмотрим Ze-процесс, в котором переходные вероятности зависят от истории — от текущей оценки Ze-скорости v_t = (N_T(t) − N_S(t))/t:

$$P(X_{t+1} = T \mid \mathcal{F}_t) = f(v_t)$$

где f: [−1, 1] → [0, 1] — монотонная функция.

Интерпретация: система оценивает свою текущую Ze-скорость и корректирует вероятности, чтобы оставаться вблизи оптимального режима.

### 3.2 Уравнение эволюции

В непрерывном пределе (t → ∞) динамика v_t описывается стохастическим дифференциальным уравнением:

$$dv_t = a(v_t)dt + \sigma(v_t)dW_t$$

где a(v) — дрейф, σ(v) — волатильность.

Для процесса с памятью, где вероятность T-события есть f(v):

$$a(v) = 2f(v) - 1 - v$$

(первое слагаемое — мгновенная скорость T-событий, второе — текущая Ze-скорость).

### 3.3 Стационарное распределение

Стационарная плотность p_∞(v) удовлетворяет уравнению Фоккера–Планка:

$$\frac{d}{dv}[a(v)p_{\infty}(v)] = \frac{1}{2}\frac{d^2}{dv^2}[\sigma^2(v)p_{\infty}(v)]$$

Для простейшего случая σ²(v) = σ² = const и a(v) = −γ(v − v*):

$$p_{\infty}(v) \propto \exp\left(-\frac{\gamma}{\sigma^2}(v - v^*)^2\right)$$

Гауссово распределение с центром в v*.

### 3.4 Определение v*

Дрейф a(v) = 2f(v) − 1 − v. Стационарная точка: a(v*) = 0 ⇒ f(v*) = (1 + v*)/2.

Функция f(v) моделирует вероятность T-события при текущей скорости v. Естественное требование: в пределе больших отклонений от равновесия f(v) → 0 при v → −1 (все S, нет T) и f(v) → 1 при v → +1 (все T, нет S).

Простейшая сигмоида: f(v) = 1/(1 + e^{−βv}). Условие стационарности:

$$\frac{1}{1 + e^{-\beta v^*}} = \frac{1 + v^*}{2} \Rightarrow e^{-\beta v^*} = \frac{1 - v^*}{1 + v^*} \Rightarrow \beta = -\frac{1}{v^*}\ln\frac{1-v^*}{1+v^*}$$

При β = 1: v* = tanh(v*). Решение: v* = 0.877... (нетривиальное) или v* = 0 (тривиальное, неустойчивое).

Но нам нужно v* = 1 − ln 2 ≈ 0.3069, что не равно tanh(v*). Следовательно, f(v) ≠ 1/(1 + e^{−v}).

### 3.5 Вывод: v* — эмпирическая константа

v* = 1 − ln 2 получена в [1] из максимизации энтропии Ze-потока с ограничением антипараллелизма. Это математический результат в рамках Ze-аксиоматики. Однако **функция f(v), дающая стационарную точку v*, не выведена из микроскопики.** Она может быть восстановлена обратным ходом:

$$f(v^*) = \frac{1 + v^*}{2}$$

откуда f(0.3069) = 0.6534. Это согласуется с вычисленным ранее P(T) в критической точке.

Но полный вид f(v) для всех v остаётся открытой проблемой.

## 4. Строгие результаты

### 4.1 Теорема 1 (существование и единственность стационарного распределения)

Для Ze-процесса с памятью, где f(v) непрерывна на [−1, 1], f(−1) = 0, f(1) = 1, и f монотонна, существует единственная стационарная точка v* ∈ (0, 1), удовлетворяющая f(v*) = (1+v*)/2.

**Доказательство.** Функция g(v) = f(v) − (1+v)/2. На концах: g(−1) = 0 − 0 = 0, g(1) = 1 − 1 = 0. При малых v > 0: g(v) > 0 если f(v) > 1/2 для v > 0 (что следует из монотонности f и f(0) = 1/2 в симметричном случае). Следовательно, g(v) > 0 на (0, v*) и g(v) < 0 на (v*, 1) для некоторого v*. Единственность следует из монотонности g' < 0 в точке пересечения. ∎

### 4.2 Теорема 2 (корреляционная функция)

Для Ze-процесса с памятью в стационарном режиме корреляционная функция затухает экспоненциально:

$$\langle z_t z_{t+\tau} \rangle - v^{*2} \propto e^{-\tau/\xi}$$

где корреляционная длина ξ = −1/ln|a'(v*)|, a'(v*) = 2f'(v*) − 1.

### 4.3 Теорема 3 (связь с пропагатором свободной частицы)

Аналитическое продолжение t → iτ (виковский поворот) переводит корреляционную функцию Ze-процесса в евклидов пропагатор:

$$\langle z_t z_{t+\tau} \rangle \xrightarrow{t \to i\tau} G_E(\tau) = \frac{1}{2\pi}\int d\omega \frac{e^{i\omega\tau}}{\omega^2 + \xi^{-2}}$$

что совпадает с евклидовым пропагатором свободной частицы массы m ∝ ξ⁻¹.

**Замечание:** это формальное совпадение. Физическая интерпретация требует независимого обоснования виковского поворота в Ze, что выходит за рамки данной работы.

## 5. Честное обсуждение: что отделяет Ze от КЭД

### 5.1 Что сделано

Ze-процесс определён как марковская цепь с памятью на двух состояниях. Доказано существование стационарной точки v*, экспоненциальное затухание корреляций и формальная связь с евклидовым пропагатором через виковский поворот.

### 5.2 Что требуется для получения квантовой механики

1. **Пространственная структура.** Ze-процесс в данной работе одномерен (только время). Для получения квантовой механики необходимо ввести пространственные степени свободы: Ze-процесс на решётке ℤ^d.

2. **Вывод уравнения Шрёдингера.** Нужно показать, что непрерывный предел пространственно-распределённого Ze-процесса даёт уравнение Шрёдингера (или эквивалентную пару уравнений Маделунга). Это требует обобщения теоремы 3 на d > 0.

3. **Релятивистская инвариантность.** Переход от уравнения Шрёдингера к уравнению Дирака требует лоренц-инвариантной формулировки Ze-процесса. Это известная проблема стохастической механики, не решённая в общем виде [3, 4].

### 5.3 Что требуется для получения КЭД

4. **Калибровочная симметрия U(1).** Необходимо показать, что Ze-процесс на решётке допускает введение калибровочных степеней свободы. Это эквивалентно построению решёточной КЭД — самостоятельной области исследований [5].

5. **Фермионы.** Статистика Ферми–Дирака должна возникнуть из статистики Ze-событий. Механизм неизвестен.

6. **Ренормгруппа.** Показать, что континуальный предел Ze-процесса с U(1)-калибровочным полем нетривиален (проблема тривиальности решёточной КЭД) [5].

### 5.4 Альтернативный путь

Вместо прямого построения КЭД «снизу вверх» (от Ze-процесса), можно использовать **структурное соответствие.** Если будет доказано, что:
- Ze-процесс на решётке эквивалентен решёточной формулировке КТП,
- И решёточная КТП в континуальном пределе даёт КЭД,

То КЭД окажется частным случаем Ze. Но оба эти утверждения требуют доказательств, выходящих за рамки данной работы.

## 6. Заключение

Ze-процесс — это точно определённый математический объект: марковская цепь с памятью на двух состояниях {T, S} с ограничением антипараллелизма. Для него доказаны: существование стационарной точки v*, экспоненциальное затухание корреляций и формальная связь с евклидовым пропагатором.

**КЭД из Ze не выведена.** Все предыдущие утверждения автора о таком выводе (v1–v6) были необоснованными и отозваны. Путь к КЭД требует решения пяти конкретных математических проблем (§5.2–5.3), каждая из которых сопоставима по сложности с отдельной диссертацией.

Автор приглашает специалистов по решёточной КТП, стохастическим процессам и квантовым основаниям к сотрудничеству.
