## Observations

1. **Mito counter dominates** across all cycling tissues in the first 700 days (τ_mito = 14 days is short relative to 700 days of elapsed time). This is consistent with the a-priori parameter design where mito is time-dominant (α = 0, β = 0.010).
2. **Neuron is a clear outlier** — all five counters lower than cycling tissues. Driver: τ_mito = 30 days for neurons (vs 14 days), so accumulation per step is halved; additionally α_tel → 0 and α_cent → 0 for this post-mitotic tissue.
3. **Tissue-load ranking** (highest → lowest): hepatocyte > CD8 T memory > HSC > β-cell > fibroblast > neuron. Driven by w_mito: hepatocyte w_mito = 0.30 is the highest; neuron despite w_mito = 0.35 is still lowest because its D_mito is half.
4. **No tissue exceeds L_crit = 0.60** within 100 divisions — expected for 700-day simulation (~ equivalent to 2 years human time for a 7-day cycling tissue). Running beyond 150 divisions should start triggering some L_crit crossings for hepatocyte.
5. **Telomere counter in fibroblast (0.120)** is the highest telomere value, consistent with a-priori *w_tel(fibroblast) = 0.40* and α_tel = 0.020.
