## Description

MitoROS is a project focused on mitochondrial reactive oxygen species (ROS) signaling and its role in cellular metabolism and aging. The project aims to develop a computational model of ROS dynamics, validate it with experimental data, and explore therapeutic interventions. This project is part of the LC/MCAOA research program.
