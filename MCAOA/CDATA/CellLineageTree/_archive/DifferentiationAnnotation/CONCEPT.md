# DifferentiationAnnotation: Conceptual Framework

## §1 Purpose
Within Dr. Jaba Tqemaladze's CytogeneticTree project, the overarching goal is to reconstruct the complete genealogical tree of cellular differentiation from zygote to terminally differentiated cells by leveraging the asymmetric age of mother and daughter centrioles as a permanent lineage barcode. The **DifferentiationAnnotation** sub-subproject serves a critical interpretative function: it maps the raw lineage-directed acyclic graph (DAG), generated by tracking centriole age, onto biologically meaningful differentiation states. Its purpose is to transform a topological tree of "who begat whom" into an annotated tree of "what became what," where each node (cell) is labeled with its inferred cell state based on molecular and morphological signatures. This annotation is essential for understanding the timing, sequence, and potential bifurcation points of fate decisions embedded within the lineage tree.

## §2 Mechanism/Basis
The mechanism operates as a multi-modal data integration pipeline. The input is a lineage DAG where each node possesses a unique centriole-age-based identifier and temporal/spatial coordinates. Annotation proceeds via a rule-based and probabilistic fusion of orthogonal data streams:
1. **scRNA-seq Marker Integration:** Single-cell RNA sequencing data from matched samples provides a high-dimensional transcriptomic profile. Cells within the lineage DAG are projected into this space using spatial/temporal registration or direct lineage barcode capture (e.g., intronic sequencing). Canonical marker genes for differentiation states are used to assign probabilistic labels (e.g., neural progenitor, cardiomyocyte precursor).
2. **Morphology Feature Mapping:** High-resolution imaging data (e.g., light-sheet microscopy) provides cellular and nuclear morphology, cell volume, and neighbor topology. Pre-trained classifiers correlate specific morphological signatures with known cell states, offering an annotation layer independent of transcriptomics.
3. **Immunofluorescence (IF) Anchor Points:** Sparse but definitive IF staining for key lineage-specific proteins (e.g., transcription factors, structural proteins) provides ground-truth anchor nodes. These anchors constrain and validate the annotations propagated from scRNA-seq and morphology through the lineage structure.

The core algorithmic challenge is the **constrained propagation** of state labels across the DAG. If a parent cell is annotated with high confidence, its daughter cells' possible states are constrained by known biological rules (e.g., a hepatoblast cannot directly produce a neuron). The lineage tree provides a powerful scaffold for smoothing and disambiguating state transitions that appear noisy in snapshot data. However, we acknowledge that dedifferentiation events (Tata et al., 2013, PMID: 24209622; Tata & Rajagopal, 2016, PMID: 27716511; Choi et al., 2021, PMID: 34587533) may violate strict DAG constraints; our model explicitly accounts for this by allowing probabilistic reversion of state labels along edges where dedifferentiation is inferred from molecular signatures. ## §3 State of the Art
Current methods for cell state annotation are largely divorced from lineage context. They operate on dissociated cells, losing the spatial and genealogical relationships that are critical for accurate fate mapping. This limitation is well-documented in benchmark studies (Luecken et al., 2022, PMID: 35027767) and motivates our lineage-aware approach. fundamental to understanding differentiation dynamics.
* **scRNA-seq Clustering & Annotation:** Standard pipelines (e.g., Seurat, Scanpy) cluster cells by transcriptomic similarity and annotate clusters using reference databases. These methods infer *pseudotime* but cannot reconstruct actual lineage trees (reference pending — general Seurat/Scanpy reviews; not verified here).
* **Lineage Tracing with Molecular Barcodes:** Techniques like CRISPR-Cas9 barcoding (GESTALT, McKenna et al. 2016 Science, [PMID: 27229144]; scGESTALT, Raj et al. 2018 Nat Biotech, [PMID: 29608178]) can reconstruct lineages but typically lack simultaneous deep transcriptional phenotyping at single-cell resolution for all barcoded clones.
* **Integrated Imaging & Sequencing:** Emerging spatial transcriptomics methods (MERFISH, seqFISH+) provide gene expression *in situ* but are limited in gene throughput and cannot yet trace deep historical lineages (reference pending — specific method paper not verified here).

**DifferentiationAnnotation** uniquely sits at the convergence of these approaches, using the intrinsic centriole-age lineage barcode to provide the temporal framework for integrating the deepest available molecular and morphological phenotyping.

## §4 Integration with Other CytogeneticTree Technologies
This subproject is a central hub for data synthesis:
* **Input from `CentrioleLineageTracer`:** Consumes the raw lineage DAG (nodes, edges, timestamps).
* **Input from `SingleCellBiopsySequencing`:** Receives transcriptomic profiles linked to specific lineage nodes via captured centriole identifiers or spatial mapping.
* **Input from `LiveImagingMorphometry`:** Ingests time-lapsed morphological feature vectors for each tracked cell.
* **Output to `FateDecisionMap`:** Provides the annotated lineage tree, which is the primary input for identifying bifurcation points and modeling fate decision logic.
* **Output to `PhylogeneticReconstructionAlgos`:** Supplies state labels that can be used as character states for phylogenetic analysis of differentiation pathways.

## §5 Gaps & What to Build
Key gaps exist between the current state of art and the fully realized **DifferentiationAnnotation** system:
1. **Spatio-Temporal Registration Engine:** No existing tool robustly aligns a live-imaging-derived lineage DAG with endpoint scRNA-seq data from dissociated cells. A probabilistic registration model must be built, potentially using the few IF anchor points as landmarks.
2. **Lineage-Aware Annotation Propagator:** Current cell annotation tools are context-free. A novel algorithm is needed that propagates state labels across a DAG, respecting lineage relationships, temporal order, and permissible state transition rules (a "differentiation grammar").
3. **Multi-Modal Confidence Integrator:** A framework to weigh and fuse conflicting evidence from transcriptomics, morphology, and IF, assigning a confidence score to each node's annotation is required.
**The build** must therefore focus on these three core software components: a Registration Engine, a Constrained Label Propagator, and a Confidence Integrator, forming the MVP of the annotation pipeline.

## Limitations

This approach has several inherent limitations that must be acknowledged:

1. **Microscopy resolution dependence:** The accuracy of centriole age tracking is fundamentally limited by imaging resolution. Sub-diffraction limit structures may be misassigned, particularly in densely packed tissues or during rapid mitotic events.

2. **Cell loss bias:** Any cell death, detachment, or loss during imaging or processing creates gaps in the lineage DAG. Missing nodes can lead to incorrect parent-child assignments and propagate annotation errors downstream.

3. **Gradient marker ambiguity:** Many differentiation markers exhibit continuous expression gradients rather than discrete on/off states. Threshold-based annotation may misclassify intermediate or transitional states, particularly at bifurcation points.

4. **Centriole age persistence assumption:** The core premise that centriole age asymmetry is universally preserved through asymmetric cell division has not been directly validated across all cell types and developmental contexts. Symmetric divisions or centriole replacement events could erase the barcode.

5. **Multi-modal registration challenges:** Aligning scRNA-seq, imaging, and IF data from different samples or modalities introduces batch effects and registration errors that may confound state assignments.

6. **Computational scalability:** Constrained propagation across large DAGs (potentially millions of nodes) requires efficient algorithms; current placeholder implementations may not scale to whole-organism reconstructions.

## Evidence base & meta-analysis

**Key claims and supporting evidence:**

1. **Centriole age as lineage barcode:**
   - McKenna et al. 2016 (PMID: 27229144) demonstrated CRISPR-based lineage tracing in zebrafish, establishing that permanent barcodes can record developmental history.
   - Raj et al. 2018 (PMID: 29608178) extended this to single-cell resolution (scGESTALT), showing barcode capture from transcriptomes.
   - *Note:* Direct evidence that centriole age asymmetry functions as a permanent barcode remains to be established. No systematic review or meta-analysis currently exists for this specific claim.

2. **Multi-modal integration for cell state annotation:**
   - Seurat v4 (Hao et al. 2021, PMID: 34062119) provides weighted-nearest neighbor integration across modalities.
   - Scanpy (Wolf et al. 2018, PMID: 29409532) offers scalable single-cell analysis framework.
   - *Gap:* No published study has integrated centriole-age lineage information with transcriptomic and morphological data for state annotation.

3. **Constrained propagation through DAG:**
   - Lineage tree inference methods (e.g., Cassa et al. 2023, PMID: [TBD — placeholder, to be replaced with valid PMID prior to submission]) use phylogenetic constraints for cell state prediction.
   - *Contradictory evidence:* Some studies suggest that differentiation is not strictly tree-like but involves plasticity and dedifferentiation (e.g., Tata et al. 2013, PMID: 24209622), which would violate the DAG assumption.

**State-of-the-art summary:**
Current cell annotation methods (Seurat, Scanpy, CellTypist) operate on dissociated cells without lineage context. No existing tool incorporates centriole-age-based lineage information. The proposed approach is novel but requires validation against established methods on benchmark datasets.

**Systematic review status:** A PRISMA-compliant systematic review of lineage tracing methods is planned but not yet conducted. Cochrane reviews are not applicable to this methodology-focused project.

## Methodology depth

**Step-by-step replication protocol (for independent replication):**

1. **Data acquisition:**
   a. Obtain time-lapse imaging data of developing tissue (e.g., zebrafish embryo) with centriole markers (e.g., Centrin-GFP).
   b. Perform scRNA-seq on matched developmental timepoints.
   c. Collect IF staining for 3+ lineage-specific markers at terminal timepoint.

2. **Lineage DAG construction:**
   a. Track centriole inheritance through cell divisions using automated tracking software (e.g., TrackMate, CellProfiler).
   b. Assign unique centriole-age identifiers to each cell.
   c. Validate tracking accuracy on 1% of paths manually.

3. **Multi-modal registration:**
   a. Align scRNA-seq data to imaging coordinates using spatial transcriptomics methods (e.g., Seurat's TransferData).
   b. Project IF anchor points onto the DAG.

4. **State annotation:**
   a. Train random forest classifier on IF anchor nodes using scRNA-seq and morphology features.
   b. Apply constrained propagation algorithm (custom Python implementation) to propagate labels through DAG.
   c. Evaluate accuracy on held-out IF nodes.

**Statistical Analysis Plan (SAP):**
- **Primary endpoint:** Annotation accuracy (proportion of correctly labeled nodes vs. IF ground truth).
- **Secondary endpoints:** (i) Correlation between centriole age asymmetry and differentiation state; (ii) Consistency of annotations across biological replicates.
- **Multiple comparison correction:** Bonferroni correction for 3 primary comparisons (α = 0.05/3 = 0.017).
- **Missing data handling:** Complete-case analysis for primary endpoint; sensitivity analysis using multiple imputation (MICE) for missing nodes.

**Controls:**
- Positive control: Simulated DAG with known ground-truth labels (generated from known differentiation trajectories).
- Negative control: Randomly permuted labels (to ensure algorithm does not overfit DAG structure).
- Technical control: Split data by imaging batch and compare results.

**Replication strategy:**
- Internal: 5-fold cross-validation on held-out IF nodes.
- External: Independent dataset from different tissue/organism (if available).

**Blinding/Randomisation:**
- For in silico analysis: Not applicable (computational pipeline is deterministic).
- For any future in vivo validation: Random allocation of embryos to treatment/control groups; blinded assessment of imaging data by second researcher.
