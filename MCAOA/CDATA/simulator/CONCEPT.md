# CONCEPT.md — CDATA-v2 Simulator

## Название
**CDATA-v2: Стохастическая, ABC-SMC-калиброванная, GSA-валидированная модель центриоль-опосредованного истощения стволовых клеток с двойной Aurora A-опосредованной инактивацией p53**

## Автор
Jaba Tqemaladze, MD — Georgia Longevity Alliance

## Проблема
Истощение пула стволовых клеток — фундаментальный механизм старения. Центриолярный аппарат играет ключевую роль в регуляции клеточного цикла и ответе на повреждения, но количественная модель, связывающая динамику центриолей с истощением стволовых клеток, отсутствовала.

## Решение
CDATA-v2 — стохастическая симуляционная модель, моделирующая:
- Динамику центриолярного аппарата (материнские/дочерние центриоли)
- Два пути Aurora A → инактивация p53 (Ser315 деградация + Ser215 транскрипционный сайленсинг)
- ATM → p53(Ser15) → центросома (spindle checkpoint → сенесценция)
- CCP5/AGBL5 деглутамилазная активность → накопление повреждений
- CEP295 → Polo/PLK1 созревание материнской центриоли

## Методология
1. **ABC-SMC калибровка**: 14 параметров против 2400 наблюдений (3 типа клеток × 4 конечные точки)
2. **Sobol GSA**: глобальный анализ чувствительности, идентификация доминантных параметров
3. **Стохастическое моделирование**: метод Монте-Карло, конфигурируемое число прогонов

## Ключевые выводы
- Aurora A → p53(Ser215) — доминантный путь (подтверждено GSA)
- CCP5/AGBL5 — критический параметр, определяющий скорость накопления повреждений
- Модель воспроизводит лимит Хейфлика для трёх типов клеток
