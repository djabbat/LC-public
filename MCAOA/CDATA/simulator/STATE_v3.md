# CDATA v3.0 — Модель с CCI, CML-дифференциацией и PCM-трафиком

**Статус:** Ядро модели обновлено (sim_core/src/centriole/mod_v2.rs). Интеграция в Python-симулятор — следующий этап.

## Что нового в v3.0 (относительно v2.0)

| v2.0 | v3.0 |
|------|------|
| Единая «энтропия» | CML_structural + CML_signaling (net CML) |
| Нет CCI | CCI = L_centriole / L_cilium |
| Нет динамики длины | Удлинение центриоли + укорочение цилии |
| Фиксированная CCP | A_CCP(t) = A_CCP(0)·exp(-t/τ_CCP) |
| Нет PCM-трафика | T_PCM = f(CML_struct) с обратной связью |
| Нет тканевой специфичности | TissueParams: 4 пресета |
| Нет различения мать/дочь | CentrioleType: Mother, Daughter, Procentriole |
| Простой порог сенесценции | P_sen = f(CML_net, T_PCM, CCI) — комбинированный |
| Нет PFT-α контроля | P7: параллельный эксперимент с ингибитором p53 |

## План интеграции

1. Реализовать `CentrioleState` из `mod_v2.rs` в Python (`cdata_v3/model.py`)
2. Добавить `TissueParams` с пресетами
3. Обновить ABC-SMC под 20+ параметров
4. Добавить CCI как observable
5. Валидировать против данных Köhrer (2023), Launay (2025)
