# THEORY.md — CDATA-v2 Simulator

## Теоретическая основа

### Центриолярная теория истощения стволовых клеток

CDATA (Centriole-Driven Abatement of Telomeric Aging) — теория, согласно которой центриолярный аппарат является первичным драйвером истощения пула стволовых клеток, а теломеры — вторичным индикатором.

### Ключевые механизмы

#### 1. Aurora A → p53 инактивация (двойной путь)
- **Ser315**: Фосфорилирование p53 по Ser315 ведёт к убиквитинированию и протеасомной деградации p53
- **Ser215**: Фосфорилирование по Ser215 подавляет транскрипционную активность p53 (сайленсинг промоторов p21, BAX, PUMA)
- **Следствие**: Клетки с гиперактивной Aurora A обходят p53-опосредованный arrest

#### 2. ATM → p53(Ser15) → центросома
- Активация ATM при повреждении ДНК → фосфорилирование p53(Ser15)
- p53(Ser15) активирует spindle assembly checkpoint
- При превышении порога → сенесценция

#### 3. CCP5/AGBL5 деглутамилаза
- Деглутамилирование тубулина центриолей
- С возрастом активность CCP5 падает → гиперацетилирование → центросомная амплификация
- Прямая связь с накоплением повреждений

#### 4. CEP295 → Polo/PLK1
- CEP295 необходим для созревания материнской центриоли
- PLK1 — мастер-регулятор митоза
- Нарушение → асимметричное деление → истощение пула

### Математический формализм
- Стохастические дифференциальные уравнения (СДУ)
- Метод Монте-Карло для симуляции деревьев
- ABC-SMC: approximate Bayesian computation with sequential Monte Carlo
- Sobol GSA: decomposition of variance → S1, ST индексы

### Предсказания модели
1. Медианный лимит Хейфлика: 45-55 поколений
2. Доминантный параметр: Aurora A → p53(Ser215) (ST > 0.4)
3. CCP5 — критический возраст-зависимый параметр
