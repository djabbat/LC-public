## 🔴 ПРОБЛЕМА 1 (КРИТИЧЕСКАЯ): Асимметрия падает экспоненциально

### Формула
```python
A_g = state.A * sigma_D * sigma_S * sigma_F * sigma_N
```

### Что происходит
Асимметрия `A` умножается на ПРОИЗВЕДЕНИЕ четырёх сигмоид каждый цикл. Это **экспоненциальный спад**:

A(n) = A(0) × (sigma_D × sigma_S × sigma_F × sigma_N)^n

При sigma ≈ 0.97 каждая (что уже идеальные значения для здоровой клетки):
- За 10 поколений: 0.8 × 0.97^10 = 0.59
- За 20 поколений: 0.8 × 0.97^20 = 0.43
- За 40 поколений: 0.8 × 0.97^40 = 0.23

**Даже идеально здоровая клетка теряет способность к асимметричному делению за 20-25 поколений.**

### Как исправить (варианты)

**Вариант A — изменить compounding на затухание:**
```python
A_g = state.A * (1.0 - delta_A)  # где delta_A = f(D, M_s, M_f, N_mat)
```
Не умножать четыре сигмоиды, а вычитать одну дельту. Тогда A падает линейно, а не экспоненциально.

**Вариант B — убрать сигмоиды, оставить только N_mat:**
```python
A_g = state.A * self._sigma_N(new_N_mat)  # Только N_mat влияет на асимметрию
```
Сигмоиды по D, M_s, M_f убрать из A. Они и так влияют через dM/dt.

**Вариант C — применить логарифмическую свёртку:**
```python
log_A = np.log(max(state.A, 1e-6))
log_A += np.log(sigma_D) + np.log(sigma_S) + np.log(sigma_F) + np.log(sigma_N)
# Добавить восстанавливающий член
log_A += alpha_recovery  # ~0.02 за поколение
A_g = np.exp(log_A)
```

## 🟠 ПРОБЛЕМА 2: M_s и M_f либо растут, либо падают — нет равновесия

### Формулы
```python
dM_s = -(mu_s + eta_s*D)*M_s + alpha_CEP - omega*D*M_s
dM_f = -(mu_f + eta_f*D)*M_f + beta_0 * M_s * P
```

### Что происходит
При одних параметрах (alpha_CEP > mu_s*M_s) — **M_s неограниченно растёт** (1.0 → 5.0 за 50 поколений).
При других (alpha_CEP < mu_s*M_s) — **M_s неограниченно падает** до 0.

**Нет гомеостаза.** В реальной клетке есть feedback-петли, удерживающие M в диапазоне.

### Как исправить

**Добавить нелинейное насыщение:**
```python
dM_s = (-(mu_s + eta_s*D)*M_s + alpha_CEP * (1.0 - M_s/M_max) - omega*D*M_s) * dt
```
где `M_max` — максимально возможный уровень maintenance (~2.0-3.0).

Или **добавить ауторегуляцию:**
```python
dM_s = (-(mu_s + eta_s*D)*M_s + alpha_CEP/(1.0 + M_s/K_s) - omega*D*M_s) * dt
```
(синтез падает при высоком M_s — классическая Михаэлис-Ментен).

## 🟡 ПРОБЛЕМА 3: Повреждения (D) стабилизируются на 0.4-0.8

### Формула
```python
dD = (r_g - lambda_eff * k_cat * D) * dt + noise
```

### Что происходит
При r_0=0.06, lambda_eff≈1.0, k_cat=0.08: стационар D* = r_0/k_cat = 0.75.
**Повреждения никогда не растут выше ~1.0** — модель не показывает возрастное накопление.

### Как исправить

**Сделать k_cat возраст-зависимым:**
```python
lambda_eff = np.exp(-p.lambda_age * g)  # УЖЕ ЕСТЬ — но падает слишком медленно
```
Увеличить `lambda_age` (сейчас 0.01 → нужно 0.03-0.05), тогда CCP5 быстрее деградирует с возрастом, и D растёт к 40-50 поколениям.

**Или добавить D_min (неустранимые повреждения):**
```python
dD = (r_g - lambda_eff * k_cat * max(0, D - D_min)) * dt + noise
```

## 🟡 ПРОБЛЕМА 4: Амплификация центриолей нестабильна

### Формула
```python
dN = sigma_N * (1.0 - lambda_eff) * dt + sigma_N * sqrt(N_mat) * noise
```

### Что происходит
Сейчас: детерминированная часть `sigma_N*(1-lambda_eff)` мала (0.02-0.05), шум доминирует. Амплификация либо не происходит совсем, либо взрывается (при sigma_N > 0.5).

### Как исправить

**Сделать пороговой:**
```python
if D > D_threshold and lambda_eff < lambda_crit:
    if random() < p_amplification:
        N_mat += 1
```

Вместо непрерывного dN — дискретное событие амплификации при превышении порога повреждений.

## 🔴 ПРОБЛЕМА 5 (АРХИТЕКТУРНАЯ): Сигмоиды перемножаются — compounding catastrophe

Это объединяет проблемы 1-4. Модель построена на **произведении сигмоид** в трёх местах:
1. `A_g` = A × σ_D × σ_S × σ_F × σ_N — экспоненциальный спад
2. `dM_s` = -(...)×M_s + ... — нет гомеостаза
3. `p_senescence` = 1-exp(-kappa × cumulative × p53) — работает, но cumulative накапливается только при N_mat > 1

### Рекомендуемый редизайн

```
Вместо:
  A *= sigma_D * sigma_S * sigma_F * sigma_N    [мультипликативно]

Сделать:
  delta_A = gamma_D * (D - D_healthy) + gamma_S * (M_s_healthy - M_s) 
          + gamma_F * (M_f_healthy - M_f) + gamma_N * max(0, N_mat - 1)
  A = max(0, A - delta_A)                        [аддитивно]
  
  И добавить восстановление:
  if D < D_healthy and M_s > M_s_healthy:
      A = min(1.0, A + recovery_rate)
```

Тогда асимметрия меняется линейно от отклонений от гомеостаза, а не экспоненциально.
