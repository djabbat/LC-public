## 0. Назначение

**Organizmal Aging** — всеобъемлющая трёхуровневая вычислительная модель (симулятор) старения организма, интегрирующая:

- **CDATA** (центриолярная теория, Counter #1 MCAOA) — первопричина накопления энтропии
- **MCAOA** (Counters #2–#6) — параллельные молекулярные треки повреждений
- **Ze-теорию** — ткане-специфические периоды самообновления и межтканевые конфликты

В отличие от существующих моделей (McAuley — только метаболизм, Horvath — статистические часы, López-Otín — концептуальные hallmarks, Kirkwood — disposable soma без конкретных механизмов), Organizmal Aging является **механистическим многомасштабным симулятором** от центриоли до смерти организма.

## 1. Три уровня модели

### Уровень #1: ЦЕНТРИОЛЬ — Накопитель энтропии

**Постулат:** Центриоль — единственная внутриклеточная структура, **не способная к самообновлению** (self-renewal). Все остальные структуры — белки, мембраны, митохондрии, ДНК, РНК — постоянно обновляются через синтез/деградацию. Центриоль передаётся как **шаблон** при делении клетки, и в дочернюю клетку всегда попадает старая центриоль.

| Свойство | Центриоль | Все остальные структуры |
|---|---|---|
| Механизм обновления | Только репликация (раз в клеточный цикл) | Постоянный синтез / деградация |
| Накопление повреждений | **Да, необратимо** | Обратимо через turn-over |
| Энтропийная роль | **Накопитель энтропии** | Самоочищение |
| Связь со старением | **Первопричина** (Counter #1 MCAOA) | Вторичные эффекты |

**Математически:**
```
S_centriole(t) = S₀ + ∫₀ᵗ η(τ) dτ

где:
  S_centriole(t) — энтропия центриоли в момент t
  S₀ — начальная энтропия (при рождении)
  η(τ) — скорость накопления повреждений (функция делений, АФК, ошибок репликации)
```

**Ключевое следствие:** Поскольку центриоль организует веретено деления и полярность клетки, её деградация вызывает **каскадные ошибки** на всех вышестоящих уровнях — нарушение асимметричного деления стволовых клеток, потеря полярности ткани, дисфункция органа.

**Литературная основа:** Tkemaladze 2023 (PMID 36583780), Tkemaladze 2005 (PMID 15886028).

### Уровень #2: 5 ТРЕКОВ MCAOA — Молекулярные счётчики

Параллельно с центриолярной энтропией (Counter #1) работают 5 молекулярных счётчиков, каждый из которых вносит независимый вклад в старение:

| # | Счётчик | Механизм | PMID | Зависимость от #1 |
|---|---|---|---|---|
| **#2** | Теломерный | Укорочение теломер при делении; лимит Хейфлика | 24138928 | Слабая (центриоль влияет на ориентацию деления) |
| **#3** | Митохондриальный | Накопление АФК, дисфункция дыхательной цепи | 39651989 | Умеренная (ошибки деления → неравное распределение митохондрий) |
| **#4** | Эпигенетический | Метилирование ДНК, потеря полярности (Cdc42) | 22542157, 36599349 | Сильная (центриоль ↔ полярность) |
| **#5** | Протеостатический | Агрегация белков, снижение шаперонов/аутофагии | 40738832 | Умеренная (центриоль ↔ организация синтеза) |
| **#6** | piRNA-опосредованный | Подавление транспозонов, стабильность генома | 38142432 | Слабая (эксплораторный) |

**Математически (агрегатор MCAOA):**
```
L_tissue(t) = Σᵢ wᵢ · fᵢ(Dᵢ(t))

где:
  L_tissue(t) — бремя старения ткани в момент t
  wᵢ — ткане-специфичный вес счётчика i
  fᵢ(x) = 1 / (1 + exp(−kᵢ(x − x_crit)))  — сигмоидальная функция повреждения
  Dᵢ(t) — состояние повреждения счётчика i
```

**Ключевое отличие от López-Otín hallmarks:** MCAOA — не описательная классификация, а **вычислительная архитектура** с измеримыми параметрами и фальсифицируемыми предсказаниями.

### Уровень #3: ТКАНЕ-СПЕЦИФИЧЕСКОЕ СТАРЕНИЕ — Ze-конфликты

**Постулат:** Разные ткани имеют разные периоды самообновления, и эти периоды создают **конфликты** при взаимодействии тканей.

| Ткань | Период самообновления | Тип старения | Уязвимость к счётчикам |
|---|---|---|---|
| Эпидермис | ~28 дней | Быстрое обновление → теломерное истощение | #2 (теломеры) |
| Кишечный эпителий | ~5 дней | Сверхбыстрое → накопление мутаций | #6 (piRNA/транспозоны) |
| Гепатоциты | ~300 дней | Медленное → протеостатический коллапс | #5 (протеостаз) |
| Нейроны | Никогда (постмитотические) | Нет обновления → митохондриальная дисфункция | #3 (митохондрии) |
| Гемопоэтические стволовые | ~месяцы (покоящиеся) | Периодическое → эпигенетический дрейф | #4 (эпигенетика) |
| Кардиомиоциты | ~0.5% в год | Почти постмитотические → АФК | #3 (митохондрии) |
| Эндотелий | ~3 года | Медленное → все счётчики | #1–#6 |
| Костная ткань | ~10 лет | Очень медленное → центриолярное | #1 (центриоль) |

**Ze-конфликт между тканями:**

Когда две ткани с разными периодами самообновления взаимодействуют (например, эндотелий сосудов и нейроны), возникает **рассогласование скоростей старения**:

```
Δ_age(tissue_A, tissue_B, t) = |L_A(t) − L_B(t)|

При Δ_age > τ_crit: функциональный разрыв → недостаточность органа
```

**Математическая модель конфликта (Ze-формализм):**
```
Z_conflict(i,j,t) = |τ_i · dL_i/dt − τ_j · dL_j/dt| · C_ij

где:
  τ_i — характерное время самообновления ткани i
  dL_i/dt — скорость накопления повреждений в ткани i
  C_ij — сила связи между тканями (матрица связности)
```

## 9. Открытые вопросы

1. ⏳ **Точная форма η(τ)** — скорость накопления энтропии в центриоли. Сейчас постулируется линейной; нужна экспериментальная валидация (ARGUS Phase B).
2. ⏳ **Матрица связности C_ij** между тканями — неизвестна; потребует калибровки на GTEx.
3. ⏳ **Количество типов тканей** — 8 базовых или больше? Нужна классификация по периодам самообновления.
4. ⏳ **Связь Counter #1 → Counters #2–#6** — количественная зависимость неизвестна (Sobol CDATA показал p=0.12 — inconclusive).
5. ⏳ **Falsifiability** — нужен протокол фальсификации с предрегистрацией на OSF.
6. ⏳ **Интеграция с Cell-DT v4.0** — использовать ли существующий код CDATA/cell_dt_core?
