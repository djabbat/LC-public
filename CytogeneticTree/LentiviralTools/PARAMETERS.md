# LentiviralTools — PARAMETERS

## Plasmid backbone

| Parameter | Value |
|---|---|
| Backbone | pLenti-CMV (or pLVX-EF1α-IRES) |
| Promoter | CMV (constitutive) or EF1α (lower silencing) |
| Insulator | cHS4 (optional, to reduce silencing) |
| Selection marker | Puromycin (or blasticidin for dual-cassette) |
| WPRE | yes (post-transcriptional enhancer) |
| Payload limit | ≤ 8 kb between LTRs |

## Packaging

| Parameter | Value |
|---|---|
| Packaging line | HEK293T |
| Transfection reagent | PEI (Polyethylenimine) MW 25k, linear |
| Plasmid ratio | 4 (pLenti) : 3 (psPAX2) : 1 (pMD2.G) |
| Harvest | 48 h + 72 h post-transfection |
| Concentration | Ultracentrifuge 20,000 g × 2 h or Lenti-X |
| Titer (target) | ≥ 10⁷ TU/mL |
| Titer assay | qPCR of integrated LTR |

## Transduction

| Parameter | Value |
|---|---|
| MOI (target) | 0.3 (to favor single integration) |
| Polybrene | 8 µg/mL |
| Spinoculation | 1,000 g × 60 min, 32 °C |
| Recovery | 24 h |
| Selection | Puromycin 2 µg/mL × 7 d |

## Clonal isolation

| Parameter | Value |
|---|---|
| FACS | 1 cell / well into 96-well |
| Medium | 50 % conditioned + 50 % fresh |
| Expansion | 3–4 weeks |
| QC panel | karyotype, fluorescence, tamoxifen response, integration count |

## Facility

| Requirement | Note |
|---|---|
| BSL-2 | required for lentivirus work |
| BSC class II | required |
| FACS access | in-house or core facility |
| Cryo-storage | LN₂ dewar, ≥ 100 vials capacity |
