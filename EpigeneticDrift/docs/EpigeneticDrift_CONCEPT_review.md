3.  **Существенная: Недостаточное обоснование формы уравнения.** Уравнение `D_{4}(n, t) = D_{4,0} + α_{4}·(n / n_{4}\*) + β_{4}·(t / τ_{4}) + γ_{4} · I(others)` представлено как данность. Нет отсылок к литературе, которая могла бы оправдать именно такую математическую форму (например, модели накопления ошибок).

4.  **Существенная: Высокий порог входа для читателя.** Документ предполагает глубокое знакомство с некоей "флагманской мета-теорией MCOA", которая сама по себе, судя по всему, ещё не опубликована (указана как "Perspective submission 2026-04-25"). Валидация модели только на основе метилирования DNA может быть недостаточной.
