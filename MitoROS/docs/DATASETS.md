## Restricted-access cohorts (require application)

- **UK Biobank** — proteomics, genomics, methylomics + age, sex, lifestyle; application via AMS portal.
- **Framingham Heart Study** (NHLBI) — longitudinal cardiovascular + aging biomarkers; dbGaP access.
- **NHANES** (CDC public-use files) — partial open; telomere length files (NHANES 1999–2002) already public.
- **Dortmund Vital Study** — EEG + health biomarkers; OSF mirror partial.
- **MPI-LEMON** — EEG aging cohort; open access.
- **Dunedin Multidisciplinary Health & Development Study** — contact via Otago University.
- **ARCHIE Mouse Phenome Database** (Jackson Lab) — open.

## Notes for calibration

1. **Use GEO accession as stable identifier** in PARAMETERS.md when citing a calibration source.
2. **Preference for longitudinal / multi-age sampled cohorts** with subject-level age covariate.
3. **Tissue-matching**: use sample metadata (tissue field) to route data to the correct MCOA tissue weight `w_i(tissue)`.
4. **Self-declaration**: every `DATASETS.md` entry must be checked for access-level (open vs restricted) before downstream calibration steps.
