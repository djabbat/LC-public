## 1. Organismal Aging — Центральный симулятор

### Назначение

Самообучающийся 4D-симулятор (3D + время), моделирующий:
- Развитие организма с зиготы через онтогенез к старению и смерти
- Старение как **основу всех заболеваний** (aging-driven diseases)
- Эффекты травм, инфекций, микробиома, макробиома (питание, среда)
- Интервенции (лекарства, диета, генотерапия)

Симулятор интегрирован с **физическим роботом ARGUS-LP**, который тестирует гипотезы на физических симуляторах (INFOGEST, клеточные культуры, органоиды) — и возвращает результаты в модель для самообучения.

### Три уровня

## 3. Уровень #1: Центриоль — накопитель энтропии

**Постулат:** Центриоль — единственная структура без самообновления. Все остальное (белки, мембраны, митохондрии, ДНК) обновляется. Центриоль — **первопричина старения**.

```
S_centriole(t) = S₀ + ∫₀ᵗ [η_div·n(τ)/n* + η_time·τ/τ* + η_ROS·ROS(τ) + η_rep·ε_rep(τ)] dτ

Составляющие энтропии:
  η_div  — повреждения при делении (polyGlu на старую центриоль)
  η_time — хронологическое накопление (TTLL/CCP дисбаланс)
  η_ROS  — окислительные повреждения (АФК → polyGlu)
  η_rep  — ошибки репликации центриоли
```

**Кодовая база:** 10 крейтов `cell_dt_*` — интегрируются в `sim_core/centriole/`.

**Экспериментальная валидация:** ARGUS-LP, Phase A (фибробласты BJ-hTERT), Phase B (HSC in vivo, Ulm).

**PubMed:** PMID 36583780 (Tqemaladze 2023), PMID 15886028 (Tqemaladze 2005).

## 4. Уровень #2: 5 треков MCAOA

> piRNA Counter #6 исключён из v1.0. Оставлен как слот расширения.

| # | Счётчик | Механизм | Ключевой PMID | Зависимость от #1 |
|---|---|---|---|---|
| **#2** | Теломерный | Укорочение теломер, лимит Хейфлика | 24138928 | Слабая |
| **#3** | Митохондриальный | АФК, mtDNA, дыхательная цепь | 39651989 | Умеренная |
| **#4** | Эпигенетический | Метилирование, Cdc42, полярность | 22542157, 36599349 | Сильная |
| **#5** | Протеостатический | Агрегация белков, шапероны, PPIA | 40738832 | Умеренная |

**Агрегатор:**
```
L_tissue(species, tissue, t) = Σᵢ₌₁⁵ wᵢ(species, tissue) · fᵢ(Dᵢ(t))

fᵢ(x) = 1 / (1 + exp(−kᵢ(x − x_crit)))   — сигмоида (каноническая)
fᵢ(x) = x                                   — линейная (только при Dᵢ << D_crit)
```

**Калибровка L_tissue через Frailty Index (Rockwood 2005, PMID 16129869):**
```
L_tissue = FI / 0.7
L_tissue < 0.30  → здоровье
L_tissue > 0.60  → L_critical: начало болезни
L_tissue > 1.00  → L_max: смерть
```

**Кодовая база:** 6 крейтов `mcoa_*` — интегрируются в `sim_core/counters/`.

**Препринт:** DOI 10.5281/zenodo.20055806.

## 5. Уровень #3: 8 тканей + Ze-конфликты + онтогенез

### Ткани (расширяемый набор)

| # | Ткань | τ_renewal (человек) | Доминирующий счётчик | Патология при L > L_crit |
|---|---|---|---|---|
| 1 | Эпидермис | ~28 дней | #2 (теломеры) | Атрофия, рак |
| 2 | Кишечный эпителий | ~5 дней | #5 (протеостаз) | Дисбиоз, рак |
| 3 | Гепатоциты | ~300 дней | #5 (протеостаз) | Фиброз, NASH |
| 4 | Нейроны | ∞ (постмитотические) | #3 (митохондрии) | Нейродегенерация |
| 5 | Гемопоэтические стволовые | ~месяцы | #4 (эпигенетика) | Иммуносенесценция, MDS |
| 6 | Кардиомиоциты | ~0.5%/год | #3 (митохондрии) | Сердечная недостаточность |
| 7 | Эндотелий | ~3 года | Все счётчики | Атеросклероз |
| 8 | Костная ткань | ~10 лет | #1 (центриоль) | Остеопороз |

### Ze-конфликты

```
Z_conflict(i, j, t) = |τᵢ · dLᵢ/dt − τⱼ · dLⱼ/dt| · C_ij(species)

При Z_conflict > Z_crit:
  → функциональный разрыв между тканями i и j
  → недостаточность органа
  → disease onset
```

**Формализм унаследован от Ze-Hierarchy:** старые ткани = monitoring (слежение), молодые = intervention (возмущение). HI → Z_conflict.

### Онтогенез

```
Zygote → Embryo → Neonate → Adult → Aging → Disease → Death
  │        │         │         │        │        │        │
  S₀   дифферен-  стартовые  пик     L>0.30   L>0.60   L→1.0
        цировка    Dᵢ₀       функции  Z растёт орган
```

### Микробиом и макробиом (подключаемые модули)

- **Микробиом:** кишечник (3 отдела), кожа, ротовая полость — влияет на L_intestinal, L_immune
- **Макробиом:** питание (INFOGEST-совместимая модель пищеварения), токсины, температура
- **Травмы:** внешние возмущения → ΔDᵢ → пересчёт L_tissue

## 6. Самообучающийся контур

```
┌──────────────────┐         ┌────────────────────────────┐
│  In silico model  │────────▶│  Физические симуляторы      │
│  (Rust + Python)  │         │                            │
│                   │         │  ARGUS-LP (клетки)         │
│  Генерация        │         │  INFOGEST (пищеварение)    │
│  гипотез H        │         │  Органоиды (ткани)         │
│                   │         │  C. elegans (организм)     │
└──────────────────┘         └────────────────────────────┘
         ▲                              │
         │      Результат R(H)          │
         └──────────────────────────────┘

P(θ | data) ∝ P(data | θ) · P(θ)    ← Байесовское обновление
```

**Цикл:**
1. Симулятор формулирует гипотезу (например: «ингибирование TTLL6 снижает скорость накопления polyGlu на 30%»)
2. ARGUS-LP ставит эксперимент (24/7, автономно)
3. Результат возвращается → Байесовское обновление параметров
4. Новая гипотеза — цикл повторяется

## 8. ЗАБОЛЕВАНИЯ КАК СЛЕДСТВИЕ СТАРЕНИЯ

Вместо «возраст-ассоциированные заболевания» — **возраст-обусловленные (aging-driven)**:

| Заболевание | L_tissue каких тканей > 0.60? | Z_conflict между чем? |
|---|---|---|
| Атеросклероз | L_endothelial > 0.60 + L_vascular_smooth > 0.60 | Эндотелий ↔ Кардиомиоциты |
| Болезнь Альцгеймера | L_neural > 0.60 | Нейроны ↔ Эндотелий (ГЭБ) |
| Рак | L_epithelial > 0.60 + random(DNA) | Эпителий ↔ Иммунная |
| Саркопения | L_muscle > 0.60 | Мышцы ↔ Нейроны (иннервация) |
| Остеопороз | L_bone > 0.60 | Кость ↔ Эндокринная |
| Диабет 2 типа | L_hepatic > 0.60 + L_pancreatic > 0.60 | Печень ↔ Поджелудочная |
| Иммуносенесценция | L_HSC > 0.60 | HSC ↔ Все ткани |
