## MONO_x_LYMPH — обзор

Пара ловит паттерны моноцитарно-лимфоцитарного перекрёста: инфекционный мононуклеоз (Эпштейна–Барр), хронический миеломоноцитарный лейкоз (ХММЛ), туберкулёз, саркоидоз, а также реактивные состояния (токсоплазмоз, ЦМВ) и миелофтиз. (Hoffbrand 2020, Gill 2015, WHO 2022).

### Ключевые ячейки — расшифровка

#### 1. MONO L2 + LYMPH H1 / H2
**Паттерн:** Моноцитопения + реактивный/крайний лимфоцитоз. 
**Физиология:** Дефицит моноцитов → макрофаги (костный мозг) не выходят в кровь; лимфоциты — компенсаторная активация. Классика: синдром активации макрофагов (САМ), гемофагоцитарный лимфогистиоцитоз (HLH). (Henter 2007, Кассирский И.А. 1970). 
**Топ-3 причины:**
1. **САМ / HLH** — 40% (аутоиммунные, наследственные, лимфома) (Fardet 2014).
2. **Инфекционный мононуклеоз (EBV)** с атипичными лимфоцитами — 25%.
3. **Т-клеточная лимфома** (агрессивная) — 10%. 
**Далее:** Коагулограмма (D-димер, фибриноген), ферритин, триглицериды, растворимый CD25, ПЦР EBV/ЦМВ, КМ-аспират. (BCSH 2016). 
**Pitfalls:** Отсутствие моноцитоза не исключает HLH; при EBV-мононуклеозе моноциты часто снижены в ранней фазе (Williams 2021).

#### 2. MONO H1 + LYMPH H1 / H2
**Паттерн:** Моноцитоз + лимфоцитоз — «вирусный мононуклеозный синдром». 
**Физиология:** Моноциты и лимфоциты растут одновременно при стимуляции антигеном (EBV, ЦМВ). CD8+ лимфоциты (атипичные) + моноциты как предшественники макрофагов. (Hoffbrand 2020). 
**Топ-3 причины:**
1. **Инфекционный мононуклеоз EBV** — 70%.
2. **ЦМВ-инфекция** (у взрослых, ВИЧ) — 15%.
3. **Токсоплазмоз** — 5%. 
**Далее:** Серология (EA-IgM, VCA-IgM), авидность, ПЦР EBV/ЦМВ, мазок крови (атипичные лимфоциты >10%). 
**Pitfalls:** EBV-инфекция у детей может давать только лимфоцитоз без моноцитоза; у пожилых — моноцитоз без лимфоцитоза (Wintrobe 2015).

#### 3. MONO H2 + LYMPH L1/L2
**Паттерн:** Высокий моноцитоз + лимфопения. 
**Физиология:** Моноцитоз при миелопролиферации (ХММЛ) или хроническом воспалении (туберкулёз) → лимфопения — следствие секвестрации в очагах или апоптоза. (WHO 2022). 
**Топ-3 причины:**
1. **ХММЛ (MDS/MPN)** — 50% (моноциты >1×10⁹/л, часто без лимфоцитоза) (Itzykson 2013).
2. **Туберкулёз диссеминированный** — 20%.
3. **САМ / бактериальный сепсис (грам-)** — 15%. 
**Далее:** Цитоморфология мазка (бласты, дизгранулоциты), биопсия КМ, флоу (CD14, CD16, CD56), IGRA/ПЦР туберкулёза, ПЦР на EBV. 
**Pitfalls:** Моноцитоз при ХММЛ часто >2×10⁹/л; при туберкулёзе моноцитоз умеренный, но с лимфопенией — красный флаг (Кассирский И.А. 1970).

#### 4. MONO H1 + LYMPH L2
**Паттерн:** Моноцитоз + глубокая лимфопения. 
**Физиология:** Активное высвобождение моноцитов при воспалении + лимфоциты разрушаются/секвестрируются (сепсис, туберкулёз, радиация). (Gill 2015). 
**Топ-3 причины:**
1. **Септический шок** — 40% (лифоциты <0.5×10⁹/л, моноциты >1.0×10⁹/л).
2. **Туберкулёз милиарный** — 25%.
3. **Апластическая анемия (вариант с моноцитозом)** — 10%. 
**Далее:** Прокальцитонин, гемокультуры, рентгенография, исключение ВИЧ, биопсия КМ. 
**Pitfalls:** Может имитировать ХММЛ — помогает наличие дизгранулоцитов и/или бластов (WHO 2022).

#### 5. MONO L2 + LYMPH L2
**Паттерн:** Панцитопения (все ростки) с особым дефицитом моноцитов и лимфоцитов. 
**Физиология:** Костный мозг подавлен — инфекции (ВИЧ, туберкулёз), токсины, опухолевая инфильтрация. (Hoffbrand 2020). 
**Топ-3 причины:**
1. **ВИЧ-инфекция (СПИД) с оппортунистической инфекцией** — 35%.
2. **Туберкулёз диссеминированный** — 20%.
3. **Панцитопения при аплазии или MDS** — 15%. 
**Далее:** ВИЧ-тест, ПЦР, IGRA, биопсия КМ, серология на EBV/ЦМВ. 
**Pitfalls:** Нормоцитарная анемия + моноцитопения + лимфопения — характерно для диссеминированного туберкулёза (Кассирский И.А. 1970).

#### 6. MONO H2 + LYMPH H2
**Паттерн:** Бицитопения (моноциты и лимфоциты значительно повышены). 
**Физиология:** Злокачественная лимфопролиферация (В- или Т-клеточная) с реактивным моноцитозом (лимфомы, лейкозы). (WHO 2022). 
**Топ-3 причины:**
1. **В-клеточная лимфома (особенно Burkitt, DLBCL) с моноцитозом** — 40%.
2. **Т-клеточный пролимфоцитарный лейкоз (T-PLL)** — 20%.
3. **ХММЛ с лимфоцитозом (вариант CMML-1/L)** — 15%. 
**Далее:** Флоу (кровь, КМ) – CD19, CD5, CD23, CD3, CD4, CD8, ZAP-70; биопсия/аспирация КМ, ПЭТ-КТ. 
**Pitfalls:** Неинфекционные причины (саркоидоз, туберкулёз) дают редко H2/H2 – исключить по гистологии.

#### 7. MONO H1 + LYMPH L0
**Паттерн:** Моноцитоз + нормальные лимфоциты. 
**Физиология:** Стимуляция макрофагальной системы без затрагивания лимфоцитарного звена (саркоидоз, гранулематоз). (Gill 2015). 
**Топ-3 причины:**
1. **Саркоидоз** — 35% (моноцитоз >0.8×10⁹/л, лимфоциты в норме).
2. **Бактериальный эндокардит (подострый)** — 20%.
3. **ХММЛ (ранняя стадия)** — 15%. 
**Далее:** АКТ, ангиотензинпревращающий фермент, эхоКГ, флоу (CD14/CD16), биопсия КМ. 
**Pitfalls:** Саркоидоз часто даёт L0/моноклональный моноцитоз – дифференцировать с ХММЛ (WHO 2022).

### Pearls

1. **MONO↑ + LYMPH↑ (H1/H1, H1/H2) — это не всегда EBV.** Если нет атипичных лимфоцитов (>10%) — исключить ХММЛ (моноцитоз >1×10⁹/л) и Т-клеточную лимфому (CD8+ клональность). (Williams 2021, Hoffbrand 2020).
2. **Лимфопения (LYMPH L1/L2) при моноцитозе ≥1.5×10⁹/л — это ХММЛ, пока не доказано иное.** Особенно у пожилых >60 лет (BCSH 2016).
3. **MONO L2 + LYMPH H2 — патогномонично для HLH/СAM с опухолевой этиологией.** Ферритин >10000 нг/мл + триглицериды >3 ммоль/л — немедленная биопсия КМ. (Henter 2007).
