## 03_acute_viral — динамика по 28 параметрам CBC

### Временная шкала (фазы)

| Фаза | Временной интервал | Характеристика |
|---|---|---|
| Hyperacute | 0–6 ч | Начало виремии, перераспределение лейкоцитов, нейтропения |
| Acute | 6–24 ч | Разворачивание адаптивного ответа, появление атипичных лимфоцитов |
| Subacute | 1–7 сут | Пик лимфоцитоза, выраженная атипия, тромбоцитопения |
| Recovery | >7 сут | Постепенная нормализация, исчезновение атипичных клеток |
