## WBC_x_LYMPH — обзор

Паттерн отражает баланс между общим пулом лейкоцитов и абсолютным числом лимфоцитов. Используется для быстрой дифференцировки реактивных (вирусных, поствакцинальных) состояний, острых и хронических лимфопролиферативных заболеваний, а также врождённых и приобретённых иммунодефицитов (Кассирский И.А., 1970; Hoffbrand, 2016).

### Ключевые ячейки — расшифровка

**1. WBC L2 / LYMPH L2**
- **Физиология:** Глобальное истощение всех ростков костного мозга с критическим дефицитом лимфоцитов.
- **Топ-3 причины:**
 - ВИЧ-инфекция стадии III-IV (СПИД) — CD4 < 200 (60%).
 - Апластическая анемия (печёночная недостаточность, лекарственная).
 - SCID (тяжёлый комбинированный иммунодефицит) — ранний дебют.
- **Следующий шаг:** Проточная цитометрия (CD3, CD4, CD19, CD16/56), ВИЧ-серология, костный мозг.
- **Pitfall:** Ложно-нормальные лимфоциты у детей при ВИЧ (трансплацентарные CD8).

**2. WBC H2 / LYMPH H2**
- **Физиология:** Массовая пролиферация лимфоидного клона с лейкоцитозом >100×10^9/л.
- **Топ-3 причины:**
 - Острый лимфобластный лейкоз (L1/L2) — 70-80%.
 - Хронический лимфолейкоз (CLL, стадия Rai III-IV) — 15%.
 - Пролимфоцитарный лейкоз (PLL — Т- или В-клеточный).
- **Следующий шаг:** Периферическая кровь — морфология (бласты, Smudge cells, prolymphocytes), проточная цитометрия (CD19, CD5, CD23, CD20, FMC7).
- **Pitfall:** Не путать с реактивным лимфоцитозом при коклюше или EBV — картина сходна, но абсолютное число редко >50×10^9/л.

**3. WBC L0 / LYMPH H2 (нормальные WBC, высокие лимфоциты)**
- **Физиология:** Хроническая клональная пролиферация без системного воспаления.
- **Топ-3 причины:**
 - CLL (B-CLL) — 90% (медленный лейкоцитоз, лимфоциты >5×10^9/л).
 - Лимфома малых лимфоцитов (SLL) с лейкемизацией.
 - Моноклональный B-клеточный лимфоцитоз (MBL) — предлейкоз.
- **Следующий шаг:** Иммунофенотипирование крови (CD5+, CD23+), серология на EBV/CMV.
- **Pitfall:** При стресс-реакции у детей — транзиторный лимфоцитоз до 8×10^9/л — исключать можно по норме LDH.

**4. WBC L1 / LYMPH H1 (лейкопения + лимфоцитоз)**
- **Физиология:** Компенсаторный лимфоцитоз при уничтожении нейтрофилов (например, при вирусном гепатите).
- **Топ-3 причины:**
 - Инфекционный мононуклеоз (EBV) — 50%.
 - Цитомегаловирусная инфекция (CMV) — 20%.
 - Агранулоцитоз (лекарственный) с реактивным лимфоцитозом.
- **Следующий шаг:** Анализ на гетерофильные антитела (Mono-test), ПЦР на EBV/CMV, LDH, костный мозг.
- **Pitfall:** Не пропустить апластический криз при парвовирусе B19 (WBC L2 + LYMPH L1).

**5. WBC H2 / LYMPH L0 (лейкоцитоз + нормальные лимфоциты)**
- **Физиология:** Острый воспалительный ответ с нейтрофильным сдвигом, лимфоциты не затронуты.
- **Топ-3 причины:**
 - Бактериальная инфекция (пневмония, пиелонефрит) — 80%.
 - Тканевая ишемия (инфаркт миокарда, панкреатит).
 - Стероидный лейкоцитоз (дексаметазон, преднизолон).
- **Следующий шаг:** CRP, прокальцитонин, посев крови, ЭКГ (при подозрении на инфаркт).
- **Pitfall:** Хронический миелолейкоз (CML) даёт лейкоцитоз с лимфоцитозом (не L0), а не с нормой.

**6. WBC L2 / LYMPH L0 (панлейкопения + нормальные лимфоциты)**
- **Физиология:** Селективное подавление нейтрофильного и моноцитарного ростка, лимфоидный сохранён.
- **Топ-3 причины:**
 - Лекарственный агранулоцитоз (клозапин, тиамазол) — 60%.
 - Синдром Фелти (ревматоидный артрит + нейтропения).
 - Дефицит G-CSF (редкий иммунный вариант).
- **Следующий шаг:** Уточнить приём лекарств, сывороточные антинуклеарные антитела (ANA), костный мозг, G-CSF.
- **Pitfall:** SCID у детей часто протекает с нормальными лимфоцитами (за счёт CD8+ T-клеток) — требуется иммунофенотипирование.

**7. WBC H1 / LYMPH H1 (умеренный лейкоцитоз + лимфоцитоз)**
- **Физиология:** Клональная B-клеточная экспансия с сопутствующим реактивным T-клеточным ответом.
- **Топ-3 причины:**
 - CLL (B-CLL) — 70%.
 - Инфекционный мононуклеоз (атипичные T-клетки) — 20%.
 - Лимфома mantle cell (клетки с CD5+, cyclin D1+).
- **Следующий шаг:** Проточная цитометрия (CD5, CD23, CD20, CD10, cyclin D1), LDH, β2-микроглобулин.
- **Pitfall:** Не путать с реактивным лимфоцитозом при коклюше (обычно дети, WBC H2, LYMPH H2).

### Pearls

1. **Правило 30-50:** При абсолютном лимфоцитозе >30×10^9/л без клиники инфекции — исключать CLL (WBC H1/H2, LYMPH H1/H2) → проточная цитометрия (Hoffbrand, 2016).
2. **Лимфопения + анемия + тромбоцитопения:** Патогномонична для апластического криза при парвовирусе B19 (WBC L2, LYMPH L2) → требуется ПЦР на B19 (BCSH Guidelines).
3. **Лейкоцитоз с нормальными лимфоцитами** — почти всегда бактериальная инфекция, даже если лимфоциты кажутся "нормальными" (Williams Hematology, 9th ed.).
