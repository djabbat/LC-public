## HCT_x_HGB — обзор

Данная пара отражает соответствие объёма эритроцитарной массы (HCT) и концентрации гемоглобина (HGB). Нарушение отношения HCT/HGB ≈ 3:1 сигнализирует о диспропорции синтеза гемоглобина и эритропоэза — классически при микроцитарных анемиях, гемоглобинопатиях или полицитемиях (Hoffbrand, 2016; Wintrobe, 2020).

### Ключевые ячейки — расшифровка

**1. HCT L2 / HGB L1 — «Гипохромная микроцитарная анемия (ЖДА vs талассемия)»**

- Физиология: HCT снижен сильнее, чем HGB → отношение HCT/HGB <3:1 → микроциты с низким MCHC, MCH. Классически при дефиците железа, талассемии, АХЗ (Hoffbrand, 2016).
- Топ-3 причины: ЖДА (60%), талассемия minor (20%), АХЗ (15%).
- Далее: сывороточное железо, ферритин, ОЖСС, трансферрин; электрофорез Hb (HbA₂, HbF); sTfR (BCSH, 2014).
- Pitfalls: β-талассемия minor даёт HCT/L1 при HGB/L1–L2; дифференцировка с АХЗ по ферритину, sTfR (Кассирский, 1970).

**2. HCT H2 / HGB L2 — «Парадоксальный паттерн (талассемия major, HbC болезнь)»**

- Физиология: HCT резко повышен (микроциты с высокой жёсткостью мембраны), HGB низкий → нарушение 3:1. Патогномонично для гемоглобинопатий C, E, талассемии major (Wintrobe, 2020; Fainstein, 2004).
- Топ-3: HbC гомозигота (60%), HbE гомозигота (25%), талассемия major (10%).
- Далее: электрофорез Hb, ретикулоциты, билирубин, ЛДГ; осмотическая резистентность эритроцитов.
- Pitfalls: артефакты счётчика при майор‑талассемии (HCT завышен из-за микроцитов).

**3. HCT L2 / HGB L0 — «Несовместимый паттерн (ошибка прибора, ПНГ, аплазия)»**

- Физиология: HCT критически низок, HGB норма — противоречие. Возможно при гигантских тромбоцитах (на HCT влияют), криопатологии, разрушении эритроцитов in vitro (ICSH, 2012).
- Топ-3: гигантские тромбоциты (50%), криоглобулинемия (25%), ПНГ с микросфероцитами (15%).
- Далее: мазок крови, ретикулоциты, АДАМТС‑13, криоглобулины; повторная сдача с разогревом.
- Pitfalls: не дифференцировать с истинной анемией без проверки мазка.

**4. HCT H1 / HGB L1 — «Макроцитоз с анемией (B12‑/фолат‑дефицит)»**

- Физиология: HCT повышен за счёт макроцитов (HCT = MCV×RBC), HGB снижен из-за мегалобластопоэза → псевдонорма HCT при анемии.
- Топ‑3: B12‑дефицит (50%), фолат‑дефицит (30%), гипотиреоз (10%).
- Далее: B12, фолаты, гомоцистеин, метилмалоновая кислота; мазок (макро‑овалоциты, гиперсегментация).
- Pitfalls: железо + B12 коморбидность даёт микро‑макро анемию.

**5. HCT H1 / HGB H1 — «Истинная полицитемия (PI>PV)»**

- Физиология: HCT и HGB высоки, отношение ≈3:1. Гиперпродукция эритроцитов при JAK2+ клоне (Williams, 2019).
- Топ‑3: полицитемия вера (70%), вторичный эритроцитоз (25%: гипоксия, лёгочная), реже Hb с высоким сродством к O₂.
- Далее: JAK2 (V617F, экзон 12), эритропоэтин (низкий при PI), SpO₂, пульс‑оксиметр; УЗИ почек, печени; ЭФГДС (портальная гипертензия).
- Pitfalls: ложно‑высокий HCT при макроцитозе (H1/L0/H0) — проверять MCV.

**6. HCT L2 / HGB L2 — «Тяжёлая нормохромная анемия (кровопотеря, гемолиз, криз)»**

- Физиология: оба показателя экстремально снижены, отношение соблюдено (≈3:1). Острое или хроническое кровотечение, иммунный гемолиз (Кассирский, 1970).
- Топ‑3: острый гемолиз (AIHA, PNH, гемоглобинурия) (40%), острая кровопотеря (35%), апластическая анемия (25%).
- Далее: ретикулоциты, ЛДГ, билирубин, прямой антиглобулиновый тест (Кумбс); мазок (сфероциты, шизоциты); кровопотеря – эндоскопия.
- Pitfalls: кровопотеря даёт нормоциты, гемолиз — макроциты (ретикулоцитоз).

**7. HCT L0 / HGB L0 — «Ложно‑нормальный HCT при макроцитозе (скрытая анемия)»**

- Физиология: HCT в норме, HGB в норме, но RBC ↓, MCV ↑. Возникает у беременных, алкоголиков, при гипотиреозе. Не выявляет анемию (Hoffbrand, 2016; Lupus, 2015).
- Топ‑3: беременность (50%), алкоголизм (25%), гипотиреоз (10%).
- Далее: MCV, MCH, RBC count, фолаты, B12, ТТГ, гамма‑ГТ.
- Pitfalls: при микро‑макро комбинации (дефицит железа + B12) HCT может стать нормальным, а HGB низким.

### Pearls

1. **Правило 3:1 — первая защита от ошибки**: если HCT/HGB ≠ 2.9–3.1 (у взрослых) — ищи артефакт: микроциты, макроциты, гигантские тромбоциты, криоглобулинемию, HbC/E (BCSH, 2014; ICSH, 2012). Смотри мазок до назначения терапии.

2. **L2/H1 — патогномоничный признак талассемии major**: отношение HCT/HGB может быть >5, HGB <60 г/л, HCT «норма» (50–60%). Не назначай феррум — это диагноз-ловушка (Wintrobe, 2020; Fainstein, 2004).

3. **H1/L1 у пожилого — классический B12-дефицит с макроцитозом**: помни, что HCT может оказаться «нормальным» при гемоглобине 70–90 г/л, если MCV >120 фл. Всегда считай клеточные индексы (Lupus, 2015; Williams Hematology, 2019).
