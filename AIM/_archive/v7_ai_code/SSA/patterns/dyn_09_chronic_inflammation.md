## 09_chronic_inflammation — динамика по 28 параметрам CBC

### Временна́я шкала (фазы)

| Фаза | Временной интервал | Характеристика |
|---|---|---|
| Hyperacute | 0–6 ч | Цитокиновый шторм, мобилизация нейтрофилов, начало синтеза белков острой фазы |
| Acute | 6–24 ч | Пик нейтрофилии, активация тромбоцитов, начало тромбоцитоза |
| Subacute | 1–7 сут | Формирование хронического ответа: анемия (нормоцитарная), тромбоцитоз, повышение СРБ/СОЭ |
| Chronic | >7 сут (месяцы) | Устойчивый тромбоцитоз, микроцитарная анемия (гепсидин‑опосредованная), высокие СОЭ и SII |
| Recovery | после устранения триггера | Постепенное снижение острофазовых маркеров, нормализация тромбоцитов, затем гемоглобина |
