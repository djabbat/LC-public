### L2 — Критически низкое RPR (<0.015)
- **Числовые границы (взрослые):** мужчины / женщины / беременные — <0.015 (единицы: безразмерный коэффициент, RDW(%) / PLT(×10⁹/л)). Для беременных нижняя граница не смещается, т.к. гестационный тромбоцитоз не достигает таких уровней.
- **Числовые границы (педиатрия):** новорождённые (<0.012), грудные (<0.013), 1–12 лет (<0.014), подростки (<0.015) — у детей PLT выше, RDW выше, поэтому порог может быть ниже с поправкой.
- **Скорость наступления имеет значение?** Да. Быстрое снижение RPR (часы–дни) указывает на острую реактивную тромбоцитемию (воспаление, кровопотеря, спленэктомия) или миелопролиферативный криз. Хроническое снижение (месяцы–годы) — первичный тромбоцитоз (ЭТ, ПМФ на ранней стадии).
- **Что это значит физиологически:** Чрезвычайно низкое отношение RDW/PLT возникает либо при массивном тромбоцитозе (PLT >800–1000 ×10⁹/л, что повышает вязкость, риск тромбоза), либо при аномально низком RDW (<10–11%) — крайне редко (например, при гомозиготной β-талассемии с однородными микроцитами). В обоих случаях — дестабилизация гемостаза.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|---------|----------------------|
| Эссенциальная тромбоцитемия (ЭТ) | Common | JAK2 V617F, CALR, MPL; тромбоцитоз >450, риск тромбозов (Hoffbrand 2020) |
| Реактивный тромбоцитоз (воспаление, железодефицит, постспленэктомия) | Common | ↑СРБ, ферритин низкий, аспления; RPR снижен умеренно |
| Хронический миелоидный лейкоз | Uncommon | BCR-ABL, лейкоцитоз, базофилия, тромбоцитоз |
| Истинная полицитемия с тромбоцитозом | Uncommon | ↑Ht, ↑Hb, JAK2 V617F |
| Тромбоцитоз при миелодисплазии (5q- синдром) | Rare | Анемия, макроцитоз, делеция 5q |
| Гомозиготная β-талассемия (очень низкий RDW) | Rare | HbF >90%, микроцитоз, анемия (Wintrobe 2019) |
| Гемоглобинопатия с однородными эритроцитами (HbS/HbC) | Rare | Электрофорез гемоглобина, целевидные клетки |

- **Опасность сейчас:** Высокий риск артериального (инсульт, ИМ) или венозного (ТЭЛА, тромбоз воротной вены) тромбоза в ближайшие часы–сутки при PLT >1000 ×10⁹/л (Hoffbrand, Williams Hematology). При истинном RPR <0.015 с PLT >1500 — геморрагический синдром (приобретенный дефицит фактора фон Виллебранда, качественный дефект тромбоцитов).
- **Минимальный набор подтверждающих тестов:** 1) Повторный CBC с подсчётом тромбоцитов (исключить псевдотромбоцитоз); 2) CРБ, ферритин; 3) JAK2 V617F, CALR, MPL; 4) BCR-ABL (если лейкоцитоз); 5) Электрофорез гемоглобина (если низкий RDW); 6) Исследование костного мозга (трепанобиопсия, кариотип).
- **Подсказки на скрытое:** Смежные параметры CBC: ↑PLT, ↓RDW, нормальное MCV (талассемия — ↓MCV), ↑Ht (полицитемия), ↑лейкоциты (ХМЛ). Ретикулоциты — при реактивном тромбоцитозе часто не изменены.
- **Pitfalls / artefacts:** 1) Псевдотромбоцитоз (криоглобулины, шок при заборе, фрагменты эритроцитов — считаются как PLT); 2) EDTA-зависимая агрегация тромбоцитов (обычно даёт псевдотромбоцитопению, но в редких случаях — псевдотромбоцитоз?); 3) Гиперлейкоцитоз (фрагменты лейкоцитов); 4) Липемия (изменяет оптический подсчёт). RDW может быть ложно низким при выраженном гемолизе (счётчик некорректно оценивает распределение). Проверить мазок периферической крови (истинный размер тромбоцитов, анизоцитоз).
- **Pearls:** 1) «Правило Виноградова» — при одновременном повышении Ht и PLT >1000 ×10⁹/л — срочный плазмаферез (Williams). 2) Низкий RDW (<11%) при высоком RDW-SD — исключить β-талассемию (ICSH 2014). 3) RPR <0.015 и отсутствие тромбозов не исключают миелопролиферативное заболевание — требуется активное наблюдение.

### L1 — Субнормальное RPR (0.015–0.03)
- **Числовые границы (взрослые):** 0.015–0.03. Женщины/мужчины — едины. Беременные: 0.012–0.025 (снижение PLT ближе к родам).
- **Числовые границы (педиатрия):** Новорождённые 0.013–0.025; грудные 0.014–0.027; 1–12 лет 0.015–0.030; подростки 0.015–0.030.
- **Скорость наступления имеет значение?** Да. Острое снижение (1–2 дня) — реактивный тромбоцитоз на фоне инфекции, кровопотери, гемолиз; хроническое — ранний миелопролиферативный процесс или компенсированная талассемия.
- **Что это значит физиологически:** Умеренный сдвиг баланса в сторону большего числа тромбоцитов (PLT 400–600) и/или нижней границы RDW (11–12%). Система гемостаза компенсирована, но при дополнительных факторах риска (атеросклероз, курение) возможны тромботические события.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|---------|----------------------|
| Реактивный тромбоцитоз (воспаление, ЖДА, кровотечение) | Common | ↑СРБ, ↓ферритин, фебрилитет |
| Аспления (функциональная/хирургическая) | Common | Тромбоцитоз, наличие телец Жолли, анизоцитоз |
| Эссенциальная тромбоцитемия (начальные стадии) | Uncommon | PLT 450–700, JAK2/ CALR |
| β-талассемия малая (низкий RDW) | Common | ↓MCV, ↑HbA₂ (Wintrobe 2019) |
| Хроническая болезнь почек (ранняя стадия) | Uncommon | ↑Креатинин, ↓EPO, нормохромная анемия |
| Гипертиреоз | Uncommon | ↑Т4, ↓ТТГ, тахикардия |
| Острый гемолиз (парвовирус, лекарства) | Rare | ↑ЛДГ, ↓гаптоглобин, ретикулоцитоз |

- **Опасность сейчас:** Ближайшие недели–месяцы — риск венозных тромбозов (особенно при сочетании с мутацией JAK2, ожирением, послеоперационным состоянием). Острых жизнеугрожающих состояний нет, но требуется контроль.
- **Минимальный набор подтверждающих тестов:** 1) Ферритин, СРБ; 2) Электрофорез гемоглобина; 3) JAK2 V617F (если PLT >450); 4) УЗИ селезенки (размер, очаги); 5) Свободные Т4, ТТГ.
- **Подсказки на скрытое:** MCV — при β-талассемии ↓; ретикулоциты — при гемолизе ↑; ЛДГ — при гемолизе или миелопролиферации; лейкоциты — при воспалении или ХМЛ.
- **Pitfalls / artefacts:** Те же, что для L2, но менее выражены. Псевдотромбоцитоз может имитировать L1. Всегда мазок периферической крови.
- **Pearls:** 1) У пациентов с ЖДА и RPR 0.02–0.03 после коррекции ферритина RPR нормализуется (Hoffbrand). 2) При β-талассемии малой RPR может быть стабильно низким — не требует терапии. 3) RPR <0.03 в сочетании с лейкоцитозом >15 ×10⁹/л — показание к исключению ХМЛ (BCSH 2019).
