### L2 — Критически низкий PDW (анизоцитоз тромбоцитов отсутствует)
- **Числовые границы (взрослые):** 
 Мужчины, женщины, беременные: PDW < 9.0 fL (или < 10.0 CV%) — нижний предел референса большинства анализаторов (ICSH 2014). Критическое снижение: PDW < 6.0 fL (или < 8.0 CV%) — крайне редко, обычно артефакт или терминальная гипоплазия.
- **Числовые границы (педиатрия):** 
 Новорождённые: < 9.5 fL (неонатальные анализаторы) / Грудные / 1-12 лет: < 9.0 fL / Подростки: < 9.0 fL (аналогично взрослым, референсы лабораторные).
- **Скорость наступления имеет значение?** 
 Да. Острое снижение (часы-дни) — всегда подозрение на артефакт (агрегация тромбоцитов). Хроническое снижение (месяцы) — аплазия или синдром Вискота-Олдрича.
- **Что это значит физиологически:** 
 Циркулирующий пул тромбоцитов представлен исключительно зрелыми, мелкими формами. Отсутствует сброс молодых крупных тромбоцитов из костного мозга (угнетение тромбопоэза) или происходит селективное разрушение молодых форм.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|---------|---------------------|
| Артефакт: EDTA-зависимая агрегация | Common | Крупные агрегаты в мазке, псевдотромбоцитопения |
| Врождённая тромбоцитопения с микротромбоцитами (синдром Вискота-Олдрича) | Rare | Микротромбоциты (MPV ↓), экзема, рецидивирующие инфекции |
| Аплазия мегакариоцитов | Rare | Панцитопения, снижение ретикулоцитов |
| Химиотерапия / лучевая терапия | Uncommon | Цитостатики, панцитопения |
| Терминальная стадия миелодисплазии (MDS с гипопродукцией) | Uncommon | Дисплазия, цитопении |
| Цирроз печени (уменьшение пула крупных тромбоцитов при спленомегалии) | Common | Спленомегалия, MPV ↓, селективная деструкция в селезёнке |
| Тиреотоксикоз (ускоренный клиренс) | Rare | Тахикардия, потеря веса |

- **Опасность сейчас:** 
 Изолированное снижение PDW без тромбоцитопении — угрозы нет. При ассоциации с тромбоцитопенией (PLT < 30×10⁹/л) и низким MPV — риск кровотечений (особенно внутричерепных) выше, чем при тромбоцитопении с высоким MPV (Wintrobe 2020). При синдроме Вискота-Олдрича — высокий риск геморрагического синдрома и ВЧК.
- **Минимальный набор подтверждающих тестов:** 
 Микроскопия мазка (оценка агрегатов, размер тромбоцитов), ретикулоциты, MPV, PLT-хром (для исключения агрегации), исследование генов WAS.
- **Подсказки на скрытое:** 
 MPV ↓, крупные тромбоциты не видны — подозрение на наследственную микротромбоцитопению. PLT нормальное, HDW ↓ — артефакт.
- **Pitfalls / artefacts:** 
 EDTA-зависимая агрегация (наиболее частая причина псевдо-L2), гемолиз при заборе, сахарном диабете (декстран).
- **Pearls:** 
 1. PDW < 9.0 fL с PLT > 150×10⁹/л — почти всегда артефакт до доказательства обратного (Hoffbrand 2020). 
 2. При синдроме Вискота-Олдрича PDW ↓ MPV ↓ — патогномоничное сочетание (Luka B.). 
 3. Оценка PDW только в сочетании с MPV: дискордантность (PDW ↓, MPV ↑) — фрагментация шизоцитами (ТМА).

### L1 — Нижний пограничный PDW
- **Числовые границы (взрослые):** 
 Мужчины, женщины, беременные: 9.0–10.5 fL (нижняя половина референса). Беременные — нижняя граница референса обычно сохраняется.
- **Числовые границы (педиатрия):** 
 Новорождённые: 9.5–11.0 fL / 1-12 лет: 9.0–11.0 fL / Подростки: 9.0–10.5 fL.
- **Скорость наступления имеет значение?** 
 Да. Острое — подозрение на артефакт. Хроническое — ареактивный тромбопоэз.
- **Что это значит физиологически:** 
 Небольшое снижение гетерогенности размеров тромбоцитов. Костный мозг не увеличивает продукцию, циркулирует преимущественно зрелая фракция.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|---------|---------------------|
| Спленомегалия (секвестрация крупных тромбоцитов) | Common | χρόνιη νόσος, гепатолиенальный синдром |
| Хронические заболевания печени | Common | Цирроз, портальная гипертензия |
| Апластическая анемия (ранняя стадия) | Rare | Панцитопения, гипоклеточный костный мозг |
| Гипотиреоз | Uncommon | Снижение метаболизма, MPV ↓ |
| Дефицит фолата / В12 | Uncommon | Макроцитарная анемия (MCV ↑), ретикулоцитопения |
| Хроническая болезнь почек (st. 4-5) | Common | Уремия, MPV ↓, дисфункция тромбоцитов |
| Приём цитостатиков (поддерживающие дозы) | Uncommon | Лейкопения, анемия |

- **Опасность сейчас:** 
 Минимальная. PDW на нижней границе референса не ассоциирован с острыми событиями.
- **Минимальный набор подтверждающих тестов:** 
 Мазок, MPV, ретикулоциты, PLT-хром, УЗИ органов брюшной полости (размер селезёнки).
- **Подсказки на скрытое:** 
 MCV ↑ (макроцитоз) — дефицит В12/фолата; MCV ↓ (микроцитоз) — анемия хронических заболеваний; лимфоцитоз — возможна лимфома с поражением селезёнки.
- **Pitfalls / artefacts:** 
 Гипокоагуляция (забор в гепарин), длительное хранение образца (тромбоциты набухают).
- **Pearls:** 
 1. PDW в пределах 9–10.5 fL с нормальным PLT — обычно вариант нормы (ICSH 2014). 
 2. При гепатоспленомегалии PDW ↓ — маркёр снижения времени жизни тромбоцитов (Wintrobe). 
 3. Следить за динамикой MPV: если MPV также снижается — риск гипопродукции.

### L0 — Оптимум (референс)
- **Числовые границы (взрослые):** 
 Мужчины, женщины, беременные: 10.5–14.5 fL (или 12.0–17.0 CV%). Беременные: 10.5–14.5 fL (нормальные изменения незначительны).
- **Числовые границы (педиатрия):** 
 Новорождённые: 11.0–15.0 fL / 1-12 лет: 10.5–14.5 fL / Подростки: 10.5–14.5 fL.
- **Скорость наступления имеет значение?** 
 Нет. Оптимум — отсутствие клинических отклонений.
- **Что это значит физиологически:** 
 Нормальная гетерогенность размеров тромбоцитов. Сбалансированный тромбопоэз и пул зрелых клеток.
- **Дифряд (топ-7 причин):** 
 Нет. Это норма.
- **Опасность сейчас:** 
 Отсутствует.
- **Минимальный набор подтверждающих тестов:** 
 Не требуется.
- **Подсказки на скрытое:** 
 Нет.
- **Pitfalls / artefacts:** 
 Ложнонормальный PDW при псевдотромбоцитозе (например, железодефицитная анемия с микроцитами, которые ошибочно считаются тромбоцитами). Проверка мазка обязательна при повышенном PLT.
- **Pearls:** 
 1. PDW в оптимуме — один из критериев качества работы анализатора (ICSH 2014). 
 2. Динамика PDW важнее однократного значения: резкое повышение при стабильном ранее уровне — ранний маркёр активации (Hoffbrand). 
 3. У здоровых доноров PDW 10.5–14.5 fL коррелирует с объёмом тромбоцитов (r=0.6).

### H1 — Повышенный PDW (выше оптимума)
- **Числовые границы (взрослые):** 
 Мужчины, женщины, беременные: 14.5–20.0 fL (верхняя половина референса). Беременные: 14.5–18.0 fL (физиологическое повышение в III триместре).
- **Числовые границы (педиатрия):** 
 Новорождённые: 15.0–18.0 fL / 1-12 лет: 14.5–18.0 fL / Подростки: 14.5–18.0 fL.
- **Скорость наступления имеет значение?** 
 Да. Острое повышение (часы) — активация тромбоцитов (ДВС, ТМА). Хроническое повышение (месяцы) — миелопролиферативное заболевание.
- **Что это значит физиологически:** 
 Умеренная гетерогенность размеров тромбоцитов. Сочетание крупных ретикулированных форм и мелких, фрагментированных или стареющих клеток. Усиленный тромбопоэз или активация.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|---------|---------------------|
| Реактивный тромбоцитоз (воспаление, анемия, постспленэктомия) | Common | Повышение PLT, CRP ↑, нормоциты |
| Дефицит железа (железодефицитная анемия) | Common | Микроцитоз (MCV ↓), гипохромия (MCH ↓) |
| Активное курение | Common | Повышение MPV, PDW |
