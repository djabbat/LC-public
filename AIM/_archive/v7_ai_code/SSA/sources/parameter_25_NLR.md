### L2 — Критическая нейтропения с относительным лимфоцитозом (life-threatening low)
- **Числовые границы (взрослые):** < 0.5 (единицы — безразмерный, ratio); мужчины/женщины/беременные — единые границы при условии нормы АЧН; беременные физиологично смещены в нейтрофилёз (снижение NLR за счёт лимфоцитоза нехарактерно, поэтому < 0.5 требует исключения нейтропении).
- **Числовые границы (педиатрия):** новорождённые < 0.4; грудные < 0.5; 1–12 лет < 0.6; подростки < 0.5 (адаптировано к возрастным нормам лимфоцитоза (Hoffbrand 2020)).
- **Скорость наступления имеет значение?** Да, острая (< 72 ч) — подозрение на агранулоцитоз (лекарственный, аутоиммунный) или вирусную супрессию (ВЭБ, ВИЧ, парвовирус). Хроническая — генетические нейтропении (эльстозит), циклическая нейтропения, перераспределительная при гиперспленизме.
- **Что это значит физиологически:** Глубокое истощение гранулоцитарного пула (костномозговая недостаточность или периферическое разрушение) при сохранном или повышенном лимфоцитарном пуле. NLR < 0.5 = АЧН < 1.0×10⁹/л (если лимфоциты в норме) (Wintrobe 2021).
- **Дифряд (топ-7 причин):**
| Причина | Частота | Ключевые ассоциации |
|---|---|---|
| Лекарственный агранулоцитоз (клозапин, тиамазол, НПВС) | common | Внезапное начало, лихорадка, язвенный стоматит |
| Аутоиммунная нейтропения (первичная, вторичная при СКВ, лимфопролиферации) | common | Спленомегалия, +АНФ, +лимфоцитоз |
| Вирусная инфекция (ВЭБ, ВИЧ, парвовирус B19) | common | Лимфоцитоз, атипичные мононуклеары, лихорадка |
| Гиперспленизм (цирроз, портальная гипертензия) | uncommon | Панцитопения, спленомегалия |
| Циклическая нейтропения (ELANE-мутация) | rare | Циклы 21–28 сут, оральный мукозит, фурункулёз |
| Синдром Костманна (тяжёлая врождённая нейтропения) | rare | С рождения, рецидивирующие инфекции, риск МДС/АМЛ |
| Дефицит фолиевой кислоты/В12 (редко как изолированная нейтропения) | uncommon | Макроцитоз, гиперсегментированные нейтрофилы |
- **Опасность сейчас:** Риск фебрильной нейтропении с бактериальным сепсисом (кишечная палочка, синегнойная палочка, стафилококк) в ближайшие часы. При АЧН < 0.5×10⁹/л — экстренная антибиотикотерапия. При АЧН < 0.1×10⁹/л — риск летального исхода (Кассирский И.А., 1950).
- **Минимальный набор подтверждающих тестов:** АЧН (absolute count), ретикулоциты, лейкоцитарная формула вручную (проверка на псевдонейтропению), АНФ, антинейтрофильные антитела, ВЭБ/ВИЧ/парвовирус (ПЦР), уровни В12 и фолиевой кислоты, трепанобиопсия (для исключения аплазии/инфильтрации).
- **Подсказки на скрытое:** MCV > 95 фл + нейтропения → дефицит В12/фолатов. Атипичные лимфоциты + нейтропения → инфекционный мононуклеоз. Спленомегалия → гиперспленизм. Зияющие моноциты → нейтрофилы — dehiscent.
- **Pitfalls / artefacts:** 
 - EDTA-индуцированная агрегация нейтрофилов → псевдонейтропения и ложно высокие лимфоциты (NLR снижен).
 - Гемолиз при заборе → разрушение лейкоцитов; использование литий-гепарина.
 - Утренняя псевдонейтропения (циркадные колебания CBC, около 40%).
 - Гигантские тромбоциты могут быть ошибочно посчитаны как лимфоциты.
- **Pearls:** 
 1. NLR < 0.5 с лимфоцитозом > 4.0×10⁹/л при отсутствии бактериальной инфекции — патогномонично для ВЭБ (Hoffbrand 2020, стр. 810).
 2. Если NLR < 0.5 у пожилого с MCV > 100 фл — всегда исключать В12-дефицит до переливания крови (Williams Hematology 2018).
 3. Medscape: при остром агранулоцитозе гранулоцитарный колониестимулирующий фактор (G-CSF) показан только при фебрильной нейтропении (BCSH 2013).

### L1 — Ниже оптимума (относительная лимфоцитопения или лёгкая нейтропения)
- **Числовые границы (взрослые):** 0.5 – 1.0 (мужчины/женщины/беременные — единые). Беременные: если NLR 0.5–1.0 — возможна физиологическая нейтрофилия до 0.5, но при истинном NLR < 1.0 нужно исключить гестационную нейтропению (ICSH 2014).
- **Числовые границы (педиатрия):** новорождённые 0.4 – 0.8; грудные 0.5 – 1.0; 1–12 лет 0.6 – 1.2; подростки 0.5 – 1.2.
- **Скорость наступления имеет значение?** Умеренно. Острая (> 1–3 суток) — вирусная инфекция или медикаментозная реакция; хроническая — вялотекущая инфекция (ВИЧ) или гиперспленизм.
- **Что это значит физиологически:** Умеренное снижение нейтрофилов (АЧН 1.0–1.5×10⁹/л) или относительное повышение лимфоцитов (вирусная стимуляция). Перераспределительный эффект (интерферон-альфа, кортикостероиды в малых дозах) (Wintrobe 2021).
- **Дифряд (топ-7 причин):**
| Причина | Частота | Ключевые ассоциации |
|---|---|---|
| Вирусная инфекция (грипп, РСВ, аденовирус) | common | Лимфоцитоз, субфебрилитет, катаральные симптомы |
| Приём иммуносупрессантов (метотрексат, азатиоприн) | common | Повышение трансаминаз, панцитопения редка |
| Дефицит железа (ранняя стадия) | uncommon | Микроцитоз, бледность |
| ВИЧ-инфекция (хроническая фаза) | uncommon | Лимфаденопатия, анемия, тромбоцитопения |
| Цирроз печени (начальный гиперспленизм) | uncommon | Спленомегалия, тромбоцитопения |
| Пароксизмальная ночная гемоглобинурия | rare | Желтуха, гемоглобинурия, тромбозы |
| Миелодиспластический синдром (нормо- или гипоклеточный) | rare | Макроцитоз, бицитопения |
- **Опасность сейчас:** Риск бактериальной инфекции умеренный (ниже, чем при L2). При АЧН 1.0×10⁹/л — профилактика при стоматологических манипуляциях не требуется (BCSH 2013).
- **Минимальный набор подтверждающих тестов:** АЧН, ретикулоциты, ВИЧ-тест (антитела + p24), ферритин, В12, фолаты, АСТ/АЛТ, УЗИ печени/селезёнки.
- **Подсказки на скрытое:** NLR < 1.0 с тромбоцитопенией — гиперспленизм или ПНГ. NLR < 1.0 с лейкопенией — лекарственная реакция. NLR < 1.0 с моноцитозом — МДС (хронические моноцитарные лейкозы).
- **Pitfalls / artefacts:** 
 - Холодовые агглютинины могут вызывать агрегацию лимфоцитов, занижая их количество (ложное повышение NLR). 
 - Лейкозные бласты с высокой клеточностью могут быть неверно подсчитаны как нейтрофилы.
- **Pearls:** 
 1. NLR < 1.0 + MCV > 95 фл = дефицит В12 до развития макроцитарной анемии (Hoffbrand 2020).
 2. При отклонении NLR в L1 — обязательный повтор через 2 недели на фоне отмены всех препаратов (Кассирский И.А., 1957).

### L0 — Оптимум (референс)
- **Числовые границы (взрослые):** 1.0 – 3.0 (мужчины/женщины/беременные). Беременные: 1.5 – 3.5 (ICSH 2014, учёт физиологического нейтрофилёза).
- **Числовые границы (педиатрия):** новорождённые 0.8 – 2.0; грудные 1.0 – 3.0; 1–12 лет 1.2 – 3.5; подростки 1.0 – 3.0.
- **Скорость наступления имеет значение?** Нет; NLR в L0 — стабильное значение при отсутствии острого/хронического стресса.
- **Что это значит физиологически:** Сбалансированные пулы нейтрофилов (маргинальный + циркулирующий) и лимфоцитов (Т, В, NK) без активного воспалительного или иммунного ответа. Соответствие нормативам ВОЗ (WHO 2011) и ICSH (2014).
- **Дифряд (топ-7 причин):** Дифференциальный диагноз не требуется; это норма.
| Причина | Частота | Ассоциации |
|---|---|---|
| Здоровый взрослый | common | Отсутствие жалоб |
| Латентный туберкулёз | rare | Исходно низкий NLR может быть в норме; отрицательный T-SPOT |
| Курение (лёгкое) | common | Лёгкий нейтрофилёз, NLR может быть > 3.0 — сдвиг в H1 |
| Лёгкая бактериальная инфекция (стадия реконвалесценции) | uncommon | NLR возвращается к норме |
| Тренированность (спортсмены) | common | Лёгкий лимфоцитоз покоя |
| Беременность II триместр | common | NLR 1.5–3.5 |
| Приём контрацептивов (эстроген/прогестерон) | uncommon | Небольшой нейтрофилёз |
- **Опасность сейчас:** Отсутствует.
- **Минимальный набор подтверждающих тестов:** Не требуется. При однократном отклонении в L0 — контроль через 6–12 мес при отсутствии симптомов.
- **Подсказки на скрытое:** Если L0 на фоне анемии — проверить железо, В12, фолаты, ферритин (анемия хронических заболеваний может не изменять NLR). При высоком RDW + L0 — ранний дефицит железа.
- **Pitfalls / artefacts:** 
 - Аспирационная пневмония может давать нормальный NLR на фоне бактериального процесса (высокая ложная надёжность).
 - Сепсис в первые 6 ч — NLR может оставаться в L0, затем сдвигаться в H1/H2.
- **Pearls:** 
 1. NLR в L0 у пациента с лихорадкой > 72 ч — исключить инфекционный эндокардит, туберкулёз, ГЛПС (Williams 2018).
 2. NLR = 2.0 ± 0.5 — классический референс Wintrobe (Lancet, 1956) с точкой отсечки для системного воспаления.

### H1 — Выше оптимума (умеренный нейтрофилёз или лимфоцитопения)
- **Числовые границы (взрослые):** 3.0 – 7.0 (мужчины/женщины/беременные). Беременные: 3.5 – 8.0 (ICSH 2014).
- **Числовые границы (педиатрия):** новорождённые 2.0 – 5.0; грудные 3.0 – 6.0; 1–12 лет 3.5 – 7.0; подростки 3.0 – 6.0.
- **Скорость наступления имеет значение?** Да. Острый нейтрофилёз (часы) — бактериальная инфекция, стресс, кортикостероиды, курение, миелопролиферация. Хронический — ожирение, диабет, курение, ХОБЛ, аутоиммунные болезни (Hoffbrand 2020).
- **Что это значит физиологически:** Преобладание нейтрофилов за счёт:
 - Истинного нейтрофилёза (усиленный миелопоэз, выброс из депо — кости, селезёнка).
 - Относительного нейтрофилёза (снижение лимфоцитов — кортикостероиды, стресс).
 -
