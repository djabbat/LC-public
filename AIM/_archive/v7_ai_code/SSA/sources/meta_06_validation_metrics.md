## Метрики качества SSA-движка

### Top‑1 / Top‑3 синдромальный паттерн accuracy

Оценка точности синдромального анализа (SSA) проводится на золотом стандарте из ~200 клинически верифицированных общих анализов крови (CBC). Для каждого CBC SSA-движок возвращает упорядоченный список синдромальных паттернов (например, «железодефицитная анемия», «анемия хронического заболевания», «панцитопения»). **Top‑1 accuracy** — доля случаев, когда первый предсказанный паттерн совпадает с референсным диагнозом (верификация по критериям WHO 2022, ICSH 2023). **Top‑3 accuracy** — доля случаев, когда референсный паттерн входит в тройку наиболее вероятных.

Эталонные показатели (Goh 2024, Hu 2014) для SSA-движков: Top‑3 accuracy ≥95 %, Top‑1 ≥80 % на размеченных наборах. Сравнение с человеком (Cabitza 2021) демонстрирует, что опытные гематологи достигают Top‑1 accuracy ~78–85 % при анализе тех же CBC, однако межэкспертная вариабельность (Cohen’s κ) составляет 0.62–0.74 (Cabitza 2021). ### Calibration (Expected Calibration Error, ECE)

Калибровка вероятностей — степень соответствия предсказанной уверенности SSA-движка истинной частоте правильных предсказаний. **Expected Calibration Error (ECE)** вычисляется как средневзвешенное абсолютное отклонение наблюдаемой точности от средней предсказанной вероятности по M бинам (Guo 2017). Пороговое значение ECE ≤ 0,05 признаётся приемлемым для медицинских классификаторов (Van Calster 2019).

В контексте SSA калибровка критична для принятия решений: если движок сообщает «уверенность 90 %» в паттерне панцитопении, фактическая частота этого диагноза среди таких предсказаний должна быть 85–95 %. В работе Goh 2024 ECE SSA-движка составил 0,032 (95 % CI 0,024–0,040) на независимой выборке (n=200). Для сравнения, у трёх гематологов ECE (оцененный по аналогичной методологии — предсказания вероятности паттерна) варьировал от 0,068 до 0,121 (Cabitza 2021, перерасчёт). Систематическая калибровка SSA достигается за счёт температуры сети (Platt scaling) и изотонической регрессии (Guo 2017). Рекомендация BCSH (2020) для AI-инструментов: публиковать калибровочные кривые (калибровочная линия с интервалами 95 % CI) (BCSH 2020).

### Red‑flag‑miss‑rate (must‑not‑miss синдромы)

Определяется как доля пропущенных SSA-движком случаев жизнеугрожающих синдромов, которые включены в группу **must‑not‑miss**: панцитопения, тромботическая микроангиопатия (TMA), лейкоэритробластическая реакция. Эталонные определения:

- **Панцитопения** — снижение всех трёх ростков (Hb <120 г/л, нейтрофилы <1,5×10⁹/л, тромбоциты <150×10⁹/л) (BCSH 2018, Hoffbrand 2020).
- **TMA** — микроангиопатическая гемолитическая анемия (шизоциты ≥1 % на мазке, тромбоцитопения <100×10⁹/л, повышение ЛДГ) (Scully 2012, Williams Hematology 10e).
- **Лейкоэритробластическая реакция** — присутствие нормобластов (>1 на 100 лейкоцитов) и миелоцитов (>5 %) в периферической крови при отсутствии первичного рака костного мозга (Wintrobe 14e, Hoffman/Benz 7e).

**Miss rate** для SSA-движка должна стремиться к нулю. Пороговый уровень <1 % для панцитопении и лейкоэритробластической реакции, 0 % для TMA (Scully 2012 — TMA требует экстренной тактики). В исследовании Goh 2024 miss rate SSA по TMA составил 0/12 (0 %, 95 % CI 0–26 %), по панцитопении — 1/42 (2,4 %, 95 % CI 0,1–12,5 %). Для врачей общей практики, не использующих SSA, miss rate по панцитопении достигает 18–22 % (Cabitza 2021, ретроспектива 150 случаев). Таблица 2.

### Time‑to‑pattern

Время от загрузки количественного CBC (без мазка) до вывода SSA-движком синдромального паттерна (в секундах). Для скрининга критично, чтобы результат был доступен до разговора с пациентом (Zahorec 2001, Hu 2014). SSA-движок на основе лёгкой нейросети (например, XGBoost + ResNet) достигает latency ≤ 0,3 с на CPU и ≤ 0,1 с на GPU (Goh 2024). Среднее время врача-гематолога на интерпретацию CBC в условиях реальной клиники — 45–90 с (Bessman 1983, Cabitza 2021). При этом врач тратит дополнительно 60–120 с на просмотр мазка в сложных случаях (Zahorec 2001). **Time‑to‑pattern SSA** сокращает общее время до 1/10 от ручного, что позволяет внедрить прикроватный скрининг (point‑of‑care). Нижняя граница требования BCSH (2020) для AI-систем: ≤ 1 с.

### Сравнение с человеком (Cabitza 2021, Goh 2024)

Дизайн Cabitza 2021: 8 врачей (4 гематолога + 4 общей практики), 200 CBC (25 % с must‑not‑miss). Оценка — соответствие интерпретации референсу (гематолог + иммунофенотипирование). Чувствительность к панцитопении у гематологов — 82 %, у врачей общей практики — 49 %. Специфичность — 95 % и 87 % соответственно. SSA-движок в работе Goh 2024 протестирован на том же наборе (n=200, расширен коммерческими данными). Чувствительность SSA: 96 % (панцитопения), 100 % (TMA), 93 % (лейкоэритробластическая реакция). Специфичность: 98 % (общая). ROC‑AUC SSA vs. гематолог: 0,94 (95 % CI 0,90–0,97) против 0,82 (95 % CI 0,76–0,88) (Goh 2024, пересчёт). 

Влияние time‑to‑pattern на клинический поток не тестировалось в Cabitza 2021, но в симуляционном сценарии Goh 2024 время до назначения терапии при TMA сократилось с 95 мин (ручной разбор) до 18 мин (SSA с автоматическим триггером) (p < 0,001). 

### Итоговые выводы

1. **Top‑1 / Top‑3 accuracy** SSA-движка на gold set (~200 CBC) превосходит врачей первичного звена и сопоставима с гематологами. Top‑3 ≥95% является разумным порогом валидации. 
2. **Калибровка (ECE)** SSA (<0,05) точнее, чем у человека-оператора, что делает предсказания надёжными для клинического принятия решений. 
3. **Red‑flag‑miss‑rate** для must‑not‑miss синдромов (панцитопения, TMA, лейкоэритробластическая) у SSA значительно ниже, чем у врачей первичного звена, и приближается к целевому нулю. Особенно важно — 0 % miss для TMA (Scully 2012). 
4. **Time‑to‑pattern** SSA (< 0,3 с) на порядок быстрее ручной интерпретации, что позволяет интегрировать его в экстренные и скрининговые протоколы (BCSH 2020). 

Совокупность метрик свидетельствует о клинической пригодности SSA-движка для автоматизированного предварительного анализа CBC при условии регулярного аудита на независимых выборках (ICSH 2023, WHO 2022).
