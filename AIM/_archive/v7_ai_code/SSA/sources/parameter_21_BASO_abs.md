### L2 — Критическая базофилопения (отсутствие)
- **Числовые границы (взрослые):** <0.01 × 10⁹/л (ниже предела обнаружения); мужчины / женщины / беременные — единые (ICSH 2014). 
- **Числовые границы (педиатрия):** <0.01 × 10⁹/л для всех возрастов (новорождённые, грудные, 1–12 лет, подростки) — отсутствие как норма. 
- **Скорость наступления имеет значение?** Нет; хроническое отсутствие — физиологично, острое возникновение (часы–дни) не описано. 
- **Что это значит физиологически:** Базофилы не являются обязательным компонентом периферической крови; их полное отсутствие в пределах погрешности метода не отражает патологии (Wintrobe 2020). 
- **Дифряд (топ-7 причин):** 
 | Причина | Частота | Ключевые ассоциации |
 |---------|---------|---------------------|
 | Норма (физиологический вариант) | Common | Без симптомов |
 | Хроническая крапивница (редко) | Rare | Зуд, волдыри |
 | Тяжёлая нейтропения (базофилы уничтожаются при цитостатической терапии) | Uncommon | Лейкопения, инфекции |
 | Ожоговая болезнь (первая неделя) | Rare | Гиповолемия, лейкоцитоз |
 | Глюкокортикоиды (высокие дозы) | Uncommon | Эозинофилия обычно, но базофилы могут исчезать |
 | Стресс-реакция (выброс катехоламинов) | Uncommon | Транзиторный лейкоцитоз |
 | Аплазия костного мозга (врождённая) | Rare | Панцитопения |
- **Опасность сейчас:** Нет; угроза жизни отсутствует. При сочетании с нейтропенией (<0.5×10⁹/л) — риск инфекций (Hoffbrand 2020). 
- **Минимальный набор подтверждающих тестов:** Повторный анализ на другой пробирке (исключить преаналитическую ошибку); клинический осмотр. 
- **Подсказки на скрытое:** Сопутствующая нейтропения (ANC) и лимфопения (ALC) для оценки общего резерва гранулоцитов. 
- **Pitfalls / artefacts:** Длительное хранение образца (базофилы дегранулируют); использование глюкокортикоидов; гемолиз. 
- **Pearls:** 
 1. Отсутствие базофилов — норма для 1–5% здоровых взрослых (Williams 2013). 
 2. Правило Виноградова: изолированная базофилопения не имеет диагностической ценности (Кассирский И.А., 1979). 
 3. При интерпретации лейкоцитоза с нулевыми базофилами необходимо проверить мазок: возможна агрегация клеток на краю стекла (ICSH 2014).

### L1 — Нижеоптимальная зона (низкие, но не критические)
- **Числовые границы (взрослые):** 0.01 – 0.02 × 10⁹/л (нижняя часть референса); мужчины / женщины / беременные — одинаково. 
- **Числовые границы (педиатрия):** 0.01 – 0.02 × 10⁹/л для всех возрастов (близко к нижней границе). 
- **Скорость наступления имеет значение?** Нет; хронически низкое значение обычно без патологии. 
- **Что это значит физиологически:** Снижение продукции или ускоренное потребление; уровень ещё не достигает критического дефицита (Hoffbrand 2020). 
- **Дифряд (топ-7 причин):** 
 | Причина | Частота | Ключевые ассоциации |
 |---------|---------|---------------------|
 | Нормальный вариант | Common | Никаких |
 | Идиопатическая базофиленопения | Uncommon | Самопроизвольно |
 | Гиперкортицизм (болезнь/синдром Кушинга) | Rare | Ожирение, гипертония, стрии |
 | Лучевая терапия (местная) | Uncommon | Поле облучения, цитопения |
 | Химиотерапия (начальная стадия) | Common | Снижение всех ростков |
 | Вирусная инфекция (грипп, ОРВИ) | Rare | Лимфопения, моноцитоз |
 | Уремня (хроническая почечная недостаточность) | Rare | Анемия, повышение креатинина |
- **Опасность сейчас:** Нет; при острой ситуации не угрожает. 
- **Минимальный набор подтверждающих тестов:** Анализ мазка (оценка морфологии базофилов), СРБ, ферритин, кортизол утренний. 
- **Подсказки на скрытое:** Оценить абсолютные эозинофилы (EOS) — часто снижаются параллельно; MCV — при микроцитозе исключить ХМЛ (но для ХМЛ характерна базофилия, а не снижение). 
- **Pitfalls / artefacts:** Псевдонизкий результат из-за дегрануляции при рН-зависимости (ICSH 2014); забор в гепариновые пробирки снижает адгезию базофилов. 
- **Pearls:** 
 1. У беременных к концу срока базофилы могут незначительно снижаться (Lurie 2008). 
 2. Изолированная базофилопения не является критерием агранулоцитоза (Кассирский И.А., 1979). 
 3. При подозрении на паразитоз с базофилениемой (положительный скрининг) на фоне низких базофилов — повторить анализ в разное время суток (суточные колебания достигают 40%) (Hoffbrand 2020).

### L0 — Оптимум (референсный диапазон)
- **Числовые границы (взрослые):** 0.02 – 0.10 × 10⁹/л (мужчины / женщины / беременные — единый референс). 
- **Числовые границы (педиатрия):** 
 - Новорождённые: 0.02 – 0.10 × 10⁹/л 
 - Грудные (до 2 лет): 0.01 – 0.08 × 10⁹/л 
 - 1–12 лет: 0.01 – 0.10 × 10⁹/л 
 - Подростки: 0.02 – 0.10 × 10⁹/л 
- **Скорость наступления имеет значение?** Нет; стабильное значение. 
- **Что это значит физиологически:** Нормальная продукция и циркуляция базофилов (до 1% от общего числа лейкоцитов) с балансом обновления (жизнь 1–2 суток). 
- **Дифряд (топ-7 причин):** В норме нет патологии. 
- **Опасность сейчас:** Отсутствует. 
- **Минимальный набор подтверждающих тестов:** Не требуется. 
- **Подсказки на скрытое:** Если клинически есть подозрение на аллергию или миелопролиферацию, но базофилы в норме — проверить EOS, триптазу тучных клеток. 
- **Pitfalls / artefacts:** Нет. 
- **Pearls:** 
 1. Референсные границы варьируют между лабораториями (0–0.2); клинически значимо только абсолютное значение, а не процент (Williams 2013). 
 2. У детей до 2 лет базофилы иногда ниже, чем у взрослых (BCSH 2014).
