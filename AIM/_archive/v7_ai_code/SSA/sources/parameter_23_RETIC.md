### Ретикулоциты (RET %, RET-He) — 5-зонный профиль

**Единицы:** RET % — проценты от эритроцитов; RET-He (эквивалент гемоглобина ретикулоцита) — пг (пикограммы). Цена деления анализатора — 0.01% для RET, 0.1 пг для RET-He (Sysmex, Abbott — ICSH 2014).

### L2 — Критическая арегенераторная зона (life-threatening low)

- **Числовые границы (взрослые):**
 - RET %: < 0.2% (абсолютное число < 10 × 10⁹/л)
 - RET-He: < 22 пг (тяжёлый гипохромный эритропоэз, функциональный дефицит железа при арегенерации)
 - Беременные: RET % < 0.5% (нормально повышенный фон до 2% — при падении ниже 0.5% критично)
- **Числовые границы (педиатрия):**
 - Новорождённые: RET % < 2% (норма 3-7%; < 2% — аплазия)
 - Грудные: RET % < 0.5%
 - 1-12 лет: < 0.5%
 - Подростки: < 0.3%
- **Скорость наступления имеет значение?** Да, критично. Острое падение (часы–дни) — парвовирусный криз при гемолизе. Хроническое (недели–месяцы) — апластическая анемия, миелодисплазия (МДС).
- **Что это значит физиологически:** Костный мозг не способен компенсировать потерю эритроцитов. Снижение пролиферации или гибель эритроидных предшественников.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
| :--- | :--- | :--- |
| Парвовирус B19-инфекция (апластический криз) | Common (при гемоглобинопатиях) | Гемолитическая анемия (серповидноклеточная, сфероцитоз), ретикулоцитопения, арегенераторная анемия |
| Апластическая анемия (тяжёлая форма) | Common | Панцитопения, RET % < 0.2%, гипоклеточный костный мозг (Camitta 1976) |
| Миелодиспластический синдром (5q-синдром, гипопластический МДС) | Uncommon | Цитопении, RET % низкий, RET-He может быть снижен при сидеробластном кольце |
| Лекарственная миелотоксичность (метотрексат, азатиоприн, интерферон) | Common | Обратимо, связь с введением препарата |
| Лучевая болезнь (острая костномозговая форма) | Rare (в мирное время) | Миелоаблация, абсолютная ретикулоцитопения |
| Синдром Швахмана-Дайемонда | Rare (педиатрия) | Нейтропения, панкреатическая недостаточность, RET % < 0.2% |
| Чистая эритроцитарная аплазия (PRCA) | Rare | Антитела к эритропоэтину, тимома, В19 |

- **Опасность сейчас:** Острая анемия с гемодинамической декомпенсацией (гипоксия миокарда, инфаркт, сердечная недостаточность) в ближайшие часы–сутки. При парвовирусном кризе — внезапная анемизация, коллапс, риск отека лёгких.
- **Минимальный набор подтверждающих тестов:** Ретикулоцитарный индекс менее 1-2%, абсолютное число ретикулоцитов (ARC), мазок периферической крови (исключить шистоциты, бласты, фрагментированные клетки), биопсия костного мозга (гипоплазия/аплазия, кольцевые сидеробласты). ПЦР на парвовирус B19.
- **Подсказки на скрытое:**
 - MCV и MCH: нормальные или сниженные при МДС (миелоидная дисплазия), нормальные при PRCA.
 - Тромбоциты: резко снижены при аплазии (тяжелая панцитопения), нормальны при парвовирусе.
 - Нейтрофилы: снижены при аплазии, нормальны при PRCA.
- **Pitfalls / artefacts:** Холодовые агглютинины (завышение RET %, занижение RET-He — ложно-нормальный RET-He). EDTA-зависимая псевдотромбоцитопения (не влияет на RET). Фрагментированные эритроциты при ДВС-синдроме (могут имитировать ретикулоцитопению, т.к. часть RET не распознаётся). Спленэктомия (истинное снижение RET-He из-за повышенной пропускной способности селезёнки — не патология). (ICSH 2014).
- **Pearls:**
 1. **Правило Виноградова (1970-е):** Ретикулоцитопения на фоне анемии (RET < 0.5%) — всегда указывает на поражение костного мозга, чаще всего неопластическое (МДС, лейкоз) или апластическое.
 2. **Hoffbrand (2020):** RET-He < 22 пг при анемии — маркер функционального дефицита железа, но при аплазии RET% падает раньше, чем RET-He.
 3. **Wintrobe (1954):** Арегенераторная анемия с RET < 0.2% — парвовирус B19 до подтверждения диагноза.

### L1 — Гипорегенераторная анемия (субоптимальная продукция)

- **Числовые границы (взрослые):**
 - RET %: 0.2–0.5% (ARC 10–30 × 10⁹/л)
 - RET-He: 22–27 пг (лёгкая гипохромия ретикулоцитов)
 - Беременные: 0.5–1.0%
- **Числовые границы (педиатрия):**
 - Новорождённые: 2–4%
 - Грудные: 0.5–1.5%
 - 1-12 лет: 0.5–1.0%
 - Подростки: 0.3–0.8%
- **Скорость наступления имеет значение?** Умеренно. Хроническое снижение (месяцы) — дефицит железа, В12/фолатов; острое (часы) — миелосупрессия после химиотерапии.
- **Что это значит физиологически:** Снижение эритроидной пролиферации костного мозга при сохранении базового фона продукции. Часто — результат недостаточности субстрата или токсического поражения.
- **Дифряд:**

| Причина | Частота | Ключевые ассоциации |
| :--- | :--- | :--- |
| Железодефицитная анемия (лёгкая–умеренная) | Very common | Микроцитоз, гипохромия, RET-He < 26 пг, ферритин < 30 нг/мл |
| Анемия хронического воспаления (АХЗ) | Common | Нормоцитарная, сниженный сывороточный железа, повыш. ферритин, IL-6 |
| Мегалобластная анемия (В12/фолиевая недостаточность) | Common | Макроцитоз (MCV > 105 фл), RET-He нормальный или снижен, нейтропения, гиперсигментированные нейтрофилы |
| МДС с низким риском (RA, RCMD) | Uncommon | Цитопении, дизэритропоэз, RET% 0.3–0.5% |
| Лекарства (ингибиторы тирозинкиназы, гемцитабин) | Uncommon | Связь с приёмом, обратимо |
| Лёгкая костномозговая гипоплазия (идиопатическая) | Rare | Панцитопения, RET% низкий, КМ гипоклеточный |
| Эндокринопатии (гипотиреоз, гипогонадизм) | Rare | Нормоцитарная анемия, RET% снижен, положительный ответ на заместительную терапию |

- **Опасность сейчас:** Низкий риск острой декомпенсации. При прогрессировании — нарастание анемии (гемоглобин < 80 г/л за 2-4 недели). При В12-дефиците — фолиевая кислота может маскировать неврологические нарушения.
- **Минимальный набор подтверждающих тестов:** Ретикулоцитарный индекс < 1.5%, ферритин, сывороточное железо, ОЖСС, В12, фолиевая кислота, мазок крови (макроциты, гиперсегментация, микроциты). Тиреотропный гормон (при подозрении на гипотиреоз).
- **Подсказки на скрытое:** MCV — если увеличен (100–115 фл) — В12/фолиевая. RDW — повышен при дефиците железа. Тромбоциты — могут быть умеренно снижены при мегалобластной анемии. RET-He — лучший маркер гипохромии (ICSH 2014).
- **Pitfalls / artefacts:** Хронический гемолиз (RET% может быть нормальным, но это маскируется). EDTA-зависимая псевдотромбоцитопения не влияет. Лейкоцитоз (не влияет на RET, но может занижать RET-He из-за сдвига щелочи). (Hoffbrand 2020).
- **Pearls:**
 1. **Виноградов (1970-е):** RET% 0.2–0.5% + микроцитоз — железодефицитная анемия до выявления дефицита.
 2. **Williams Hematology (2015):** RET-He < 27 пг — чувствительный маркер дефицита железа на ранних стадиях, даже до снижения MCV.
 3. **Клинический трюк:** RET-He снижен при железодефиците, но нормален при В12-дефиците, что помогает диф. диагностике без ферритина (ICSH 2014).

### L0 — Оптимум (целевая зона)

- **Числовые границы (взрослые):**
 - RET %: 0.5–2.5% (ARC 25–80 × 10⁹/л)
 - RET-He: 28–35 пг (нормальный гемоглобин)
- **Числовые границы (педиатрия):**
 - Новорождённые: 3–7% (высокий физиологический гемолиз)
 - Грудные: 1–3%
 - 1-12 лет: 0.5–1.5%
 - Подростки: 0.5–2.0%
- **Скорость наступления:** Не имеет значения. Это стабильное состояние.
- **Что это значит физиологически:** Продукция эритроцитов костным мозгом сбалансирована с разрушением для поддержания стабильного гемоглобина. RET-He > 28 пг — адекватная гемоглобинизация в период созревания.
- **Дифряд (зона не требует дифференцировки):** Норма. Исключение: может быть маскирован патологический гемолиз с компенсацией (RET% на верхней границе нормы, RET-He снижен — скрытый дефицит железа). (Мониторинг при хронических анемиях).
- **Опасность сейчас:** Нет.
- **Минимальный набор подтверждающих тестов:** Не требуется.
- **Подсказки на скрытое:** Если RET% на нижней границе (0.5%), но пациент старше 60 лет — норма. RET-He на нижней границе (28 пг) — может быть признаком латентного железодефицита (BCHS 2011).
- **Pitfalls / artefacts:** Курение — повышает RET% на 0.5–1.0% за счёт стимуляции эритропоэтином. Высота над уровнем моря (гипоксия) — RET% может быть до 4% (адаптация).
- **Pearls:**
 1. **Hoffbrand (2020):** RET-He — единственный маркер, который не зависит от ретикулоцитарной анизоцитоза и лучше коррелирует с функциональным железом.
 2. **Клинический факт:** У пациентов с гемоглобинопатией (серповидноклеточная, талассемия) RET% 2–6% — норма (компенсаторный эритропоэз), но RET-He < 26 пг указывает на неэффективный эритропоэз (ICSH 2014).
 3. **Wintrobe (1954):** RET% 1–2% — идеальное состояние, но при анемии любой степени — требует оценки RET-He.
