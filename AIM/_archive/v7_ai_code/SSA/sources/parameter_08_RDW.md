### RDW-CV (Коэффициент вариации ширины распределения эритроцитов по объёму)

---

### L2 — Критически низкий анизоцитоз (гомогенность, угнетение эритропоэза)
- **Числовые границы (взрослые):** < 10.5 % (мужчины/женщины); беременные — < 11.0 % (Hoffbrand 2020).
- **Числовые границы (педиатрия):** новорождённые — < 14.0 %; грудные — < 12.5 %; 1-12 лет — < 12.0 %; подростки — < 11.5 % (Brugnara 2007, ICSH 2014).
- **Скорость наступления имеет значение?** Да. Хроническое снижение характерно для *талассемии* или *железодефицита* на фоне равномерного подавления; острое — редко, при клональном выходе (например, *новый эритроидный клон* при *ЭМП*).
- **Что это значит физиологически:** Эритроциты имеют узкое распределение объёмов (гомогенный пул). Это указывает на стабильную популяцию без активного неэффективного эритропоэза. Основной механизм — либо *подавление деления* (мегалобластоз), либо *истощение запасов* с однотипным микроцитозом (на поздних стадиях ЖДА). Истинное снижение RDW почти всегда патология (Wintrobe 2019).
- **Дифряд (топ-7 причин):**
| Причина | Частота | Ключевые ассоциации |
|---------|---------|---------------------|
| Талассемия (малая/большая) | Common | MCV ↓, MCH ↓, HbA₂ ↑ (β-талассемия), нормо/гипохромия |
| Железодефицитная анемия (тяжёлая, истощённая) | Common | Очень низкий ферритин, насыщение трансферрина < 5 %, MCV ↓ |
| Анемия хронических болезней (АХБ) с равномерным микроцитозом | Uncommon | Ферритин ↑/N, сыв. железо ↓, ОЖСС ↓ |
| Лечение цитостатиками (равномерное подавление эритропоэза) | Uncommon | MCV ↑ при гидроксимочевине, MDH × алкилирующие агенты |
| Гемоглобинопатия E (HbE) | Rare | MCV ↓, гипохромия, мишеневидные клетки |
| Легкая/средняя*мегалобластная*анемия* (B12/фолиеводефицитная, начальная) | Rare (L2) | MCV уже ↑, но RDW может быть N за счёт равномерной макроцитоза |
| Гипосидеремия при хроническом диализе | Uncommon | Рефрактерность к ЭПО, низкий уровень сыв.железа |
- **Опасность сейчас:** Риск нетяжёлых *инфекций* или *нарушений роста* (у детей) — гемодепрессия. Острой угрозы жизни нет.
- **Минимальный набор подтверждающих тестов:** MCV, MCH, ретикулоциты, сыв. железо, ОЖСС, ферритин, электрофорез гемоглобина (с HbA₂ и HbF).
- **Подсказки на скрытое:** MCV низкий — талассемия (RDW ↓→ талассемия, RDW ↑ → ЖДА; правило Bessman). Если MCV высокий — исключить B12/фолиевую (но RDW при них почти всегда повышено, L2 редко).
- **Pitfalls / artefacts:** *Агглютинация* — ложное падение RDW? Нет, она скорее повышает RDW. *Гемолиз* — может имитировать низкое RDW при разрушении крупных клеток? Теоретически возможно.
- **Pearls:**
 - «Правило Bessman»: RDW > 14.5 % при MCV < 80 фл → ЖДА (чувствительность 80–90 %). RDW < 14.5 % при MCV < 80 фл → талассемия (Williams Hematology 2020).
 - «Анизоцитарный индекс»: при *талассемии* RDW-CV почти всегда N (L0 или L1), при ЖДА — H1.
 - Снижение RDW-CV < 11 % при *повышении* MCV — признак *алкогольной анемии* (точнее, равномерной макроцитоза от алкоголя).

### L1 — Ниже оптимума (субоптимальная вариабельность)
- **Числовые границы (взрослые):** 10.5 – 11.5 % (мужчины/женщины); беременные — 11.0 – 12.0 %.
- **Числовые границы (педиатрия):** новорождённые — 14.0 – 15.5 %; грудные — 12.5 – 14.0 %; 1-12 лет — 12.0 – 13.5 %; подростки — 11.5 – 13.0 %.
- **Скорость наступления имеет значение?** Нет. Субклинический диапазон.
- **Что это значит физиологически:** Небольшое сужение пула эритроцитов — может быть нормой (у части популяции) или отражать *неглубокую гомогенизацию* (начальные формы талассемии, легкий дефицит железа). У здоровых лиц с высоким MCV (ретикулоциты ↓) — вариант.
- **Дифряд (топ-7 причин):**
| Причина | Частота | Ключевые ассоциации |
|---------|---------|---------------------|
| Конституциональный вариант (здоровые) | Common | N MCV, N гемоглобин |
| Талассемия (лёгкая гетерозиготная) | Common | MCV ↓, MCH ↓, но HbA₂ ↑ |
| ЖДА (начальная, лёгкая) | Common | Ферритин ↓, MCV N или ↓ |
| Хронический стресс (АХБ) | Common | Ферритин ↑, сыв. железо ↓ |
| Хроническая болезнь почек (ХБП, не на диализе) | Common | MCV N, гемоглобин ↓ |
| Микросфероцитоз наследственный (лёгкий) | Rare | MCHC ↑, прям. билирубин ↑ |
| Гемоглобинопатия C (HbC) | Rare | Мишеневидные клетки, MCV ↓ |
- **Опасность сейчас:** Нет. Мониторинг.
- **Минимальный набор подтверждающих тестов:** MCV, MCH, ферритин, сыв. железо, ОЖСС.
- **Подсказки на скрытое:** Если MCV ↓ → электрофорез гемоглобина + железо. Если MCV N → ферритин и ретикулоциты.
- **Pitfalls / artefacts:** *Агглютинация* — может повышать RDW, = L1 не характерен. *Гемолиз* — разрушает крупные клетки («отрицательная селекция»), что занижает RDW? Спорно.
- **Pearls:**
 - «Пограничное значение 11.5 %» — точка разделения между талассемией и ЖДА при MCV < 80 (Hoffbrand 2020).
 - При *ХБП* RDW часто L1 из-за равномерного подавления эритропоэза (эндогенный ЭПО низкий).

### L0 — Оптимум (Референсный диапазон)
- **Числовые границы (взрослые):** 11.5 – 14.5 % (мужчины/женщины); беременные — 12.0 – 15.0 % (BCSH 2014).
- **Числовые границы (педиатрия):** новорождённые — 15.5 – 17.5 %; грудные — 14.0 – 16.5 %; 1-12 лет — 13.5 – 15.5 %; подростки — 13.0 – 15.0 %.
- **Скорость наступления имеет значение?** Нет. Это норма.
- **Что это значит физиологически:** Здоровая гетерогенность ретикулоцитов и зрелых эритроцитов — отражает нормальный костномозговой ответ и сбалансированный пул.
- **Дифряд (топ-7 причин):** Отсутствует (норма). Исключение — компенсированные состояния (например, при ХБП с ЭПО-терапией, если RDW корригируется до нормы).
- **Опасность сейчас:** Нет.
- **Минимальный набор подтверждающих тестов:** Не требуется.
- **Подсказки на скрытое:** Если все показатели в норме — клиническая норма.
- **Pitfalls / artefacts:** *Липемия* — может увеличить MCV (за счёт хиломикронов → ложное ↑ RDW). Требуется коррекция плазмы (ICSH 2014).
- **Pearls:**
 - «Золотой стандарт» — RDW-CV 12–13 % при MCV 85–95 фл — наиболее здоровые (Williams 2020).
 - У *профессиональных доноров* RDW может быть слегка снижен (L1) — не патология, а физиологическая адаптация (ретикулоциты N).

### H1 — Повышенный анизоцитоз (умеренный/выраженный)
- **Числовые границы (взрослые):** 14.5 – 22.0 % (мужчины/женщины); беременные — 15.0 – 23.0 %.
- **Числовые границы (педиатрия):** новорождённые — 17.5 – 22.0 %; грудные — 16.5 – 20.0 %; 1-12 лет — 15.5 – 19.5 %; подростки — 15.0 – 19.0 %.
- **Скорость наступления имеет значение?** Да. *Острое* повышение (часы-дни) → *гемолиз* или *постгеморрагическая анемия*. *Хроническое* — *дефицит железа*, *мегалобластоз*, *воспаление*.
- **Что это значит физиологически:** Гетерогенность эритроцитов по объёму (анизоцитоз). Отражает неэффективный эритропоэз (фолиево-В12 дефицит, гиперрегенерацию с выбросом макроретикулоцитов, железодефицитную с микроцитами) или сосуществование разных популяций («популяционный хаос»).
- **Дифряд (топ-7 причин):**
| Причина | Частота | Ключевые ассоциации |
|---------|---------|---------------------|
| Железодефицитная анемия (ЖДА) | Common | MCV ↓, MCH ↓, ферритин ↓↓↓, повышение насыщ.трансферрина? ↓ |
| Мегалобластная анемия (B12/фолат) | Common | MCV ↑, MCH ↑, гиперсегментация нейтрофилов, ↓ LDH |
| Гемолитические анемии (в т.ч. AIHA, микросфероцитоз) | Common | Ретикулоциты ↑, билирубин ↑, LDH ↑ |
| Постгеморрагическая анемия (острая) | Common | Ретикулоциты ↑, MCV ↑ (за счёт ретикулоцитов), тромбоциты ↑ |
| Смешанный дефицит (Fe + B12, Fe + фолат) | Uncommon | MCV пограничный, двухмодальный гистограмма |
| Миелодиспластический синдром (МДС) | Uncommon | Макроцитоз, дизплазия, цитопении |
| Начало терапии ЭПО (репарация) | Common | Появление нормобластов, ретикулоцитоз |
- **Опасность сейчас:** Зависит от степени анемии. При *тяжёлой гемолитической анемии* — риск *тромбозов* (при AIHA — гемоглобин < 60 г/л), *апластических кризов* при парвовирусе B19 (микросфероцитоз). При *ЖДА* — *синдром пикацизма*, риск *инсультов* (особенно у детей) при Hb < 70 г/л (BCSH 2020).
- **Минимальный набор подтверждающих тестов:** Ретикулоциты, MCV, MCH, ферритин, В12, фолаты (Эритроцитарный), LDH, билирубин (прямой/непрямой), электрофорез гемоглобина, прямой антиглобулиновый тест.
- **Подсказки на скрытое:** *MCV* — ключ: MCV ↓ → ЖДА/талассемия; MCV ↑ → В12/фолат/МДС; MCV N → гемолиз, постгеморрагическая. *Ретикулоцитарный индекс*: < 2 → гипопролиферативная (дефицит), ≥ 2 → гемолиз/кровопотеря.
- **Pitfalls / artefacts:** *Холодовые агглютинины* — ложное ↑ MCV + ↑ RDW. Требуется подогрев крови (температура 37 °C) (ICSH 2014). *Липемия* — ложное ↑ MCV и RDW. *EDTA* — хранение > 4 часов → набухание клеток (псевдо ↑ MCV
