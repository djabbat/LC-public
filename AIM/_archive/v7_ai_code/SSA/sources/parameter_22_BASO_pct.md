### L2 — Критическая абазофилия (life-threatening low)
- **Числовые границы (взрослые):** мужчины/женщины/беременные — 0% (неопределяемое количество в мазке). Абсолютное число: < 0.01 х 10⁹/л.
- **Числовые границы (педиатрия):** новорождённые / грудные / 1-12 лет / подростки — 0% (неопределяемое количество).
- **Скорость наступления имеет значение?** Да. Острое исчезновение (часы-дни) — маркер агранулоцитоза или острого лейкоза с замещением костного мозга. Хроническое отсутствие — апластическая анемия.
- **Что это значит физиологически:** Почти полное отсутствие продукции базофилов в костном мозге либо массивное разрушение на периферии. Отражает глубокое угнетение гранулопоэза.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
| :--- | :--- | :--- |
| Апластическая анемия | Uncommon | Панцитопения, низкий ретикулоцитарный индекс |
| Острый лейкоз (особенно M0–M2) | Common | Бласты в периферической крови, цитопении |
| МДС с гипоплазией | Uncommon | Дисплазия в других ростках, цитогенетические аномалии |
| Миелотоксическая химиотерапия | Common | Прием цитостатиков, лучевая терапия |
| Агранулоцитоз (лекарственный) | Common | Селективное исчезновение нейтрофилов + базофилов |
| Терминальная стадия ХМЛ (бластный криз) | Rare | Гиперлейкоцитоз, базофилия сменяется абазофилией |
| Вирусная инфекция (аплазия на фоне EBV, парвовирус B19) | Rare | Транзиторная, обычно на фоне лихорадки |

- **Опасность сейчас:** Риск бактериального сепсиса (особенно грамотрицательная флора) в течение 24–48 часов из-за сопутствующей нейтропении. Кровотечения — если тромбоциты также < 20 х 10⁹/л. Причина: отсутствие гранулоцитов + дефицит тромбоцитов.
- **Минимальный набор подтверждающих тестов:** 1. Трепанобиопсия костного мозга с гистологией. 2. Иммунофенотипирование костного мозга (CD123, CD203c, FceRI). 3. Цитогенетика (del5q, del7, BCR-ABL). 4. Вирусологический скрининг (EBV, CMV, парвовирус B19). 5. Анализ на лекарственные антитела / повреждение (drug-induced).
- **Подсказки на скрытое:** Проверить ANC (абсолютное число нейтрофилов) — если < 0.5 х 10⁹/л, то агранулоцитоз. Снижение ретикулоцитов — аплазия. MCV часто повышен при апластической анемии. Абсолютное число эозинофилов — обычно также снижено.
- **Pitfalls / artefacts:** 1. ЭДТА-зависимая дегрануляция базофилов (редко, но возможно — мазок свежей крови с микроскопией). 2. Разложение образца > 4 часов при комнатной температуре. 3. Цирроз печени с гиперспленизмом — панцитопения, включая базофилы.
- **Pearls:** 1. "Правило Виноградова": при апластической анемии базофилы исчезают одними из первых среди гранулоцитов (Hoffbrand 2020). 2. Отсутствие базофилов + наличие клональной цитопении — маркер тяжелой МДС (WHO 2011). 3. Абазофилия на фоне лейкемида — исключить бластный криз ХМЛ с доминантой не базофильной линии (Wintrobe 2019).

### L1 — Умеренное снижение базофилов (клинически значимая базопения)
- **Числовые границы (взрослые):** мужчины/женщины/беременные — 0.1–0.4%. Абсолютное число: 0.01–0.04 х 10⁹/л.
- **Числовые границы (педиатрия):** новорождённые / грудные / 1-12 лет / подростки — < 0.5% (аналогично взрослым).
- **Скорость наступления имеет значение?** Умеренно. Острое снижение (дни) — реакция на стресс/глюкокортикоиды. Хроническое — миелотоксичность или гиперспленизм.
- **Что это значит физиологически:** Умеренное подавление образования базофилов в костном мозге или ускоренный апоптоз на периферии. Часто отражает системное воспаление.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
| :--- | :--- | :--- |
| Терапия глюкокортикоидами (дексаметазон, преднизолон) | Common | Подавление ИЛ-3/ИЛ-5 |
| Тиреотоксикоз | Uncommon | Повышенный Т3/Т4, сниженный ТТГ |
| Острая инфекция (сепсис, пневмония) | Common | Нейтрофильный лейкоцитоз, сдвиг влево |
| Гиперкортицизм (болезнь/синдром Кушинга) | Rare | АКТГ-зависимое или АКТГ-независимое |
| Хроническая болезнь почек (ХБП 3-5) | Uncommon | Гиперпаратиреоз, уремия |
| Острая фаза лучевой болезни | Rare | Доза ≥ 1 Гр |
| Химиотерапия (таксан, платина) | Common | Транзиторно, до 14 дней |

- **Опасность сейчас:** Риск мал. При сепсисе — ANCN < 0.5 х 10⁹/л. Базопения сама по себе не вызывает острых событий. Требует коррекции первопричины.
- **Минимальный набор подтверждающих тестов:** 1. Повторное исследование с оценкой абсолютного числа (автомат + мазок). 2. ТТГ, Т3, Т4. 3. Кортизол сыворотки (утренний/ночной). 4. BUN, креатинин, расчет СКФ. 5. Культура крови (если лихорадка). 6. Биопсия костного мозга — только при длительной (> 3 мес) базопении + других цитопениях.
- **Подсказки на скрытое:** Снижение эозинофилов (зеркальная базопения) — стероидный эффект. Повышение нейтрофилов — воспаление. Тромбоцитоз — реактивный. Анемия + нейтропения + базопения — гипоплазия.
- **Pitfalls / artefacts:** 1. Анализ не в протоколе: базофилы плохо дифференцируются при низком общем лейкоцитозе. 2. Лекарственная интерференция (НПВП, антиконвульсанты). 3. Спящий или неправильный референс (лаборатория может использовать 0–1% как норму — не заметят снижение).
- **Pearls:** 1. Базопения на фоне глюкокортикоидов — дозозависимый эффект; отмена восстанавливает через 48–72 часа (BCSH 2016). 2. Сочетание базопении + лимфопении — классический симптом инфекционного стресса (Hoffbrand 2020). 3. Локальная базопения (изолированная) при ХБП — маркер неблагоприятного прогноза при трансплантации (Wintrobe 2019).
