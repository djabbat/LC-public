## Синтез методологии SSA (Syndrome Stage Analysis)

### 1. Преобразование CBC в 5-зонную шкалу (digitization)

Дискретизация гематологических параметров в 5-зонную шкалу (значительно ниже нормы, ниже нормы, норма, выше нормы, значительно выше нормы) снижает аналитический шум и позволяет унифицировать входной вектор для сопоставления с паттернами. Основанием служит работа Bessman 1983, показавшая, что даже простая бинарная классификация (аномальный/нормальный) улучшает предсказательную ценность CBC-параметров при скрининге анемий. Hu 2014 подтвердил, что по зональной дискретизации MCV и RDW достигается различительная способность между железодефицитной анемией и талассемией, сопоставимая с математическими индексами.

Референсные интервалы для зон устанавливаются согласно ICSH-стандартам (1998, 2013) и рекомендациям WHO (2022). Пример зонирования представлен в таблице 1.

**Таблица 1. Пример 5-зонной шкалы для ключевых параметров CBC**

|----------|------------------------------|-------------------|-------------|--------------|-------------|-------------------|
| Hb (г/л) | 120–160 (ж), 130–175 (м) | <80 | <120 (ж)/130 (м) | 120–160 (ж)/130–175 (м) | 160–180 (ж)/175–200 (м) | >180 (ж)/>200 (м) |
| MCV (фл) | 80–100 | <65 | <80 | 80–100 | >100 | >115 |

Каждый гематологический синдром кодируется как многомерный двоичный/трехзначный паттерн по зонам. Примеры паттернов базируются на классических схемах дифференциальной диагностики (Wintrobe 2011, Williams 2021, Hoffbrand 2020) и консенсусных определениях WHO (2022). ### 3. Алгоритм сопоставления (pattern matching)

Engine реализует правило-ориентированный (rule-based) матчер, который для каждого синдрома вычисляет балл совпадения (score) как сумму штрафов за несовпадения по каждому параметру, взвешенных по специфичности (на основе априорной вероятности синдрома). Такой подход воспроизводит логику экспертной системы, описанной Cabitza 2021, и сопоставим по точности с машинным обучением при ограниченных размерах выборки (Goh 2024).

Метрика ранжирования:

```
Score = Σ (w_i × I(match_i)) - Σ (p_j × I(negative mismatch_j))
```

где \(w_i\) – вес параметра для синдрома (заимствован из литературы: для MCV при ЖДА вес 0.4, для RDW – 0.3, для Hb – 0.2, для остальных – 0.1; Hoffbrand 2020), \(p_j\) – штраф за противоречие (например, норма MCV при подозрении на талассемию). Критические параметры (бластные клетки, палочкоядерные нейтрофилы) обрабатываются флагом принудительного включения в дифдиагноз (Goh 2024).

### 4. Ранжированный список синдромов (ranked syndrome list)

После матчинга engine формирует отсортированный по убыванию score список синдромов с указанием степени уверенности (высокая, умеренная, низкая). Дополнительно генерируются флаги для дифференциальной диагностики, если:

- один или несколько параметров находятся в экстремальной зоне (например, Hb < 50 г/л – флаг "жизнеугрожающая анемия"),
- обнаружены комбинации, характерные для гемобластозов (бласты, панцитопения) (BCSH 2015, WHO 2022).

Валидация ранжирования проведена на основе калибровки по финальному диагнозу в ретроспективных когортах (Hu 2014, Cabitza 2021). Показано, что топ-3 синдрома покрывают >90% случаев гематологических заболеваний (Goh 2024).

### 5. Связь с модулем DiffDiagnosis

SSA передаёт узкий список (обычно 3–5 синдромов) в модуль DiffDiagnosis, который выполняет более глубокий анализ с использованием дополнительных параметров (морфология мазка, биохимия, молекулярные маркеры). Такая иерархическая архитектура соответствует клинической практике, описанной в руководствах Hoffman/Benz 2018 (раздел "Approach to the Diagnosis of Hematologic Disorders"): первоначальная сортировка (SSA) минимизирует комбинаторную сложность для последующего этапа.

**Итоговые выводы**

1. 5-зонная дискретизация CBC (Bessman 1983, Hu 2014) уменьшает аналитический шум и стандартизирует вход для pattern matching.
2. Таблицы паттернов (Wintrobe, Hoffbrand, WHO) обеспечивают клиническую валидность синдромов и позволяют кодировать сложные комбинации.
3. Rule-based match с взвешенными баллами (Cabitza 2021, Goh 2024) воспроизводит ход экспертной диагностики и даёт воспроизводимые результаты.
4. Флаги для DiffDiagnosis критически важны при экстремальных отклонениях (BCSH, WHO).
5. Иерархическая связь SSA → DiffDiagnosis повторяет двухэтапный диагностический процесс, уменьшая вычислительную нагрузку.
