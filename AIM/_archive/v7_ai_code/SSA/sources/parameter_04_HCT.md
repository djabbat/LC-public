### L2 — Критически низкий гематокрит (декомпенсированная анемия)

- **Числовые границы (взрослые):** Мужчины < 20%; Женщины (небеременные) < 18%; Беременные < 15% (Hoffbrand 2020, Williams 2018). Единицы: % (SI: 0.20/0.18/0.15 L/L).
- **Числовые границы (педиатрия):** Новорождённые (<1 нед) < 30%; Грудные (2-6 мес) < 18%; 1-12 лет < 20%; Подростки: < 18% (девочки), < 20% (мальчики) (BCHS 2016, Williams 2018).
- **Скорость наступления имеет значение?** Да. Острое снижение (геморрагия, гемолиз) — преобладает гиповолемия (HCT не отражает ОЦК) и гипоксия тканей при сохранном ОЦК. Хроническое снижение — адаптация (сдвиг кривой диссоциации O2, увеличение 2,3-DFG), но риск декомпенсации при сердечной нагрузке.
- **Что это значит физиологически:** Критическое снижение кислородной ёмкости крови. Нарушение доставки O₂ относительно метаболических потребностей. Порог анаэробного метаболизма и тканевой гипоксии (Wintrobe 2014).
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---|---|---|
| Острая массивная кровопотеря | Common (ург. хирургия/травма) | Травма, ЖКК, разрыв аневризмы |
| Гипергемолиз (гемолитический криз) | Common (аутоиммунный) | Лихорадка, желтуха, Hb-урия |
| Аплазия/гипоплазия костного мозга (KKM) | Uncommon (AA/криз при ПНГ) | Нейтропения, тромбоцитопения |
| Дефицит B12/фолатов (панцитопения) | Uncommon (В12-дефицитная анемия) | Глоссит, сенсорная атаксия, MCV > 120 |
| Гемоглобинопатия (талассемия major) | Rare (классич. Боль в костях) | Гепатоспленомегалия, микроцитоз |
| Миелофтиз (миелосупрессия химио/радио) | Uncommon (после ХТ) | Панцитопения, фебрильная нейтропения |
| Гемодилюция (гиперпротеинемия/беременность) | Common (функциональная) | Разведение, нормальный ОЦК эритроцитов |

- **Опасность сейчас:**
 - Часы: анаэробный метаболизм миокарда → ишемия, инфаркт; лактат-ацидоз; кома; шок.
 - Сутки: прогрессирование гипоксии ЦНС, ПОН.
 - Недели: усугубление застойной сердечной недостаточности (высокий выброс).
- **Минимальный набор подтверждающих тестов:**
 - Ретикулоциты (скорректированный / индекс продукции RPI).
 - MCV, MCH, RDW.
 - Общий билирубин, ЛДГ, гаптоглобин (гемолиз).
 - Креатинин, мочевина (см. кровопотеря → азотемия).
 - Эвалюация потерь: кал на скрытую кровь/эритроциты.
- **Подсказки на скрытое:**
 - MCV > 120 + макроцитоз +/– тромбоцитопения → B12/фолаты.
 - MCV < 80 + RDW низкий → ß-талассемия trait/HbH.
 - Ретикулоциты низкие (RPI < 2) → аплазия/дефицит.
 - Ретикулоциты высокие (RPI > 2, RDW высокий) → гемолиз/кровопотеря.
- **Pitfalls / artefacts:**
 - *Холодовые агглютинины*: завышение MCV, занижение RBC → HCT артефактино низок (инкубация 37°C).
 - *Гликированный Hb-анализ* не влияет.
 - *Парапротеинемия*: гипервязкость может завысить HCT (объём клеток сохраняется, но центрифугирование нарушено).
- **Pearls:**
 - *Правило Виноградова:* HCT < 20% в сочетании с бледностью и тахикардией покоя — показание для немедленной трансфузии (Hoffbrand 2017, Кассирский И.А., 1956).
 - *Hoffbrand:* При острой кровопотере HCT может оставаться нормальным в первые 6-12 часов до гемодилюции; ориентируйся на клинику.
 - *Wintrobe:* HCT 20% у кошек — сопоставимо с HCT 12% у людей (метаболизм); не экстраполируй.

### L1 — Сниженный гематокрит (компенсированная анемия)

- **Числовые границы (взрослые):** Мужчины: 20–39%; Женщины (небеременные): 18–35%; Беременные: 15–31% (WHO 2011, Williams 2018). Единицы: %.
- **Числовые границы (педиатрия):** Новорождённые: 30–44%; Грудные: 18–30%; 1-12 лет: 20–34%; Подростки: 18–35% (BCHS 2016).
- **Скорость наступления имеет значение?** Да. Хроническое снижение (недели-месяцы) компенсируется адаптацией (повышение 2,3-DFG, сердечный выброс). Острое (часы) — плохая переносимость.
- **Что это значит физиологически:** Умеренная тканевая гипоксия, компенсируемая тахикардией, ренальной продукцией ЭПО (до 5-10 кратной). Может быть бессимптомной (до HCT 25-30%) (Hoffbrand 2020).
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---|---|---|
| Железодефицитная анемия (ЖДА) | Very common (любой возраст) | Микроцитоз, гипохромия, повышение TIBC, ↓ ферритин |
| Анемия хронических заболеваний (АХЗ) | Very common (воспаление, онко, аутоиммун) | Нормоцитоз, нормальный TIBC, повышение ферритина (парадокс) |
| Талассемия minor (носительство) | Common (этнические группы: Средиземноморье, ЮВА) | Микроцитоз, RDW нормальный, повышение HbA₂ > 3.5% |
| Дефицит B12/фолатов | Uncommon (дети, элитная диета, мальабсорбция) | Макроцитоз, глоссит, панцитопения (поздние стадии) |
| Гемолитические анемии (наслед. сфероцитоз, Г6ФД, АИГА) | Uncommon (рецидивы) | Ретикулоцитоз, повышение ЛДГ/билирубин, ↓ гаптоглобин |
| Аплазия/миелодисплазия (MDS) | Uncommon (пожилые) | Панцитопения, макроцитоз (при MDS), бласты (при AA) |
| Хроническая почечная недостаточность (ХПН) | Common (3-4 стадия) | Нормоцитарная анемия, ↓ эндогенного ЭПО, ↑ креатинин |

- **Опасность сейчас:**
 - Часы/сутки: низкий риск прямого вреда при HCT > 25% (> 3g/dL Hb). Риск декомпенсации при физической нагрузке, хронической CHD.
 - Недели: прогрессирование до L2 (кровопотеря, гемолиз), сердечная недостаточность (HCT < 30%).
- **Минимальный набор подтверждающих тестов:**
 - Ретикулоциты (RPI, корректированный).
 - MCV, MCH, MCHC, RDW.
 - Ферритин, % насыщения трансферрина (< 16% — ЖДА).
 - Креатинин, СКФ.
 - Электрофорез Hb (исключить талассемию).
 - B12, фолаты (RBC фолат предпочтительнее сывороточного).
- **Подсказки на скрытое:**
 - MCV < 72 fL + RDW нормальный → талассемия / АХЗ + железо.
 - MCV < 72 fL + RDW повышен → ЖДА / талассемия + сидеробластная анемия.
 - MCV > 95 fL + макроовалоциты + нейтрофилы гиберсегментация → B
