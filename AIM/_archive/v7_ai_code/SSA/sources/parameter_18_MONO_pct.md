### L2 — Моноцитопения критическая (life-threatening low)
- **Числовые границы (взрослые):** < 0.5% (относительное) при абсолютном содержании < 0.1 × 10⁹/л; мужчины/женщины/беременные — одинаково (WHO 2022, Coulter 2017).
- **Числовые границы (педиатрия):** новорождённые < 1% (референс 2–10%); грудные < 0.5%; 1–12 лет и подростки < 0.5% (Bain 2015).
- **Скорость наступления имеет значение:** острое падение (часы–дни) — риск фульминантной инфекции (особенно грибковой, микобактериальной); постепенное (недели–месяцы) — аплазия/гемопатия.
- **Что это значит физиологически:** истощение или угнетение моноцитарного ростка костного мозга; нарушение фагоцитоза, антигенпрезентации, хемотаксиса — компрометация врождённого и адаптивного иммунитета (Hoffbrand 2020).
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|----------|---------------------|
| Апластическая анемия / тяжёлая миелотоксичность (химиотерапия, облучение) | Common + Therapy-induced | Панцитопения, ретикулоциты ↓, нейтропения |
| Острый лейкоз (особенно AML M5b) с вытеснением нормального кроветворения | Common | Бласты, цитопении, DIC |
| Сепсис/синдром активации макрофагов — потребление моноцитов | Uncommon in acute phase | ФДГ, цитокиновый шторм, гемофагоцитоз |
| Вирусные инфекции (ВИЧ, Эпштейна–Барр, CMV) на стадии анергии | Uncommon | Лейкопения, лимфопения, CD4+ ↓ |
| Тяжёлая стероидная терапия (декомпенсация) | Common (iatrogenic) | Дексаметазон, метилпреднизолон > 2 мг/кг |
| Гемофагоцитарные лимфогистиоцитозы | Rare | Ферритин > 500, триглицериды ↑, NK-активность ↓ |
| Врождённые моноцитопении (GATA2 deficiency, MLL deficiency) | Very rare | Дисморфии, рецидивирующие инфекции |

- **Опасность сейчас:** в ближайшие 24–72 ч — фатальная бактериальная/грибковая инфекция (атипичные микобактерии, аспергилл); при отсутствии заместительной терапии — септический шок (Wintrobe 2018).
- **Минимальный набор подтверждающих тестов:** ПЦР на цитомегаловирус, Эпштейна–Барр, ВИЧ; проточная цитометрия крови (CD14, CD16); костный мозг (аспират+биопсия); уровень ферритина, триглицеридов (исключить гемофагоцитоз).
- **Подсказки на скрытое:** снижение абсолютного числа моноцитов (MONO#) — объективнее; нейтропения — риск сепсиса; ретикулоцитопения — аплазия; лимфопения — вирус.
- **Pitfalls / artefacts:** липемия → занижение % моноцитов; использование стероидов перед забором (транзиторно); EDTA-зависимая агрегация нейтрофилов — относительная моноцитопения; хранения образца > 4 ч — гибель моноцитов (ICSH 2017).
- **Pearls:** 1) Правило Виноградова: моноцитопения + нейтропения + лимфопения = криз аплазии. 2) При абсолютной моноцитопении < 0.1 × 10⁹/Л показана изоляция и профилактика грибков (Hoffbrand 2020). 3) У пациентов на химиотерапии моноцитопения коррелирует с риском фибрильной нейтропении (Williams Hematology 2020).

### L1 — Моноцитопения лёгкая/субоптимальная
- **Числовые границы (взрослые):** 0.5–1.5% (относительно) при абсолютном 0.1–0.3 × 10⁹/Л; мужчины/женщины/беременные одинаково.
- **Числовые границы (педиатрия):** новорождённые 1–2%; грудные 0.5–1.5%; 1–12 лет 0.5–1.5%; подростки 0.5–1.5%.
- **Скорость наступления имеет значение:** острая (≥ 3 дней) — стресс-индуцированная; хроническая (месяцы) — миелофиброз, MDS.
- **Что это значит физиологически:** снижение выброса моноцитов из костного мозга при длительном воспалении, токсическом воздействии или дефиците предшественников.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|----------|---------------------|
| Длительная глюкокортикоидная терапия | Common | Кушинг, сахарный диабет, лимфопения |
| Хроническая почечная недостаточность (уремия) | Common | Креатинин > 450, анемия, эритропоэтин ↓ |
| Миелодиспластический синдром (MDS) с моноцитопенией | Uncommon | Дисплазия, цитопении, кольцевые сидеробласты |
| Тяжёлый дефицит железа (редко) | Rare | MCV ↓, ферритин ↓, анемия |
| Витамин B12/фолатов перед лечением | Uncommon | MCV ↑, нейтропения, гомоцистеин |
| Саркоидоз (в стадии фиброза) | Rare | ACE ↑, гиперкальциемия, лимфаденопатия |
| Хроническая болезнь печени (цирроз) | Common | Гиперспленизм, тромбоцитопения, лейкопения |

- **Опасность сейчас:** риск бактериальных инфекций (стафилококк, стрептококк) при комбинации с нейтропенией; остеомиелит, абсцессы. Ближайшие недели — прогрессия основного заболевания.
- **Минимальный набор подтверждающих тестов:** абсолютное число лейкоцитов + формула, ферритин, B12/фолаты, креатинин, кортизол, мазок периферической крови.
- **Подсказки на скрытое:** панцитопения → MDS/AA; лимфоцитоз → вирус; моноцитоз в прошлом → динамика.
- **Pitfalls / artefacts:** хранение крови > 6 ч → распад моноцитов; ЭДТА при низком pH; лейкоз-ассоциированные псевдопатологии (агрегаты). 
- **Pearls:** 1) Моноцитопения + тромбоцитопения + макроцитоз — часто MDS (WHO 2011). 2) При 10-дневной терапии дексаметазоном моноциты падают на 60–80% (Checketts 2016). 3) Правило Кассирского: хроническая моноцитопения — метка гипоплазии костного мозга.

### L0 — Оптимум (референс)
- **Числовые границы (взрослые):** 2.0–10.0% (относительно); 0.2–0.8 × 10⁹/Л абсолютно. Мужчины/женщины/беременные — одинаково (референс ICSH 2014). Беременность — 2–9% (транзиторный нейтрофилез).
- **Числовые границы (педиатрия):** новорождённые 2–10%; грудные 2–8%; 1–12 лет 2–8%; подростки 2–8% (Bain 2015).
- **Скорость наступления имеет значение:** любое изменение вне референса требует оценки.
- **Что это значит физиологически:** адекватный моноцитарный резерв, баланс продукции и потребления, нормальная иммунная реакция.
- **Дифряд:** не требуется (в норме). Исключение — скрытое состояние, маскированное лейкоцитозом.
- **Опасность сейчас:** нет.
- **Минимальный набор подтверждающих тестов:** при случайном отклонении — повтор.
- **Подсказки на скрытое:** при лейкоцитозе > 50×10⁹/Л относительный % может быть ложно низким — считать абсолютное.
- **Pitfalls / artefacts:** липемия → искажение рассеяния; аномальное созревание.
- **Pearls:** 1) Истинно нормальный % моноцитов — 3–5% в ручном подсчете (Bain 2015). 2) Автоматические анализаторы часто завышают моноциты при наличии миелоцитов/метамиелоцитов (ICSH 2017). 3) Оптимум не исключает субклинической инфекции.

### H1 — Моноцитоз умеренный
- **Числовые границы (взрослые):** 10–20% (относительно); 0.8–2.0 × 10⁹/Л абсолютно. Половых различий нет. Беременные — 10–18% (возможен физиологический сдвиг).
- **Числовые границы (педиатрия):** новорождённые 10–20%; грудные 10–18%; 1–12 лет 10–18%; подростки 10–15%.
- **Скорость наступления имеет значение:** острое (1–3 дня) — инфекционный моноцитоз; хроническое (недели–месяцы) — MDS/CMML, гранулематозы.
- **Что это значит физиологически:** активация моноцитарной линии в ответ на хроническую инфекцию, тканевое повреждение, миелопролиферацию.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|----------|---------------------|
| Туберкулёз (лёгочный/внелёгочный) | Common | Лихорадка, ночная потливость, кашель, ПЦР+ |
| Подострый бактериальный эндокардит | Common | Шум, эмболы, гемокультура +, IgM РФ |
| Миелодисплазия/миелопролиферация (CMML) | Uncommon | Мон-ы >1000, дисплазия, ЯПУ, бласты <20% |
| Хронический миелоидный лейкоз (в ранней фазе) | Common | Нейтрофилёз, базофилия, Ph+ |
| Саркоидоз | Uncommon | ACE↑, лимфаденопатия, гистология |
| Системная красная волчанка | Uncommon | Антитела anti-dsDNA, васкулит |
| Лекарственная реакция (DRESS, фенитоин) | Rare | Кожная сыпь, эозинофилия, лимфаденопатия |

- **Опасность сейчас:** при истинном CMML — риск бластной трансформации (до 15%/год). При эндокардите — эмболии, клапанная деструкция. При туберкулёзе — дыхательная недостаточность.
- **Минимальный набор подтверждающих тестов:** абсолютный моноцитоз (MONO#), мазок крови, ПЦР на M. tuberculosis, посев крови, ACE, ANA, проточная цитометрия на CMML (CD14+CD16+), костный мозг при >1000 абсолютных.
- **Подсказки на скрытое:** базофилия → CML; эозинофилия → DRESS; нейтропения → MDS; тромбоцитоз → Ph- негативные МПН.
- **Pitfalls / artefacts:** лимфоциты с псевдо-моноцитарной морфологией; young neutrophils → ошибочный моноцитоз; транзиторный подъём при стрессе (катехоламины).
- **Pearls:** 1) Абсолютный моноцитоз > 1 × 10⁹/Л в течение > 3 месяцев — подозрение на CMML (WHO 2011). 2) Моноцитоз >15% при туберкулёзе — неблагоприятный прогноз (Лука Б. 2010). 3) Правило Файнштейна: дифференцировать инфекционный моноцитоз от опухолевого помогает анемия и тромбоцитоз.

### H2 —
