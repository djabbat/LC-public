### L2 — Критическая иммунотромбоцитопения (деплеция тромбоцитарно-нейтрофильного резерва)

- **Числовые границы (взрослые):** SII < 50 × 10⁹/л (при условии NLR < 1,0 и PLT < 50 × 10⁹/л). Единицы: SII безразмерный (×10⁹/л). Для беременных — аналогично, с поправкой на гестационную тромбоцитопению (PLT > 70 × 10⁹/л обычно безопасны). (Hoffbrand 2020)
- **Числовые границы (педиатрия):** Новорождённые — SII < 30 × 10⁹/л (NLR часто < 0,5, PLT < 50 × 10⁹/л). Грудные (1-12 мес) — SII < 40 × 10⁹/л. 1-12 лет — SII < 50 × 10⁹/л. Подростки — как взрослые.
- **Скорость наступления имеет значение?** Да. Острая деплеция (часы-дни) — септический шок, ТТП/ГУС, острая лекарственная реакция. Хроническая (недели-месяцы) — апластическая анемия, ВИЧ, миелофтиз.
- **Что это значит физиологически:** Истощение КМ резерва нейтрофилов и тромбоцитов. SII ↓ указывает на одновременную недостаточность миелоидного ростка (нейтропения) и мегакариоцитарного (тромбоцитопения). Лимфоциты могут быть нормальны или снижены (диффузная лимфопения при аплазии).
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
| :--- | :--- | :--- |
| Тяжёлый сепсис / септический шок | Common | Потребление тромбоцитов + угнетение КМ (TNF-α, IL-6) |
| Тромботическая тромбоцитопеническая пурпура (ТТП) | Uncommon | ADAMTS13 <10%, микроангиопатическая гемолитическая анемия |
| Апластическая анемия (тяжёлая форма) | Uncommon | Панцитопения, гипоцеллюлярный КМ, клональная мутация |
| Острый промиелоцитарный лейкоз (APL) | Rare | t(15;17), DIC, геморрагический синдром |
| Гемофагоцитарный лимфогистиоцитоз (HLH) | Rare | Фебрильная цитопения, ферритин >500 нг/мл, CD25 ↑ |
| Хроническая болезнь почек (терминальная стадия) | Common | Уремия → ингибиция КМ, гемодиализ → адгезия PLT |
| Лекарственная цитопения (химиопрепараты, интерфероны) | Common | Миелосупрессия (тимоглобулин, кладрибин, немасштабируемые дозы) |

- **Опасность сейчас:** Спонтанное внутричерепное кровоизлияние (PLT < 10 × 10⁹/л) в ближайшие часы. Фебрильная нейтропения (ANC < 0,5 × 10⁹/л) → бактериемия, septic shock. Острый тромбоз (ТТП/ГУС) → нестабильность гемодинамики.
- **Минимальный набор подтверждающих тестов:** ФЛУО (р-тр 120 мин) для неходжкинских лимфом, ТТГ, кортизол, АКТГ — исключить вторичную недостаточность коры надпочечников. Мазок периферической крови (поиск бластов, шизоцитов, миелофиброза). ADAMTS13 активность. КМ биопсия (аспират + трепанобиопсия с иммунофенотипированием).
- **Подсказки на скрытое:** MCV >100 фл → дефицит B12/фолатов. RDW ↑ → гемолиз (при ТТП). Ретикулоциты ↓ → аплазия. Моноциты ↓ → агранулоцитоз.
- **Pitfalls / artefacts:**
 - EDTA-зависимая псевдотромбоцитопения (агрегация in vitro) → забор в цитрат.
 - Холодовые агглютинины → ложное повышение MCV, снижение PLT.
 - Гемолиз при заборе → ложно высокий HB → SII искусственно увеличен.
 - Lipemia → искажение оптического счета PLT → использовать ИФА.
 - Leucocytosis >50 × 10⁹/л → занижает PLT при импедансном счете → коррекция по формуле (PLT = измеренное × WBC/100).
- **Pearls:**
 1. Правило Виноградова: при критическом SII и шизоцитах >1% мазка — думай о ТТП до исключения DIC (Hoffbrand 2020).
 2. Wintrobe: SII <50 с ANC <0,5 и PLT <20 — показание для трансфузии тромбоконцентрата и G-CSF.
 3. Если SII <50 на фоне кортикостероидов — исключай оппортунистическую инфекцию (CMV, EBV, аспергилл).

### L1 — Субоптимальный SII (умеренная иммунотромбоцитопения)

- **Числовые границы (взрослые):** 50–200 × 10⁹/л. Для беременных: 50–150 × 10⁹/л (из-за умеренной гестационной тромбоцитопении).
- **Числовые границы (педиатрия):** Новорождённые — 30–100 × 10⁹/л. Грудные — 40–150 × 10⁹/л. 1-12 лет — 50–200 × 10⁹/л. Подростки — 50–200 × 10⁹/л.
- **Скорость наступления имеет значение?** Да. Острое (1-2 нед) — ITP, виремия. Хроническое (мес) — дефицит B12/фолатов, ВИЧ.
- **Что это значит физиологически:** Неполная недостаточность КМ или иммуноопосредованное разрушение. NLR <1,5 + PLT 50–100 × 10⁹/л → компенсированная тромбоцитопения. SII >50 — сохранён минимальный резерв гемостаза.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
| :--- | :--- | :--- |
| Идиопатическая тромбоцитопеническая пурпура (ITP) | Common | Аутоиммунная, без нейтропении, PLT 20-80 × 10⁹/л |
| Вирусные инфекции (CMV, EBV, COVID-19) | Common | Лихорадка, лимфопения, NLR ↑ в фазу кризиса |
| Дефицит витамина B12/фолатов | Common | MCV >100, RDW ↑, анемия |
| Миелодиспластический синдром (MDS) | Uncommon | Цитопения, макроцитоз, клональная эволюция |
| Хроническая болезнь почек (3-4 стадия) | Common | Уремия, вторичный гиперпаратиреоз |
| Лекарственная тромбоцитопения | Common | Гепарин, ванкомицин, фуросемид, парацетамол |
| Спленомегалия (любого генеза) | Common | Секвестрация PLT, SLD ↑, PDW ↓ |

- **Опасность сейчас:** Геморрагический синдром (петехии, экхимозы) при PLT 20-50 × 10⁹/л. Риск в/ч кровоизлияния низкий (отсутствует при PLT >30 × 10⁹/л). Нейтропения не выражена.
- **Минимальный набор подтверждающих тестов:** В12, сывороточный фолат, эритроцитарный фолат. Антитела к тромбоцитам (PAIgG). Серология на HIV, HCV, HBV, CMV. УЗИ брюшной полости (спленомегалия, эхо-признаки портальной гипертензии).
- **Подсказки на скрытое:** NLR <1,5 + абсолютная лимфопения (<1,0 × 10⁹/л) → ВИЧ. MCV >100 → B12/фолаты. PLT ↑ после спленэктомии при ITP.
- **Pitfalls / artefacts:**
 - Холодовые агглютинины — ложная тромбоцитопения.
 - EDTA-агрегация — проверка с цитратом.
 - Гемолиз — ложно сниженные PLT при импедансе.
 - Липемия — завышение MPV, неправильный PLT.
- **Pearls:**
 1. Wintrobe: при SII 50-200 + MCV >100 — сначала коррекция B12/фолатов, затем переоценка.
 2. Если SII стабилен >3 мес — риск тромбоза низок (компенсация).
 3. У беременных трактовать как проявление ITP, если PLT <70 после 20 недели.

### L0 — Оптимальный SII (нормальный гемостаз и иммунитет)

- **Числовые границы (взрослые):** 200–500 × 10⁹/л. Для беременных: 150–450 × 10⁹/л (из-за гестационной гемодилюции). Единицы: безразмерные.
- **Числовые границы (педиатрия):** Новорождённые — 100–400 × 10⁹/л. Грудные — 150–500 × 10⁹/л. 1-12 лет — 200–500 × 10⁹/л. Подростки — 200–500 × 10⁹/л.
- **Скорость наступления:** Не применимо. Стабильный гомеостаз.
- **Что это значит физиологически:** Полноценная миелопоэтическая функция. PLT и ANC в референсе, лимфоциты здоровы. SII в этом окне указывает на отсутствие активного воспаления, тромбоза или деплеции.
- **Дифряд (топ-7):** Не требуется. Это референсная зона.
- **Опасность сейчас:** Отсутствует.
- **Минимальный набор тестов:** Не требуется. При наличии клинических симптомов — корреляция с NLR.
- **Подсказки на скрытое:** SII стабильно в L0 — исключает миелопролиферативные и миелодиспластические процессы. Если есть подозрение — PLT >400 и ANC >8 — перейти в H1.
- **Pitfalls / artefacts:** Ложно нормальный SII при наличии холодовых агглютининов (PLT in vivo снижены, in vitro — норма). EDTA-агрегация — имитация L0 при маскированной тромбоцитопении.
- **Pearls:**
 1. Wintrobe: идеальный скрининг гомеостаза — SII в L0 — гарантирует отсутствие тромбоцитоза и нейтропении.
 2. При подозрении на раннюю MDS — нормальный SII не исключает клональное поражение (нужно HL-анализ).
