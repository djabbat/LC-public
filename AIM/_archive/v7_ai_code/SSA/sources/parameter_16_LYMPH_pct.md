### L2 — Критическая лимфопения (life-threatening low)

- **Числовые границы (взрослые):** <5% (все группы, включая беременных) — единицы: % от лейкоцитов.
- **Числовые границы (педиатрия):**
 - новорождённые: <10%
 - грудные (1–12 мес): <15%
 - 1–12 лет: <10%
 - подростки (13–18 лет): <5% (адаптировано к взрослым нормам)
- **Скорость наступления имеет значение?** Острое снижение (часы–сутки) — тяжёлый сепсис, массивный выброс кортизола, системное воспаление; хроническое (недели–месяцы) — аплазия, ВИЧ-инфекция, иммуносупрессия.
- **Что это значит физиологически:** Истощение лимфоцитарного пула вследствие чрезмерного апоптоза (сепсис, SIRS), прямого токсического действия (химиотерапия, радиация), разрушения в тимусе/периферических лимфоидных органах (ВИЧ), либо замещения костного мозга (апластическая анемия, лейкозы с подавлением нормального ростка).
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|----------|----------------------|
| Сепсис / SIRS | common | Нейтрофилёз (или нейтропения), тромбоцитопения, лихорадка, положительная гемокультура |
| Химиотерапия / иммуносупрессия | common | Ятрогенная цитопения (нейтропения + лимфопения), недавнее лечение |
| ВИЧ-инфекция (СПИД) | common | Высокая вирусная нагрузка, CD4+ <200/мкл, оппортунистические инфекции |
| Острая лучевая болезнь | uncommon | Аплазия всех ростков, тошнота, рвота, мукозит |
| Апластическая анемия | uncommon | Панцитопения, низкий ретикулоцитарный индекс, гипоклеточный КМ |
| COVID-19 (тяжёлое течение) | uncommon (по эпидемиологии) | Лимфопения, IL-6↑, D-димер↑, поражение лёгких |
| B-клеточная лимфома/лейкоз с терминальной стадией | rare | Бласты в крови, гепатоспленомегалия, анемия |

- **Опасность сейчас:** Септический шок, полиорганная недостаточность (SIRS → MODS), молниеносные оппортунистические инфекции (Pneumocystis, CMV, EBV, герпес), гемофагоцитарный синдром. При острой лучевой — смерть от кровотечения/инфекции в ближайшие дни–недели.
- **Минимальный набор подтверждающих тестов:** Абсолютное число лимфоцитов (ALC), проточная цитометрия (CD3, CD4, CD8, CD19, CD16/56), ВИЧ-серология (p24 Ag, антитела), ПЦР на HHV-6, CMV, EBV, мазок периферической крови (атипичные лимфоциты, бласты).
- **Подсказки на скрытое:** Сочетание с нейтропенией (агранулоцитоз), тромбоцитопенией (аплазия, лейкоз), низким MCV/MTAC (железодефицит на фоне хронической болезни).
- **Pitfalls / artefacts:** Высокий лейкоцитоз (нейтрофильный или бластный) может механически снизить % лимфоцитов; лимфопения на фоне безнадежно большого количества нейтрофилов — не истинная. Мазок — обязательно для оценки качества подсчёта.
- **Pearls:** 1) При сепсисе снижение ALC ниже 0.7×10⁹/л — независимый предиктор летальности (Hoffbrand 2020). 2) Хроническая ВИЧ-инфекция: относительная лимфопения + нейтропения + тромбоцитопения — классическая триада (Wintrobe 2021). 3) Лимфопения с эозинофилией ≤1% — указание на гиперкортицизм (Cushing, экзогенные стероиды) (Кассирский И.А., 1970).

### L1 — Умеренная лимфопения (ниже оптимума)

- **Числовые границы (взрослые):** 5–19% (мужчины, женщины, беременные).
- **Числовые границы (педиатрия):**
 - новорождённые: 10–19%
 - грудные: 15–24%
 - 1–12 лет: 10–24%
 - подростки: 5–19%
- **Скорость наступления имеет значение?** Острая (часы) — стресс-реакция, экзогенные стероиды; хроническая (недели–месяцы) — аутоиммунные заболевания, сахарный диабет, ВИЧ без СПИДа.
- **Что это значит физиологически:** Перераспределение пула — лимфоциты экстравазируют в ткани (адгезивный ответ) или селезёночное депонирование, либо апоптоз в ответ на глюкокортикоиды; умеренное подавление лимфопоэза.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|----------|----------------------|
| Стрессовая реакция (операция, инфаркт, пневмония) | common | Нейтрофилёз, палочкоядерный сдвиг, транзиторность |
| Глюкокортикоидная терапия / эндогенный гиперкортицизм | common | Эозинофилия/базофилия, гипергликемия, остеопороз |
| Хроническая сердечная/почечная недостаточность | common | Нормоцитарная анемия, анизоцитоз эритроцитов |
| Аутоиммунные заболевания (СКВ, РА) | common | Антинуклеарные АТ, РФ, гемолитическая анемия |
| ВИЧ-инфекция (ранняя стадия) | uncommon | Снижение CD4+ > 200/мкл, лихорадка, фарингит |
| Глюколипидозы (НПС, болезнь Гоше) | rare | Спленомегалия, тромбоцитопения, боли в костях |
| Туберкулёз (милиарный) | uncommon | Лимфопения + анемия + лихорадка + гепатоспленомегалия |

- **Опасность сейчас:** При стрессе — риск бактериальной суперинфекции; при аутоиммунных — волчанка может перейти в криз (гематологический). В целом, угроза жизни низкая, если нет сопутствующей цитопении.
- **Минимальный набор подтверждающих тестов:** ALC, СРБ, прокальцитонин, кортизол плазмы (8:00), АТ к ВИЧ, CD4-счёт, мазок крови (лимфоциты — мелкие, зрелые, без атипии).
- **Подсказки на скрытое:** Нейтрофилёз >70% практически исключает первичный лимфолиз; моноцитоз может указывать на туберкулёз, саркоидоз.
- **Pitfalls / artefacts:** Псевдолимфопения при значительном нейтрофилёзе в анализаторе (если не считать абсолютные числа). Мазок — лимфоциты нормальной морфологии.
- **Pearls:** 1) Относительная лимфопения + нейтрофилёз — «стресс-паттерн» (Hoffbrand 2020). 2) При подозрении на эндогенный гиперкортицизм: лимфопения + эозинопения + базофилопения — высокоспецифично (Williams 2015). 3) ВИЧ-инфекцию исключаем при любом упорном снижении лимфоцитов (Файнштейн Ф.Э., 1998).

### L0 — Оптимум (референс)

- **Числовые границы (взрослые):** 20–40% (мужчины, женщины, беременные).
- **Числовые границы (педиатрия):**
 - новорождённые (1-й день): 20–40%
 - грудные (1–12 мес): 50–70% (физиологический лимфоцитоз)
 - 1–12 лет: 40–60% (возрастная гиперергия)
 - подростки (13–18 лет): 25–45%
- **Скорость наступления имеет значение?** Не актуально — это норма.
- **Что это значит физиологически:** Сбалансированный лейкоцитарный профиль; достаточный резерв лимфоцитов для иммунного ответа, нормальная регуляция гемопоэза.
- **Дифряд:** не требуется.
- **Опасность сейчас:** Отсутствует.
- **Минимальный набор подтверждающих тестов:** Только контрольный анализ при наличии симптомов.
- **Подсказки на скрытое:** При норме лимфоцитов % следует проверять абсолютное число лимфоцитов (ALC) — если оно снижено, но % в норме за счёт параллельного снижения лейкоцитов, возможно апластическая анемия (псевдонорма).
- **Pitfalls / artefacts:** У детей до 6 лет нормальный % лимфоцитов выше (физиологический лимфоцитоз) — не расценивать как H1.
