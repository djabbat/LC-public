### L2 — Критическая эозинопения (life-threatening low)
- **Числовые границы (взрослые):** < 0.01 × 10⁹/л для мужчин, женщин и беременных.
- **Числовые границы (педиатрия):** < 0.01 × 10⁹/л для всех возрастных групп.
- **Скорость наступления имеет значение?** Острая эозинопения (часы–дни) — маркер септического шока или синдрома активации макрофагов, хроническая (недели–месяцы) — типична для аплазии кроветворения (Hoffbrand 2020).
- **Что это значит физиологически:** полное подавление продукции эозинофилов в костном мозге (цитокиновый шторм, стероид-индуцированная супрессия ИЛ-5) или их тотальная дегрануляция и апоптоз при гипервоспалении (WHO 2011).
- **Дифряд (топ-7 причин)**:

| Причина | Частота | Ключевые ассоциации |
|---------|---------|----------------------|
| Септический шок / бактериемия | Common | Гипотония, лактат-ацидоз, DIC |
| Синдром активации макрофагов (MAS) | Uncommon | Ферритин > 5000 нг/мл, гепатоспленомегалия |
| Острый лейкоз / КМ-аплазия | Common | Панцитопения, бласты |
| Гиперкортицизм (эндогенный / iatrogenic) | Common | Глюкокортикоиды > 7 дней |
| COVID-19 («цитокиновый шторм») | Uncommon | Высокий CRP, ИЛ-6 |
| Тиреотоксический криз | Rare | Тахиаритмия, T3/T4 ↑ |
| Химиотерапия (особенно ара-С, антрациклины) | Common | Надир лейкоцитов 7–14 день |

- **Опасность сейчас:** риск фатальной бактериальной/грибковой инвазии при нейтропении (но не специфично для эозинофилов). При MAS — полиорганная недостаточность в часы–сутки. При ДВС-синдроме — геморрагический шок (BCSH 2012).
- **Минимальный набор подтверждающих тестов:** лейкоцитарная формула (ручной просмотр), ПКТ, ИЛ-6, ферритин, триглицериды, Т3/Т4, кортизол крови/слюны — для MAS: CD163+, перфорин NK-клеток.
- **Подсказки на скрытое:** проверить абсолютные нейтрофилы — при аплазии они также < 0.5 × 10⁹/л; тромбоциты < 20 × 10⁹/л требуют исключения ДВС.
- **Pitfalls / artefacts:** разрушение эозинофилов при хранении > 4 ч (лабильность), недосчёт при гиперлейкоцитозе (> 100 × 10⁹/л), вено-окклюзионная болезнь печени (ложная эозинопения).
- **Pearls:**
 - Кассирский И.А. (1960): «Эозинофилы — зеркало адаптации». Эозинопения < 0.01 — признак срыва адаптации.
 - Wintrobe 2008: персистирующая эозинопения при сепсисе — худший прогноз выживаемости.
 - Williams 2020: введение ГКС менее 6 ч — эозинопения; > 24 ч — эффект исчезает.

### L1 — Эозинопения (ниже оптимума)
- **Числовые границы (взрослые):** 0.01 – 0.02 × 10⁹/л для мужчин, женщин, беременных.
- **Числовые границы (педиатрия):** 0.01 – 0.04 × 10⁹/л для новорождённых; 0.02 – 0.04 × 10⁹/л для 1–12 лет; 0.02 – 0.05 × 10⁹/л для подростков.
- **Скорость наступления имеет значение?** Острая (часы) — стресс-эозинопения (после травмы, операции); хроническая (дни–месяцы) — требует исключения эндокринопатии или миелодисплазии (ICSH 2014).
- **Что это значит физиологически:** угнетение эозинофил-поэза за счёт избытка эндогенных кортизола (Кушинг) или катехоламинов (феохромоцитома), либо сниженный ответ на ИЛ-5 при МДС.
- **Дифряд (топ-7 причин)**:

| Причина | Частота | Ключевые ассоциации |
|---------|---------|----------------------|
| Системная глюкокортикостероидная терапия | Common | Доза > 20 мг/сут преднизолона |
| Синдром Кушинга (эндогенный) | Uncommon | Лунообразное лицо, остеопороз |
| Острый инфаркт миокарда (КФК↑) | Uncommon | Эозинопения в первые 24–48 часов |
| Посттравматическое воспаление | Common | Операция, ожоги > 20% |
| Феохромоцитома | Rare | ГАП(↑), метанефрины + |
| Миелодиспластический синдром (МДС-много) | Uncommon | Аутоиммунные осложнения |
| Алкогольный цирроз / печеночная недостаточность | Uncommon | Асцит, билирубин ↑ |

- **Опасность сейчас:** нет прямой угрозы при значениях > 0.01. Риск — упустить латентную нейтропению (частый спутник) повышает восприимчивость к ИБС (Wintrobe 2008).
- **Минимальный набор подтверждающих тестов:** КТ надпочечников, кортизол суточной мочи или слюны (ночной), метанефрины плазмы, биопсия КМ при МДС.
- **Подсказки на скрытое:** проверить базофилы — при МДС они часто отсутствуют; MCV — макроцитоз указывает на МДС; ретикулоциты при анемии — при МДС низкие.
- **Pitfalls / artefacts:** хранение образца > 6 ч до анализа (эозинофилы разрушаются первыми), гепаринизированные пробирки (меньше артефакт, но не для CBC), t(9;22)+ лейкоз (ложная эозинопения).
- **Pearls:**
 - Лука Б. (1981): «Сочетанная нейтропения + эозинопения — исключай цитостатическая болезнь».
 - Hoffbrand 2020: при уровне кортизола > 690 нмоль/л — эозинофилы < 0.02.
 - BCSH 2010: адреналин-тест (0.3 мг п/к) повышает эозинофилы на 50% при норме; отсутствие — указывает на невосполнимость бассейна.

### L0 — Оптимум (норма)
- **Числовые границы (взрослые):** 0.02 – 0.40 × 10⁹/л (общий консенсус BCSH 2012; референс варьирует 0.02–0.50, ICSH 2014).
- **Числовые границы (педиатрия):** 0.02 – 0.50 × 10⁹/л для новорождённых (выше за счёт активации Th2 в родах); 0.02 – 0.40 × 10⁹/л для 1–12 лет; 0.02 – 0.40 × 10⁹/л для подростков.
- **Скорость наступления имеет значение?** Стабильный бассейн. Острое < 0.01 — патология. Хроническое потребление 0.02–0.05 — норма (ICSH 2014).
- **Что это значит физиологически:** сбалансированная продукция (ИЛ-5, эотаксин), нормальный костномозговой резерв, минимальное потребление в ткани (слизистые, легкие, ЖКТ).
- **Дифряд (топ-7 причин)**:

| Причина | Частота | Ключевые ассоциации |
|---------|---------|----------------------|
| Идиопатическое здоровое состояние | Common | Без клиники |
| Устойчивая физическая нагрузка | Common | Без связи с патологией |
| Циркадный ритм (утром ≤ вечера) | Common | В пределах нормы |
| Эпизодическое стресс-угнетение | Common | После операции, но < 48 ч |
| Компенсированная гипотироидия | Uncommon | ТТГ ↑, Т4 ←→ |
| Гельминтоз в стадии ремиссии | Uncommon | Ранее лечение активной инвазии |
| Тканевое потребление (неб. хронич. раны) | Uncommon | Регенерация |

- **Опасность сейчас:** нет. Клинические решения не принимаются по одному этому показателю.
- **Минимальный набор подтверждающих тестов:** при изолированном снижении < 0.02 — лейкоцитарная формула + ручной подсчёт (исключить техническую ошибку).
- **Подсказки на скрытое:** оценить общий IgE (может быть повышен при паразитозе даже при норме эоз), оценить кортизол.
- **Pitfalls / artefacts:** счёт на автоматических гематологических анализаторах может варьировать (CV% ~15–20% при малых значениях); надо верифицировать вручную.
- **Pearls:**
 - Wintrobe 2008: оптимальный уровень — 0.02–0.05 × 10⁹/л — соответствует чистоте пула без активного высвобождения.
 - ICSH 2014: утренние значения на 15–20% ниже вечерних (кортизол).
 - Файнштейн Ф.Э. (1975): «Норма эозинофилов — признак адекватного компенсаторного ответа без неоплазии».

### H1 — Эозинофилия лёгкой–умеренной степени (выше оптимума)
- **Числовые границы (взрослые):** > 0.40 – 1.5 × 10⁹/л для мужчин, женщин, беременных (BCSH 2012).
- **Числовые границы (педиатрия):** > 0.50 – 1.5 × 10⁹/л для новорождённых; > 0.40 – 1.5 × 10⁹/л для 1–12 лет; > 0.40 – 1.5 × 10⁹/л для подростков.
- **Скорость наступления имеет значение?** Острая (дни–недели) — паразитарная инвазия или ЛС-реакция; хроническая (месяцы) — исключать HES, лимфому (Hoffbrand 2020).
- **Что это значит физиологически:** активация Тh2-цитокинов (ИЛ-5, ИЛ-13 - при аллергии, паразитозе) или инфильтрация костного мозга (неоплазия); модулируемое увеличение продукции (ICSH 2014).
- **Дифряд (топ-7 причин)**:

| Причина | Частота | Ключевые ассоциации |
|---------|---------|----------------------|
| Инвазия нематодами (токсокароз, аскаридоз) | Common (эндемичные зоны) | IgE > 1000, боль в животе |
| Аллергический ринит / астма | Common | Курация пикфлоуметрии |
| DRESS-синдром | Common (при диагностике) | Лихорадка, сыпь, лимфаденопатия |
| Лимфома Ходжкина | Uncommon | B-симптомы, зуд |
| Эозинофильный гастроэнтерит | Uncommon | Абдоминалгия, анемия |
| Васкулит (Churg-Strauss/EGPA) | Rare | Астма, мононевропатия, ANCA |
| Лекарственная аллергия (НПВП, антибиотики) | Common | Связь с приемом ЛС |

- **Опасность сейчас:** низкая при отсутствии орган-специфических симптомов. В случае Churg-Strauss — риск васкулита коронарных артерий.
- **Минимальный набор подтверждающих тестов:** анализ кала на яйца глист (2–3 раза), серолог (токсокароз, трихинеллез, шистосомоз), IgE общий; при DRESS — эозинофилы + лихорадка + поражение органов (развернуть по RegiSCAR); ЭхоКГ (исключить эндомиокардит — поздний фиброз); исключить лимфому (ПЭТ).
- **Подсказки на скрытое:** проверить пациентов на IgE, tryptase, IL-5 в сыворотке; смешанная эозинофилия + лимфоцитоз (необязательно злокачеств).
- **Pitfalls
