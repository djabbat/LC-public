### L2 — Life-threatening low тромбокрит
- **Числовые границы (взрослые):** мужчины, женщины, беременные — <0.05% (единицы: %).
- **Числовые границы (педиатрия):** новорождённые, грудные, 1–12 лет, подростки — <0.05%.
- **Скорость наступления имеет значение:** острое падение (часы–дни) → немедленный риск кровотечения; хроническое (недели–месяцы) → адаптация частичная, но спонтанные геморрагии при PLT <10×10⁹/л всё равно возможны (Hoffbrand, 2020).
- **Что это значит физиологически:** резкое снижение общей массы тромбоцитов (PCT = PLT × MPV). Возникает при аплазии/инфильтрации костного мозга, потреблении (ДВС, ТТП, ГУС), деструкции (ИТП, лекарственная) или потере (массивные трансфузии, спленомегалия).
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ключевые ассоциации |
|---------|---------|----------------------|
| Иммунная тромбоцитопения (ИТП) | Common | Изолированная тромбоцитопения, нормальные мегакариоциты (Hoffbrand, 2020) |
| ДВС-синдром | Common | Шистоциты, удлинение ПВ/АЧТВ, повышение D-димера |
| Тромботическая микроангиопатия (ТТП/ГУС) | Uncommon | Шистоциты, ЛДГ ↑, креатинин ↑, ADAMTS13 ↓ |
| Апластическая анемия | Uncommon | Панцитопения, гипоклеточный костный мозг (WHO, 2011) |
| Лекарственная тромбоцитопения | Common (гепарин, хинин, НПВС) | Острое начало, анамнез приёма (BCSH, 2012) |
| Спленомегалия (секвестрация) | Common | При циррозе, болезнях накопления, гемоглобинопатиях |
| Гипопролиферация (миелофтиз) | Uncommon | Крупные опухоли, лейкоз, миелофиброз (Williams Hematology, 2021) |

- **Опасность сейчас:** спонтанное внутричерепное кровоизлияние (ВЧК) при PLT <5–10×10⁹/л; желудочно-кишечные, лёгочные, геморрагические церебральные события в часы–сутки (Wintrobe, 2019). Хронические формы — упорные петехии, экхимозы, кровоточивость дёсен, меноррагии.
- **Минимальный набор подтверждающих тестов:** повторный ОАК с подсчётом тромбоцитов (цитратная пробирка), мазок периферической крови, ПВ/АЧТВ/фибриноген/D-димер, ЛДГ, билирубин, прямой антиглобулиновый тест, АТ к тромбоцитам (опционально), УЗИ селезёнки, миелограмма (аспират/биопсия).
- **Подсказки на скрытое:** MPV — снижен (гипопролиферация) или повышен (потребление/деструкция). Ретикулоциты — повышены при гемолизе (ТТП). MCV/анемия — при аплазии или кровопотере. PDW — повышен при гетерогенном пуле.
- **Pitfalls / artefacts:** псевдотромбоцитопения (EDTA, сателлизм), гигантские тромбоциты (автоматический счётчик занижает PLT, MPV завышен — PCT может быть артефактно «нормальным», см. мазок). Холодовые агглютинины (агрегация тромбоцитов). Гемолиз при заборе (разрушение тромбоцитов). Lipemia — интерференция.
- **Pearls:** 1) Правило Виноградова: при PLT <20×10⁹/л рассматривать ИТП до подтверждения, даже при нормальном MPV. 2) Williams Hematology (2021): если PCT <0.05% и MPV >12 фл — макротромбоциты частично компенсируют гемостаз, но риск ВЧК всё равно высок. 3) При ДВС PCT коррелирует с активностью потребления — прогностически значим (ICSH, 2014).

### L1 — Субоптимальный тромбокрит
- **Числовые границы (взрослые):** 0.05–0.14%.
- **Числовые границы (педиатрия):** новорождённые, грудные — 0.05–0.11%; 1–12 лет — 0.05–0.14%; подростки — 0.05–0.14%.
- **Скорость наступления имеет значение:** умеренная «адаптация» — хроническая гипопродукция (витамин B12/фолаты, химиотерапия) менее опасна, чем острое потребление (тромботическая микроангиопатия).
- **Что это значит физиологически:** снижение массы тромбоцитов, но не достигающее порога спонтанных кровотечений. Обычно соответствует PLT 30–100×10⁹/л при нормальном MPV.
- **Дифряд (топ-7 причин):**

| Причина | Частота | Ассоциации |
|---------|---------|------------|
| Секвестрация (гиперспленизм) | Common | Спленомегалия, умеренная панцитопения |
| Хроническая ИТП | Common | Изолированная тромбоцитопения, нормальные мегакариоциты |
| Лекарственная (супрессия) | Uncommon | Химиотерапия, интерфероны, антибиотики |
| B12/фолат дефицит | Uncommon | Макроцитоз, нейтропения, палочки в нейтрофилах |
| Вирусные инфекции (ВИЧ, EBV, HCV) | Common | Мегалобластоидные изменения, плазматизация лимфоцитов (Wintrobe, 2019) |
| Миелодисплазия | Uncommon | Дисплазия в мазке, цитогенетика |
| Хроническая болезнь почек | Common | Азотемия, анемопоэтин ↓, тромбоцитопения (аккупунктурный фактор) |

- **Опасность сейчас:** риск кровоточивости при травмах/инвазивных процедурах; спонтанные петехии на ногах; экхимозы. Хронические формы — меноррагии, кровоточивость дёсен.
- **Минимальный набор подтверждающих тестов:** ОАК с ретикулоцитами, MPV, PDW; мазок (шистоциты, дисплазия); B12, фолаты ферритин; серология на ВИЧ/HCV/HBV; УЗИ селезёнки; при подозрении на МДС — миелограмма.
- **Подсказки на скрытое:** MPV — низкий (апластическая / опухолевая супрессия) или высокий (ИТП, потребление). PDW — повышен при гетерогенности (ИТП). MCV — высокий при B12/фолат дефиците; низкий при анемии хронических заболеваний.
- **Pitfalls / artefacts:** EDTA-псевдотромбоцитопения (повтор в цитрате). Гигантские тромбоциты (завышение MPV). Артефактное занижение PCT из-за микроагрегатов при lipemia.
- **Pearls:** 1) Если PCT <0.12% и MPV >10 фл — заподозрить ИТП (Hoffbrand, 2020). 2) При хроническом гиперспленизме PCT может быть <0.10% без тромбоцитопении анамнестически неопасно (Williams Hematology, 2021). 3) B12-дефицит может давать выраженную макроцитарную анемию с тромбоцитопенией — коррекция приводит к нормализации PCT в течение 1–2 недель.

### L0 — Оптимальный тромбокрит (референс)
- **Числовые границы (взрослые):** 0.15–0.35% (единицы: %).
- **Числовые границы (педиатрия):** новорождённые — 0.12–0.41%; грудные — 0.14–0.36%; 1–12 лет — 0.15–0.35%; подростки — 0.15–0.35%.
- **Скорость наступления не имеет значения:** стабильное состояние.
- **Что это значит физиологически:** нормальная общая масса тромбоцитов (примерно PLT 150–400×10⁹/л и MPV 7–11 фл). Обеспечивает адекватный гемостаз.
- **Дифряд:** не требуется (дифференциально-диагностический поиск не показан, если нет других отклонений).
- **Опасность сейчас:** отсутствует.
- **Минимальный набор подтверждающих тестов:** не требуется.
- **Подсказки на скрытое:** если PCT нормален, но PLT заметно ниже 150×10⁹/л — следует уточнить MPV (возможна компенсация макротромбоцитами). Пример — синдром Бернара–Сулье (MPV >12 фл, PLT ~100, PCT в норме).
- **Pitfalls / artefacts:** при гигантских тромбоцитах автоматический анализатор может завысить MPV и, соответственно, PCT, даже при истинно сниженном PLT. Верификация мазком.
- **Pearls:** 1) Правило Wintrobe: нормальный PCT — не исключает тромбоцитопатию. 2) При железодефиците часто снижен MPV (5–7 фл), PLT может быть повышен — PCT может оставаться нормальным (Hoffbrand, 2020). 3) В педиатрии у новорождённых физиологически более высокий PCT — до 0.41% (ICSH, 2014).
