## Отёчные синдромы: дифференциальная диагностика по Виноградову

### 1. Алгоритм первичной оценки отёчного синдрома

1. **Локализация дебюта:**
 * *Стопы/голени (нисходящие)* → сердечная, венозная, лимфатическая.
 * *Лицо/веки (восходящие)* → почечная (нефротическая/нефритическая), эндокринная (микседема — лицо, но и голени).
 * *Абдоминальный/асцит* → печёночная (портальная гипертензия, гипоальбуминемия), тяжелая сердечная (констриктивный перикардит, ПЖ-недостаточность), гипопротеинемия.
2. **Время появления/суточный ритм:**
 * К вечеру (ортостаз) → сердечная, венозная.
 * Утром (горизонтальное положение) → почечная, микседема.
3. **Характер ямки (тест пальцевого давления):**
 * Быстро расправляется («водянистая») → сердечная, почечная, гипопротеинемия.
 * Медленно расправляется (тестоватая / «слизеподобная» с набуханием тканей вокруг) → лимфостаз, микседема (при гипотиреозе — слизистый отёк, но с ямкой преимущественно в нижней трети голени).
 * Отсутствие ямки (синюшная, холодная) → липо-дерматосклероз при ХВН, лимфедема III ст. (слоновость).

### 2. Сердечные отёки (застойная сердечная недостаточность)

**Ключевые признаки (V. Parin, A. Vinogradov):**
* Симметричные, плотные, холодные на ощупь, цианотичные (дистальный цианоз: стопы, кисти акроцианоз + отёк = акроцианоз с отёком).
* Возникают при физической нагрузке/к вечеру, проходят после ночного отдыха/диуреза.
* Сопутствуют: одышка (ортопноэ), никтурия, гепатомегалия, набухание шейных вен, влажные хрипы в лёгких.
* Pitting-тест: ямка есть, расправляется умеренно.

**Минимальный параклинический набор:**
* ЭКГ, ЭхоКГ (насосная функция: ФВ, диастолическая дисфункция, размеры камер, ДЛА).
* Нат-рийуретический пептид (NT-proBNP) — норма исключает СН.
* Биохимия: электролиты (особенно Na+ для прогноза/терапии), креатинин, печеночные ферменты (холестаз при застойной печени).

**Узловые точки:**
1. Разделить: **систолическая** vs **диастолическая** СН (тактика лечения разная: ингибиторы АПФ/бета-блокаторы vs контроль ЧСС/диуретики/антагонисты альдостерона).
2. Определить **левожелудочковая** vs **правожелудочковая** недостаточность. Отёки — маркёр *правожелудочковой* (или бивентрикулярной) декомпенсации.

**Ошибки Виноградова:**
* Нельзя назначать мощные диуретики при остром инфаркте миокарда без оценки сократимости — резкое снижение преднагрузки может ухудшить выброс.
* Не путать с ХВН (см. табл. ниже).

### 3. Почечные отёки

#### А. Нефритические (острый гломерулонефрит)
* **Начало:** остро, после инфекции (бета-гемолитический стрептококк).
* **Локализация:** лицо (утренняя одутловатость лица), затем веки, голени.
* **Характер:** рыхлые, мягкие, тёплые, бледные (но цианоза нет).
* **Сопутствуют:** гипертензия, олигурия, макрогематурия («мясные помои»).
* **Параклиника:** протеинурия (< 3 г/сут), гематурия, цилиндрурия (эритроцитарные), повышение АТ к стрептококку, снижение С3-комплемента.

#### Б. Нефротические (гломерулонефрит, амилоидоз, диабет)
* **Признак:** массивные отёки (анасарка + асцит + гидроторакс + гидроперикард).
* **Локализация:** системные, «восходящие без цианоза».
* **Параклиника:** *массивная протеинурия* (> 3,5 г/сут), гипоальбуминемия (< 25 г/л), гиперлипидемия, цилиндрурия (жировая, восковидная).

**Узловые точки (по Виноградову):**
* При нефротических отёках — экстренное определение сывороточного альбумина и липидного спектра для начала терапии ГК/цитостатиками.
* Дифференцировать амилоидоз (поражение ЖКТ, печени, макроглоссия, дефицит факторов свертывания, прокрашивание Конго красным в биоптате).

**Ошибки:**
* Мочегонная терапия при низком онкотическом давлении (нефротический синдром) без восполнения альбумина неэффективна и токсична.

### 4. Печёночные отёки (портальная гипертензия + гипоальбуминемия)

**Признаки:**
* Доминирует **асцит** (упорный, большой, напряжённый).
* Отёки ног — вторичны (из-за сдавления нижней полой вены асцитом и гипопротеинемии).
* Желтуха, «печёночные знаки» (сосудистые звездочки, пальмарная эритема, гинекомастия, атрофия яичек).
* Спленомегалия, расширение вен пищевода/геморрой.

**Параклиника:**
* Асцитическая жидкость: *транссудат* (белок < 15 г/л, SAAG > 1,1 г/дл).
* Биохимия: снижение альбумина, удлинение ПВ/МНО, повышение билирубина/АЛТ/АСТ, холестаз.

### 5. Эндокринные отёки (микседема)

**Признаки:**
* **Микседема (истинная):** плотный, тестоватый отёк, *не оставляющий ямки* при надавливании (выпот мукополисахаридов в кожу). Локализация: лицо (одугловатость, сужение глазных щелей), веки, губы, язык (макроглоссия), кисти («руки-лопаты»).
* **Покатость плеч, сухая шелушащаяся кожа, брадикардия, гипотония, запоры, заторможенность.**
* **Псевдоотёки:** при гипотиреозе может быть истинный гидроторакс/перикардит с транссудатом (выпот накапливается из-за снижения лимфатического дренажа).

**Дифференциальный ряд с ХВН:** см. табл.

### 6. Венозный застой (ХВН, тромбоз глубоких вен)

| Признак | Варикоз/ХВН | Острый ТГВ | Сердечная СН |
| :--- | :--- | :--- | :--- |
| **Локализация** | Нижняя треть голени, варикозные узлы | Вся конечность (бедро/голень) | Симметрично обе |
| **Кожа** | Теплая, венозный рисунок, пигментация (гемосидероз), липодерматосклероз (затвердевание в нижней трети) | Цианотичная, лоснящаяся, расширение подкожных вен | Цианотичная, холодная |
| **Ямка** | Есть в ранней стадии, плотная при липодерматосклерозе | Есть, болезненная | Есть |
| **Боль** | Тяжесть, распирание, судороги ночью | Острая, распирающая, боль по ходу вен | Нет (или тяжесть) |
| **Симптом Хоманса** | Реже | + (боль в икре при тыльном сгибании стопы) | - |

**Ключевые признаки ТГВ (опасный имитатор отёка):**
* Внезапное начало, асимметрия, цианоз, боль.
* УЗДГ вен (дуплексное сканирование) — «золотой стандарт».
* D-димер — скрининг (высокая чувствительность, низкая специфичность).

**Ошибки Виноградова:**
* При подозрении на ТГВ — пальпация вен противопоказана до УЗИ (риск эмболии).
* Не назначать мочегонные при венозном застое — они усугубляют гемодинамику, уменьшают ОЦК без улучшения венозного оттока.

### 7. Лимфостаз (лимфедема)

**Признаки:**
* Асимметричные (чаще одна конечность), плотные, *мягкие в начале*, *затем плотно-фиброзные*.
* Ямки нет (на поздней стадии/при слоновости). Кожа утолщена, шероховатая, на тыле стопы складка (симптом Капоши–Стеммера).
* Начало: после травмы/операции (удаление лимфоузлов), инфекции (рожа/филяриоз), или идиопатическое (у женщин).
* Редко: асцит (хилёзный асцит — при нарушении лимфатического оттока из грудного протока).

**Дифференциальный ряд:**
1. Исключить венозный тромбоз (как *острую* причину).
2. Исключить опухоль (раковая инвазия лимфатических узлов: рак молочной железы, шейки матки, простаты).
3. Исключить рожу (эритема, повышение температуры, болезненность).

### 8. Гипопротеинемия (нефротический, печёночный, экссудативная энтеропатия)

**Признак:** системные отёки (анасарка) при снижении онкотического давления (альбумин < 25 г/л) — чаще у пациентов с нефротическим синдромом, болезнью Крона/целиакией, тяжёлым циррозом.

**Дифференциальный ряд:**
* **Альбумин низкий (↓) + протеинурия → нефротический синдром.**
* **Альбумин низкий + протеинурия 0 + признаки мальабсорбции/потери белка в ЖКТ (α1-антитрипсин в кале) → экссудативная энтеропатия.**
* **Альбумин низкий + цирроз/портальная гипертензия → печёночная недостаточность.**

### 9. Асцит. Плевральный выпот: транссудат vs экссудат

**Критерии Лайта (для плевральной жидкости) — «золотой стандарт»:**
* Экссудат (воспалительный/злокач/эмболия) если ≥1 критерия:
 1. **Отношение белка выпота к сыворотке > 0,5.**
 2. **Отношение ЛДГ выпота к сыворотке > 0,6.**
 3. **ЛДГ выпота > 2/3 верхней границы нормы сывороточной ЛДГ.**
* Если все критерии отрицательны — **транссудат** (портальная гипертензия, СН, гипоальбуминемия, микседема, урогемоторакс, ХПН).

**Критерии SAAG (Serum-Ascites Albumin Gradient) для асцита — предпочтительнее Low SAAG:**
* **SAAG = Сывороточный альбумин — альбумин асцита.**
* **SAAG ≥ 1,1 г/дл → Транссудат/высокое давление (портальная гипертензия):** цирроз, сердечная недостаточность (констриктивный перикардит > ПЖН), массивная тромбоэмболия.
* **SAAG < 1,1 г/дл → Экссудат/низкое давление:** инфекция (туберкулёз, СБП), опухоль (перитонеальный карциноматоз), панкреатит, лимфома, билиарный/панкреатический выпот.
