## Синдром протеинурии

**1. Алгоритм диагностики**
1. Сбор анамнеза: давность (выявлена случайно или при скрининге), связь с лихорадкой/физической нагрузкой (функциональная протеинурия), приём НПВС, препаратов золота, пеницилламина.
2. Уточнить суточную потерю белка (суточная моча): <0,3 г/сут — следы; 0,3–3,5 г/сут — умеренная; >3,5 г/сут — нефротический уровень.
3. Оценка селективности (электрофорез белков мочи): высокая (альбумин) — минимальные изменения; низкая (IgG, α2-макроглобулин) — мембранозный, фокально-сегментарный гломерулосклероз (ФСГС).
4. Исключение ортостатической протеинурии (сбор мочи лёжа и после ходьбы).
5. При стойкой протеинурии >1 г/сут — биопсия почки.

**2. Дифференциальный ряд (от наиболее опасного к менее опасному)**
| Нозология | Ключевой признак | Параклинический критерий |
|-----------|------------------|---------------------------|
| Амилоидоз | Макроглоссия, гепатоспленомегалия | Окраска Конго красным, сывороточный амилоид A |
| Множественная миелома | Костная боль, анемия | Электрофорез сыворотки (M-градиент), белок Бенс-Джонса |
| Диабетическая нефропатия | Длительный СД, ретинопатия | HbA1c, экскреция альбумина |
| Гломерулонефриты | Анамнез инфекций, системных заболеваний | Иммунологические маркеры (C3, C4, ANCA) |
| Гипертонический нефроангиосклероз | Длительная АГ, компенсированная функция почек | УЗИ почек: симметричное сморщивание |
| Ортостатическая протеинурия | Только в вертикальном положении | Сбор мочи лёжа <0,3 г/сут |

**3. Узловые признаки**
- Суточная потеря >3,5 г/сут = нефротический синдром, требует немедленной индукции ремиссии (глюкокортикоиды, цитостатики).
- Протеинурия в сочетании с гематурией → гломерулонефрит с экстракапиллярными полулуниями (критический прогноз).

**4. Опасные имитаторы**
- **«Функциональная» протеинурия** (лихорадка, стресс) — может маскировать дебют гломерулонефрита. Критерий: исчезает после выздоровления.
- **Серопозитивный ревматоидный артрит** — амилоидоз почек развивается через 5–10 лет.

**5. Типичные ошибки**
- Поздняя диагностика амилоидоза: подмена понятий «идиопатическая протеинурия» при отсутствии биопсии.
- Неучёт ортостатического компонента у подростков → ложное заключение о хроническом нефрите.
- Игнорирование моноклонального пика у пожилых (>60 лет) при протеинурии >1 г/сут.

## Синдром гематурии

**1. Алгоритм**
1. Макро- vs микроскопическая (эритроцитурия >3–5 в п/зр).
2. Колориметрия мочи (красный цвет) → исключить гемоглобинурию/миоглобинурию (осаждение эритроцитов).
3. Фазовая микроскопия осадка:
 - **Дисморфные эритроциты** (>40%) — гломерулярное происхождение (IgA-нефропатия, нефрит).
 - **Изоморфные** — урологическое (камень, опухоль, инфекция).
4. Анамнез: инфекция верхних дыхательных путей за 1–3 дня до (синфарингит) vs 2–3 недели (постстрептококковый).
5. При гломерулярном типе → антистрептолизин-О (АСЛО), C3, C4, IgA, ANCA.

**2. Дифференциальный ряд**
| Тип гематурии | Основные причины | Ключевые тесты |
|---------------|------------------|----------------|
| Гломерулярная | IgA-нефропатия, постстрептококковый нефрит, люпус-нефрит, мембранозный | Эритроцитарные цилиндры, иммунные маркеры |
| Урологическая | Мочекаменная болезнь, опухоль (почечно-клеточный рак, рак мочевого пузыря) | УЗИ, КТ-урография, цистоскопия |
| Травматическая | Катетеризация, травма | Анамнез, исчезает через 2–3 дня |
| Идиопатическая | Доброкачественная семейная гематурия | Нормальная биопсия, стабильная функция |

**3. Узловые признаки**
- **Макрогематурия без дизурии** → урологическая причина, требует цистоскопии в первую очередь.
- **Гематурия + протеинурия >1 г/сут** → биопсия почек (риск прогрессирования).
- **Боли в пояснице + уретерогидронефроз** → немедленная урологическая декомпрессия (камень, папиллярный некроз).

**4. Опасные имитаторы**
- **Гемоглобинурия** (гемолиз, маршевая миоглобинурия) — осадок мочи без эритроцитов.
- **Красящие вещества** (свёкла, рифампицин) — без осадка.
- **Цистит** (микрогематурия) может маскировать карциному in situ.

**5. Типичные ошибки**
- Диагностика «IgA-нефропатии» без биопсии: только при стойкой гематурии >12 мес и потере >1 г/сут.
- Игнорирование цистоскопии у пациентов >40 лет с макрогематурией.

## Нефритический синдром

**1. Диагностические критерии**
- Гематурия (часто макроскопическая) + протеинурия (умеренная) + отёки (периорбитальные) + АГ + олигурия + азотемия.
- Развивается остро или подостро.

**2. Алгоритм дифференцировки**
1. Исключить инфекционный фон:
 - **Постстрептококковый (острый нефрит)**: латентный 2–3 нед после ангины/пиодермии, АСЛО повышен, C3 снижен (нормализуется через 6–8 нед).
 - **IgA-нефропатия (болезнь Берже)**: синфарингит (1–3 дня), нормальный C3.
 - **Инфекционный эндокардит** (Staph. aureus, стрептококк) — шумы, лихорадка, положительная гемокультура.
2. Системные васкулиты: ANCA-позитивность (гранулематоз с полиангиитом, микроскопический полиангиит), анти-GBM (синдром Гудпасчера — лёгочное кровотечение + нефрит).
3. Люпус-нефрит: ANA, анти-dsDNA.

**3. Дифференциальный ряд (по скорости прогрессирования)**
| Форма | Прогрессирование | Ключ к диагнозу |
|-------|------------------|------------------|
| Остронефритический (постстрептококковый) | Быстрое разрешение (недели) | Ответ на гипотензивную и диуретики |
| Быстропрогрессирующий (БПГН) | Недели–месяцы → ХПН | Биопсия: полулуния >50% |
| Хронический нефрит | Годы | Торпидная гематурия, межклеточный склероз |

**4. Узловые точки**
- **Олигурия + АГ + отёки** → биопсия в первые 24–48 ч (БПГН — без лечения потеря органа за 1–2 мес).
- **Нормализация C3 через 6–8 нед** указывает на постстрептококковый нефрит (при люпус-нефрите — не нормализуется).
- **Креатинин >176 мкмоль/л + гематурия** → биопсия (прямой риск диализа).

**5. Опасные имитаторы**
- **Тромботическая микроангиопатия (ТМА)**: шизоциты, тромбоцитопения, Анемия (гемолитико-уремический синдром/тромбоцитопеническая пурпура).
- **Злокачественная АГ**: вызванный ею нефрит — экстренное снижение АД (нифедипин под язык).

**6. Типичные ошибки**
- Пропуск БПГН: принятие за постстрептококковый из-за высокого АСЛО (ложно успокаивает).
- Задержка с биопсией при ANCA-позитивности (показана плазмаферезная терапия).
- Недиагностика ТМА (отказ от биопсии, задержка с экулизумабом).

## Нефротический синдром

**1. Диагностические критерии**
- Протеинурия >3,5 г/сут (суточная моча) + гипоальбуминурия <25 г/л + отёки (периферические, асцит, анасарка) + гиперлипидемия (холестерин >6,5 ммоль/л).

**2. Алгоритм нозологической диагностики**
1. У детей (1–8 лет): минимальные изменения (MCD) — ответ на кортикостероиды (преднизолон 2 мг/кг/сут 4–6 нед).
2. У взрослых: биопсия (мембранозный, ФСГС, мембранозная волчаночная нефропатия, амилоидоз).
3. При длительности >6 мес + рецидивы → наличие «полулуний» или фибринозной депозиции.
4. Исключить амилоидоз (Конго красный), диабетическую нефропатию (HbA1c, ретинопатия).

**3. Дифференциальный ряд (по биопсии)**
| Нозология | Возраст | Ключевой признак |
|-----------|---------|-------------------|
| Минимальные изменения (MCD) | Дети/молодые взрослые | Высокая селективность, ответ на ГКС |
| Фокально-сегментарный гломерулосклероз (ФСГС) | Любой | Резистентность к ГКС, коллапсирующие изменения |
| Мембранозный | >45 лет | IgG-депозиты, Анти-ФЛА2Р (маркер) |
| Амилоидоз | >60 лет | Окраска Конго, макроглоссия |
| Диабетическая нефропатия | СД в анамнезе | Узелки Киммелстила-Уилсона |
| Волчаночный нефрит (V класс) | Женщины | ANA+, анти-dsDNA+, низкий C3/C4 |

**4. Узловые точки**
- **Гиперлипидемия** — независимый предиктор прогрессирования (назначают статины).
- **Тромбозы**: потеря антитромбина III с мочой → профилактика НМГ (низкомолекулярным гепарином) при альбумине <20 г/л.
- **Инфекции**: потеря IgG → риск пневмококковой/стрептококковой инфекции (вакцинация, пенициллин).

**5. Опасные имитаторы**
- **Диабетическая нефропатия**: не требует биопсии, если есть ретинопатия.
- **Мембранозная нефропатия (паранеопластическая)**: ассоциация с раком лёгкого, толстой кишки, лимфомой (искать опухоль).
- **Амилоидоз AL-типа**: множественная миелома (белок Бенс-Джонса)
