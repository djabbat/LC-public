## КРИТИЧЕСКИЕ РЕШЕНИЯ (всегда помнить)

### R²=0.84 — ИЗЪЯТО (2026-04-13)
`null_model_r2.py` использует SIMULATED (синтетические) данные. R²=0.84 НЕ является валидацией на реальных данных. Изъято из всей документации с 2026-04-13.
**Правильные цифры:** R²(MCAI)=0.745, R²(CHIP)=0.611, R²(Telo)=0.465 (in-sample, реальные данные).

LOI v20 тестировал только молодые HSC → инконгруэнтная фальсификация.

### CEP135-KD → SMO-M2 замена (2026-04-14)
CEP135-KD разрушает весь центриолярный каркас неспецифически — результат неинтерпретируем.
Заменено на SMO-M2 (конституционно активный Smoothened) — тестирует Hh-ось специфически.

### R²=0.84 был на синтетике — УРОК
Ошибка: использование результата скрипта на синтетических данных как валидации. Правило: всегда проверять источник данных (синтетика vs реальные литературные) перед цитированием R².

## ВАЖНЫЕ НАУЧНЫЕ УТОЧНЕНИЯ

### Область применения ¬R-аргумента
Аргумент верифицирован: in vitro прогениторы (Peters-Hall 2020). In vivo HSC — тестируемая предпосылка (Phase 0 + Arm E).

### CASIN ≠ CCP1/AGBL5
CASIN действует через Cdc42 → восстанавливает центросомную полярность (downstream от PTM).
CCP1/AGBL5 — убирают polyGlu напрямую (upstream). CDATA предсказывает: CCP1 даст превосходящий эффект.

### Co-culture rescue — временный (P11)
Rescue старых клеток молодыми — паракринный, через нишевый Hh/Wnt. Временный: через N делений PTM накапливаются снова. Relapse без отторжения = уникальное CDATA-предсказание.

### CD150low HSC = CDATA-предсказанная субпопуляция
Клетки с меньшим числом делений → меньше PTM → лучше функционируют. Именно это CDATA предсказывает должно существовать внутри старого пула.
