## Cell-DT v4.0 (СЛЕДУЮЩАЯ MAJOR VERSION)

### 4.0.1 — D(t)→ep_age интеграция [ПРИОРИТЕТ P0]
**Проблема:** ABL-2 парадокс: ep_rate=0, только центриоль → R²=0.833 > FULL R²=0.778. Причина: ep_age=ep_rate×T линейно доминирует в аналитическом приближении.

**Решение:**
```
ep_age(t) = ep_rate_base × t + k_ep × ∫₀ᵗ D(τ) dτ
```
ep_rate как независимый параметр исчезает. alpha однозначно доминирует в Sobol.

**Статус:** [ ] Требует peer review → [ ] Реализация → [ ] Sobol re-run → [ ] ✅

### 4.0.2 — ROS-уравнение исправление [P1]
**Проблема:** R²(ROS)=−0.512. pi_0=0.99 на границе оптимизации = неидентифицируемость.
**Решение:** Пересмотреть формулу ROS-компонента; добавить данные ROS в калибровочный датасет (>7 точек).

**Статус:** [ ] Анализ → [ ] Исправление → [ ] ✅

### 4.0.3 — Sobol full Rust-ODE (не аналитическое приближение) [P1]
**Проблема:** Текущий Sobol использует аналитическое приближение → S1(ep_rate) может быть завышен.
**Решение:** Sobol N=16384 на полной Rust-ODE (всех 8 крейтов).

**Статус:** [ ] Требует реализации после 4.0.1 → [ ] ✅

## ⚠️ КРИТИЧЕСКИЕ ЗАМЕЧАНИЯ peer review (2026-04-15 ultra-strict, DO NOT FUND 28/100)

### P-CRIT-1: Phase 0 тестирует корреляцию, НЕ причинность [FATAL FLAW]
**Проблема:** Phase 0 (GT335 vs. возраст, Ki67 co-culture) — дескриптивные данные. Они не доказывают, что polyGlu на центриоли ПРИЧИНЯЕТ функциональный спад. 100 других теорий (эпигенетический дрейф, митохондриальная дисфункция) предсказывают те же корреляции.
**Решение (reviewer-рекомендация):** Ввести Arm 0-CAUSAL: Инду­цируемый TTLL6-GFP (PACT-домен, материнская центриоль) в **молодых** LSK → форсированное polyGlu на унаследованной центриоли → конкурентная трансплантация vs. WT молодых. Предсказание: молодые HSC с гиперглутамилированной центриолью покажут сниженный химеризм = PTM **достаточно** для индукции старения.
**Статус:** [ ] Требует обсуждения с Geiger → [ ] Дизайн протокола → [ ] ✅

### P-CRIT-2: CCP1-OE (Arm E) — плейотропный эффект [MAJOR]
**Проблема:** CCP1 деглутамилирует гистоны, тубулин, и другие цитозольные белки. Положительный эффект от CCP1-OE может быть эпигенетическим, а не специфически центриолярным.
**Решение:** Добавить контроль специфичности: CCP1 с мутацией активного сайта (каталитически мёртвый) + CCP1, адресованный только к центриоли (PACT-CCP1 фьюжн). Только центриоль-специфический rescue → доказательство центриолярного механизма.
**Статус:** [ ] Обсудить с Geiger → [ ] ✅

### P-CRIT-3: Bradford-Hill самооценка — исключить из гранта [MINOR]
**Проблема:** Самостоятельная оценка своей теории по BH — «unserious» по мнению reviewer. Ослабляет credibility.
**Решение:** Убрать BHCA-числа из LOI. Использовать BHCA только во внутренней документации и в Aging Cell статье.
**Статус:** [ ] Обновить LOI v22 → [ ] ✅

### P-CRIT-4: Модель переобучена — не цитировать как evidence
**Проблема:** LOO-CV mean=-0.093 = переобученность. Модель в текущем виде не добавляет научного веса LOI.
**Решение:** В LOI упоминать модель только как «computational framework for hypothesis generation», не как «validation». Цитировать: «Cell-DT v4.0 integration of D(t)→ep_age planned to resolve ABL-2 paradox».
**Статус:** [ ] Обновить текст LOI → [ ] ✅
