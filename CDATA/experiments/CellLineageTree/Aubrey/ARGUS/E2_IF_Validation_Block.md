## 4. ПРОТОКОЛ ИММУНОФЛУОРЕСЦЕНЦИИ (для ARGUS)

### 4.1 После live-cell съемки

```
1. Удалить среду из чашки
2. Промыть 2× PBS (1 мл, 2 мин)
3. Фиксация:
   - Для Centrin-1 20H5 и CEP164: 4% PFA + 0.2% Triton X-100 + 1 mM MgCl₂, 10 мин, RT 
   - Для Centrobin: -20°C метанол, 10 мин, -20°C 
4. Промыть 3× PBS, 5 мин каждую
5. Блокировка: 10% FBS в PBS (или 1% BSA + 0.1% Triton), 30-60 мин 
6. Инкубация с первичными антителами (4°C, overnight в влажной камере)
7. Промыть 3× PBST (PBS + 0.05% Triton), 5 мин 
8. Инкубация с вторичными антителами (RT, 1 ч, темнота)
9. DAPI (1:500) на 2-й промывке
10. Монтирование с DAKO Faramount
```

### 4.2 Рекомендованные комбинации для дубль-окрашивания

| Цель | Комбинация | Каналы |
|------|------------|--------|
| **Материнская vs Дочерняя** | CEP164 (Alexa 555) + Centrobin (Alexa 488) | красный / зеленый  |
| **Центриольный контроль** | Centrin-1 (Alexa 488) + γ-Tubulin (Alexa 647) | зеленый / дальний красный  |
| **Верификация CP110** | CP110 (Alexa 555) + Centrin-1 (Alexa 488) | красный / зеленый  |
