# Zeiss IM 35 / ICM 405 — Инвертированный микроскоп

## Что это

**Zeiss IM 35 / ICM 405** (West Germany, 1980-е) — инвертированный оптический микроскоп для исследования живых клеток в культуре.

## Возможности (из manual)

| Режим | Для чего |
|---|---|
| **Brightfield** | Обычное наблюдение |
| **Phase contrast** | **Неокрашенные живые клетки в культуре** ← критично для стволовых клеток |
| **DIC** (дифференциальный интерференционный контраст) | Высокий контраст без окраски |
| **Darkfield** | Тёмное поле |
| **Epi-fluorescence** | **С HBO 50W ртутной лампой** — FITC/TRITC/DAPI каналы — **центриолярные маркеры** |

Объективы (из таблицы DIC): Plan 6.3/0.16, Plan 16/0.35, Plan-Neofluar 16/0.5 W Oil, Plan-Neofluar 25/0.8 W Oil, LD-Plan 40/0.6, Plan 40/0.65, Planapo 63/1.4 Oil, Plan 100/1.25 Oil.

**Это ПРОФЕССИОНАЛЬНЫЙ ИССЛЕДОВАТЕЛЬСКИЙ МИКРОСКОП**, а не демонстрационный.

## Как это ПОМОЖЕТ диссертации CDATA — РЕВОЛЮЦИОННО

Из теоретической → в **экспериментально-теоретическую**.

### Уровень 1 — фигуры для диссертации ($300-600)
- Собственные phase contrast изображения культуры клеток
- DIC делящихся клеток
- Покажут компетенцию кандидата в области клеточной биологии

### Уровень 2 — **11-я PubMed публикация** (дополнительно $300-500)
С флуоресцентными антителами к γ-тубулину / центрину → изображения асимметричного наследования центриолей → Peer-reviewed paper → **добавить к 10 PubMed как физическое экспериментальное подтверждение CDATA**.

Нужно дополнительно:
- Антитела anti-γ-tubulin (+ secondary fluorophore) ~$200
- DAPI / Hoechst для ядер ~$20
- Культура стволовых клеток (MSC, HSC) — коллаборация с клиникой или институтом
- CO2 инкубатор, стерильный бокс — есть в любой лаборатории TSU

### Уровень 3 — экспериментальный PhD falsification (6-12 мес)
