### Минимальный плагин

```php
// plugins/generic/myPlugin/MyPlugin.php
class MyPlugin extends GenericPlugin {
    public function register($category, $path, $mainContextId = null) {
        $success = parent::register($category, $path, $mainContextId);
        if ($success && $this->getEnabled()) {
            HookRegistry::register('Templates::Common::Footer', [$this, 'callbackFooter']);
        }
        return $success;
    }

    public function callbackFooter($hookName, $args) {
        $smarty =& $args[1];
        $output =& $args[2];
        $output .= $smarty->fetch($this->getTemplateResource('footer.tpl'));
        return false;
    }

    public function getDisplayName() { return 'My Plugin'; }
    public function getDescription() { return 'Does something'; }
}
```

## Обзор платформы

**longevity.ge** — научно-издательская платформа Грузинского Альянса Долголетия (Georgia Longevity Alliance), основанного 16 января 2016 года. Платформа объединяет два рецензируемых открытых журнала, посвящённых науке о долголетии, борьбе со старением и продлению жизни человека.

## Уникальные концепции платформы

### Ze Framework
Авторский теоретический конструкт, объединяющий биологию и физику. Включает новую информационно-обрабатывающую парадигму с проверяемыми предсказаниями (например, замедление времени через Ze-счётчики). Публикуется в Longevity Horizon.

### xTwin (Виртуальный двойник)
Методологическое требование журнала Annals of Rejuvenation Science: каждое эмпирическое исследование должно пройти in silico валидацию через цифровую модель испытуемого (xTwin). Обеспечивает стандарт воспроизводимости.

### Грузия как хаб клинических испытаний
Альянс использует благоприятное регуляторное окружение Грузии для клинических испытаний. В 2019 году проводились испытания сенолитической комбинации дасатиниб + кверцетин на людях.

## Позиционирование

- Широкий тематический диапазон (от планарий до космоса)
- Готовность публиковать смелые теоретические конструкты (Ze Theory, xTwin)
- Миссионерский тон редакционной политики
- Двуединство: научная строгость + футуристическое видение
