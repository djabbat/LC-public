## Checklist (✓/✗ + объяснение)

### 1. Operationalised falsifiability (numeric thresholds) — ✗
Числовые пороги присутствуют (MCID=5.4, α=0.05, power=0.80, N≥55), но в разных частях пакета даны противоречивые значения α: в одной секции α=0.05 (Bonferroni 0.025), в другой — p<0.001 (CONCEPT.md, блок "Falsifiability" после "Primary outcome metric"). Нет единого, однозначного набора. Также не указана статистическая мощность для вторичных гипотез. Операционализация неполная.

### 2. Pre-registration plan (OSF placeholder + date) — ✓
Указан OSF placeholder (`osf.io/TBD` и `osf.io/XXXXX`) и дата (2026-09-01). План включает дизайн, первичные/вторичные исходы, план анализа. Приемлемо, хотя два разных placeholder'а создают неконсистентность.

### 3. Sample size calc (power analysis) — ✓
Формула `n = (Z_α/2 + Z_β)² · 2σ² / δ²` с подстановкой (1.96, 0.84, 10, 5.4) даёт N=55/группу. Учтён dropout 20% → 132. Есть sensitivity analysis для разных σ. OK, но примечание с противоречивыми PMID снижает доверие.

### 4. Risk matrix ≥5 rows — ✓
Три разных матрицы, каждая содержит ≥5 строк, указаны probability, impact, mitigation. Однако они не унифицированы, некоторые probability указаны словами (Low/Medium/High) — приемлемо, но несогласованность между документами указывает на нарушение внутренней согласованности (см. п.9).

### 5. Limitations section — ✗
Раздел limitations присутствует, но в трёх разных версиях (8, 6, 7 пунктов) с разным содержанием и формулировками. Нет единого, непротиворечивого раздела. Некоторые limitations повторяются, некоторые противоречат друг другу. Не выполнен.

### 6. Consortium / collaboration plan — ✓
Перечислены роли: Lead PI, Co-I (Clinical, Technical, Statistics), потенциальные партнёры (Insignia Health, Fraunhofer IGD, TSU, University of Copenhagen). Имена частично TBD — допустимые placeholder'ы. План есть.

### 7. Reference reality + match — ✗ КРИТИЧЕСКИЙ ПРОВАЛ
- **Hibbard 2004**: в разных местах указаны PMID 15527447 и 15333167. PMID 15527447 на самом деле относится к Hibbard 2005 (short-form PAM). То есть одна и та же работа цитируется с двумя разными PMID, один из которых неверен. Это нарушение п.7b (несоответствие тексту).
- **Tao et al. (2026)**: DOI TBD, нет реального идентификатора. Нельзя проверить существование.
- **Blumenthal-Lee (2024)**: DOI TBD, нет реального идентификатора.
- **Tqemaladze (2026)**: DOI TBD, нет реального идентификатора.
- **Hibbard (2009) PAM manual**: проприетарный документ без DOI/ISBN.
- **Mayo Clinic, NIH MedlinePlus**: веб-ссылки, не научные статьи.

Только Hibbard 2004 (корректный PMID 15333167) соответствует. Остальные — невалидны или не имеют идентификатора. Условие 7 не выполнено.

### 8. No fabrication markers — ✗
В разделах ссылок присутствуют "DOI TBD", "pre-print DOI TBD". Это недопустимые placeholder'ы для научных ссылок (правило: placeholder разрешён только в pre-reg и risk matrix). Также обнаружено два разных PMID для одной работы — признак неаккуратного редактирования. Условие 8 нарушено.

### 9. Internal consistency core docs — ✗
Множественные грубые противоречия:
- **Falsifiability**: α=0.05 / p<0.001 / Bonferroni 0.025 — не согласованы.
- **Risk matrix**: три разных набора рисков с разными вероятностями/воздействиями (например, в одном "Privacy breach: Low/High", в другом "2/5, score 10").
- **Limitations**: три разных списка (8,6,7 пунктов) с разным содержанием.
- **Sample size justification**: одна версия ссылается на PMID 15527447 (ошибочно), другая — на 15333167.
- **Pre‑registration**: дублирующиеся записи с разными placeholder'ами.

Документы не согласованы между собой, методы не соответствуют теоретической рамке (фальсифицируемость). Условие 9 не выполнено.
