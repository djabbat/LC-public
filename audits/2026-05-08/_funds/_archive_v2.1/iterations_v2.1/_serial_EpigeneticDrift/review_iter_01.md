## Checklist (✓/✗ + объяснение по каждому из 9 условий)

### 1. Операционализированная фальсифицируемость (числовые пороги) — ✓
Пороги присутствуют: p<0.05, d≥0.3, 80% power, f²=0.15, |r|>0.3, p<0.01. Явно указаны условия Falsification Condition 1–3. В OPEN_PROBLEMS количественные правила для исходов A–D.

### 2. Pre-registration plan (OSF placeholder + date) — ✓
Указан placeholder https://osf.io/TBD, planned date 2026-09-01. Принято.

### 3. Sample size calculation (power analysis) — ✓
Для ABL-2 эксперимента: ΔD₄=0.2, α=0.05, power=0.80, two-tailed t-test → N=64/group. Для γ₄₃: f²=0.15, 5 predictors → N=92. Формула упомянута, софт G*Power.

### 4. Risk matrix ≥5 строк — ✓
Ровно 5 строк: ABL‑2 confound, separation β₄/α₄, no universal D₄, coupling not measurable, insufficient tissue diversity. Каждая с Probability/Impact/Mitigation.

### 5. Limitations section — ✓
Явный раздел в CONCEPT.md из 5 пунктов: causality, reversibility, layer inconsistency, tissue specificity, measurement noise. Без приукрашиваний.

### 6. Consortium / collaboration plan — ✓
Placeholder list из 3 групп с указанием ролей: Primary PI (теория), Prof. Horvath (DNA methylation clocks), Dr. Brunet (stem cell experiments).

- Другие ссылки (обрывочный список) не могут быть проверены полностью из-за отсутствия полного reference list.
