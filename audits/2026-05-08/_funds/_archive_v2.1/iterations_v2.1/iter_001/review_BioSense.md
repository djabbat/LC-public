## Peer Review: BioSense

### Verdict
**REVISE_MAJOR** — Проект имеет интересную концепцию, но текущая версия документов демонстрирует фундаментальные проблемы в обосновании центральной теории, отсутствие эмпирической поддержки для двух из трёх модулей и неопределённые критерии фальсификации. Допуск к финансированию невозможен без устранения перечисленных ниже P0-проблем.

---

### Scores (1-5)
| Критерий | Оценка | Обоснование |
|---|---|---|
| **Premise** | 2 | Ze Theory — нишевая, неверифицированная независимыми группами модель. Утверждение об универсальности χ_Ze для ВСР и ольфакции — голословно. |
| **Method** | 2 | Для ЭЭГ — приемлемо, но ad-hoc параметры (δ=0.1, 25–35 Гц) не обоснованы. Для ВСР и ольфакции — только эскизы, алгоритмы не верифицированы даже на синтетических данных. |
| **Evidence** | 3 | Единственный модуль с данными — ЭЭГ. Результаты нестабильны: d от 0.11 (ns) до 1.694, AUC=0.715 — ниже порога "good" (0.8), который сами авторы устанавливают. Для ВСР и ольфакции — ноль доказательств. |
| **Falsif** | 1 | Критерии фальсификации отсутствуют. Любой null-результат объясняется "недостаточной мощностью" или "неоптимальным контекстом". Теория не может быть опровергнута в рамках заявленного дизайна. |
| **Deliv** | 2 | План амбициозен до нереалистичности: три модуля, собственный APU, Турин-сенсор к 2028. Отсутствуют оценки бюджета, команды, запасных вариантов. MVP к Q4 2026 — сомнительно без предварительной валидации ВСР. |
| **Novelty** | 3 | Идея применить Ze к ВСР и ольфакции — новая, но опирается на слабый фундамент. Комбинация трёх сенсоров в одном браслете — engineering, не научное достижение. |
| **Risk** | 5 | Чрезвычайно высокий: центральная теория может быть ошибочна; ВСР и ольфакция — научные "линии на песке"; аппаратная разработка зависит от незрелых технологий (Турин). |

### Top 3 P0 issues

1. **Необоснованность Ze Theory как универсального биомаркера.**  
   v* = 0.45631 выводится из нерецензированной модели (уравнение v* = 1 - v*^2). Нет ни одной независимой репликации χ_Ze, кроме группы Ткемаладзе. Ссылка на "теоретическую модель динамических систем" без указания публикации — недопустима. **Требование:** предоставить либо peer-reviewed обоснование v* (статья в журнале с импакт-фактором >5), либо показать, что χ_Ze статистически значимо коррелирует с возрастом на независимом датасете (не использованном для калибровки) с AUC > 0.8.

2. **Отсутствие предварительных данных для модулей ВСР и ольфакции.**  
   Проект претендует на финансирование, основываясь на результатах только по ЭЭГ. Для ВСР нет даже пилотной симуляции на существующих базах (PhysioNet). Для ольфакции — нет ни одного опубликованного протокола, показывающего, что паттерны VOC различают возраст с AUC>0.7. **Требование:** до подачи заявки провести анализ хотя бы на одном датасете для ВСР (например, PhysioNet Ann Arbor ECTDB) и предоставить результаты (p-value, effect size). Для ольфакции — обзор литературы с метаанализом чувствительности VOC-маркеров возраста.

3. **Нефальсифицируемость гипотез.**  
   Нигде не указано, какой результат заставит авторов признать теорию неверной. Например: "Если χ_Ze не коррелирует с возрастом в ВСР при n ≥ 100 и достаточной мощности (β ≥ 0.8), то Ze-подход к ВСР отвергается". Сейчас любой null-результат будет списан на "недостаточную мощность", "неоптимальную полосу", "артефакты". **Требование:** для каждого модуля явно сформулировать H0 и H1 с критериями принятия/отвержения, включая минимальный размер эффекта и размер выборки.

**Дополнительные замечания:**
- В CONCEPT.md указано, что χ_Ze(молодые 20–30) = 0.87±0.04 на Cuban датасете, а в KNOWLEDGE.md — 0.5287±0.036. Противоречие: 0.87 vs 0.53. Скорее всего, в CONCEPT.md значения приведены для 25–35 Гц полосы, а в KNOWLEDGE.md — для широкополосных? Необходимо унифицировать и указать контекст.
- Проект не учитывает recent advances в wearables (например, Apple Watch уже измеряет HRV с RMSSD и pNN50, есть коммерческие e-nose). Утверждение "новая платформа" не обосновано преимуществом перед существующими.
- Отсутствует budget estimate и компетенции команды. Для EIC Pathfinder и NIH R01 это обязательные части.

**Резюме:** Текущий уровень проработки позволяет только начать переговоры с соавторами, но не подавать заявку в жёсткие научные фонды. После устранения P0-проблем (особенно пункта 1) и предоставления пилотных данных по ВСР можно будет повторно рассмотреть. CONCEPT.md и KNOWLEDGE.md нуждаются в существенной переработке для устранения внутренних противоречий и переоценки силы доказательств.
