## Архитектурные модули (целевые)

### Rust — Ядро симуляции

| Модуль | Файл/Папка | Назначение |
|--------|-----------|-----------|
| `DataIngestion` | `src/core/data_ingestion/` | Импорт CSV, JSON, Excel, статистических баз |
| `Normalization` | `src/core/normalization/` | Приведение к сетке 1 мес (0–300 мес) |
| `CVAnalysis` | `src/core/cv_analysis/` | CV для поперечных анатомических данных |
| `RangeAnalysis` | `src/core/range_analysis/` | Range_90_10 для эндокринологических данных |
| `TransitionDetection` | `src/core/transition_detection/` | Выявление переходов (алгоритм §2.2–2.3) |
| `FibrosisTracker` | `src/core/regeneration/` | Трекинг необратимых повреждений |

### Three.js / WebGL — Визуализация

| Модуль | Файл/Папка | Назначение |
|--------|-----------|-----------|
| 3D-модель | `src/viz/model3d/` | LOD × 4 уровня, антропометрическая интерполяция |
| Таймлайн | `src/viz/timeline/` | 0–300 мес, переходы, автовоспроизведение |
| Режимы | `src/viz/modes/` | Метафора (эмбриологическая) / Анатомический |
| Правая панель | `src/viz/panels/` | 3 окна: Психика / Когнитивные / Социальное |

## Ключевые числовые параметры (сводка → детали в PARAMETERS.md)

| Параметр | Значение |
|---------|---------|
| Возрастной диапазон | 0–25 лет (0–300 мес) |
| Шаг таймлайна | 1 месяц |
| Радиус кластера переходов | 6 месяцев |
| Целевой FPS | ≥ 30 (десктоп) |
| Число параметров | 24 |
| Органов в MVP | 5 (мозг, сердце, печень, почки, лёгкие) |
| Гормонов в MVP | 4 (GH, IGF-1, T/E2, кортизол) |
