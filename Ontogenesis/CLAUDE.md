# CLAUDE.md — Ontogenesis

Инструкции для Claude Code при работе с проектом Ontogenesis.

## Контекст проекта

**Ontogenesis** — интерактивная платформа симуляции онтогенеза человека (0–25 лет).
Проект входит в экосистему **AIM** (`~/Desktop/AIM/`).
Текущий статус: **Этап 0** (сбор и гармонизация данных).

## Правила разработки

### Язык кода
- **Ядро симуляции, обработка данных** → **Rust**
- **3D-рендер, UI** → **Three.js / WebGL** (JavaScript/TypeScript)
- **Скрипты ETL, импорт данных** → Python (вспомогательный)
- **Все текстовые задачи** (документация, статьи, описания) → **DeepSeek API** через `~/Desktop/AIM/llm.py`

### Числовые параметры
- Все числовые параметры хранятся в `PARAMETERS.md` — единый источник истины
- При изменении параметра: обновить `PARAMETERS.md` И `CONCEPT.md` одновременно
- Переходы между стадиями **только эмпирически** — не постулировать возраст/количество

### Структура папок
```
src/           — Rust + JS исходный код
data/          — сырые и нормализованные данные
scripts/       — ETL, импорт, вспомогательные скрипты
docs/          — документация, публикации
```

### Разрешённые файлы в корне
`README.md`, `CONCEPT.md`, `CLAUDE.md`, `TODO.md`, `PARAMETERS.md`, `MAP.md`, основной launcher `.sh`

Все остальные файлы — в соответствующие подпапки.

## Связь с экосистемой AIM

- Общие правила экосистемы: `~/Desktop/AIM/CLAUDE.md`
- DeepSeek API ключ: `~/.aim_env → DEEPSEEK_API_KEY`
- Точка входа LLM: `~/Desktop/AIM/llm.py`
- Модели: `deepseek-chat` (быстро) · `deepseek-reasoner` (сложные рассуждения)

## Алгоритм переходов (ключевая логика)

Реализуется в `src/` (Rust), параметры в `PARAMETERS.md §Transition Detection`:
- **Продольные данные:** изменение > порога SD от индивидуальной траектории
- **Поперечные данные:** CV(A,t) или Range₉₀₋₁₀ выходят за `mean + 2×SD`
- **Кластеризация:** радиус 6 месяцев, мин. стабильный период 3 месяца

## Публикации

При написании статей:
1. Использовать DeepSeek API
2. Включить самоцитирование (PMID 36583780, PMID 20480236, Zenodo CDATA/Ze)
3. Формат DOCX: Calibri, US Letter, 1" поля, APA 7
4. Статьи добавить в `~/Desktop/AIM/NEEDTOWRITE.md`
