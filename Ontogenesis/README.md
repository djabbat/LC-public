# Ontogenesis

**Интерактивная платформа симуляции онтогенеза человека от 0 до 25 лет**

Проект визуализирует физическое, эндокринное, когнитивное и социальное развитие человека с эмпирически выявленными переходами и модулем регенерации тканей.

## Что это

Ontogenesis — научно обоснованная интерактивная 3D-среда, в которой:

- 3D-модель тела меняет пропорции с возрастом (данные WHO, NHANES, CAESAR)
- Гормональные профили (GH, IGF-1, T/E2, кортизол) визуализируются как свечение органов
- Психологические, когнитивные и социальные параметры (24 итого) отображаются на правой панели
- Переходы между стадиями развития выявляются **эмпирически** (алгоритм CV / Range₉₀₋₁₀)
- Модуль регенерации показывает восстановление тканей с возрастной модуляцией

| Компонент | Технология |

## Структура проекта

```
Ontogenesis/
├── CONCEPT.md          — Полная концепция платформы (v4.1)
├── PARAMETERS.md       — Все числовые параметры (алгоритм, визуализация, регенерация)
├── TODO.md             — Задачи по этапам (Этапы 0–3)
├── MAP.md              — Карта файлов и архитектура
├── CLAUDE.md           — Инструкции для Claude Code
├── README.md           — Этот файл
├── src/                — Исходный код (Rust + JS, создаётся на Этапе 0–1)
├── data/               — Сырые и нормализованные данные (WHO, NHANES и др.)
├── scripts/            — Вспомогательные скрипты (импорт, ETL)
└── docs/               — Документация, публикации, методология
```

## Этапы реализации

| Этап | Срок | Содержание |
|------|------|-----------|
| Этап 0 | 12–18 мес | Сбор и гармонизация данных, алгоритм переходов |
| Этап 1 | 4–6 мес | Прототип 3D-визуализации, таймлайн |
| Этап 2 | 4–6 мес | Интеграция данных в 3D, правая панель |
| Этап 3 | 4–6 мес | Модуль регенерации, финализация |

Общий срок MVP: **24–36 месяцев** (старт 2026-03-27)

## Связь с экосистемой AIM

- Возрастные нормы гормонов/биомаркеров → используются в `AIM/lab_reference.py`
- Возрастная модуляция регенерации → совместима с `AIM/treatment_recommender.py`
- CDATA: нижняя граница CDATA-симуляции (25+) = верхняя граница Ontogenesis
