# Формальная теория Ontogenesis

## 1. Аксиоматическое основание

Теория Ontogenesis строится на следующих неделимых постулатах:

**Аксиома 1 (Этагенез).** Индивидуальное развитие и старение от зиготы до смерти представляет собой единый непрерывный процесс — этагенез (Etagenesis). Этот процесс может быть операционализирован как траектория в многомерном пространстве параметров, разделённая на эвристические, но значимые фазы.

**Аксиома 2 (Четырёхдоменная структура).** Состояние индивида в любой момент времени `t` полностью описывается вектором `S(t) = [M(t), P(t), F(t), S(t)]`, где:
*   `M(t)` — состояние морфологического домена (структурные параметры).
*   `P(t)` — состояние физиологического домена (функциональные и биохимические параметры).
*   `F(t)` — состояние психологического домена (когнитивные, эмоциональные и личностные параметры).
*   `S(t)` — состояние социологического домена (роли, события, сетевые параметры).

**Аксиома 3 (Динамическая сопряжённость).** Изменения в одном домене могут вызывать, модулировать или быть следствием изменений в других доменах. Эта сопряжённость (`γ`) является динамической величиной, которая может меняться в зависимости от фазы этагенеза и конкретных параметров.

## 2. Временная модель: Пять нейробиологических фаз

Основной временнóй решёткой является эмпирически выведенная модель пяти фаз (DOI 10.1038/s41467-025-65974-8):
```
Фаза I (Детство):      t ∈ [0, 9) лет
Фаза II (Подростково-молодёжная): t ∈ [9, 32) лет
Фаза III (Взрослая):   t ∈ [32, 66) лет
Фаза IV (Раннее старение): t ∈ [66, 83) лет
Фаза V (Позднее старение): t ∈ [83, 120] лет
```
Переломные точки {9, 32, 66, 83} рассматриваются как гипотезы, подлежащие репликации и уточнению на данных платформы. Трёхфазная модель Frolkis {Ontogenesis, Mesogenesis, Gerontogenesis} является комплементарным, более крупномасштабным представлением.

## 3. Математический аппарат: Dual Latent Change Score (LCS)

### 3.1. Базовая модель изменения
Для параметра `y` (принадлежащего одному из четырёх доменов) в дискретные моменты времени `t = 1, 2, ..., T` изменение моделируется как:
```
Δy_t = α_y + β_y * y_{t-1} + ζ_{y,t}          (Уравнение 1: Авторегрессионное изменение)
y_t = y_{t-1} + Δy_t                         (Уравнение 2: Наблюдаемый уровень)
```
где:
*   `Δy_t` — латентное (истинное) изменение параметра `y` между `t-1` и `t`.
*   `α_y` — константа изменения (аддитивный компонент, "thrust").
*   `β_y` — параметр самообратной связи (autoregressive). `β_y < 0` указывает на регулирование к заданному уровню (homeostatic pull), `β_y > 0` указывает на кумулятивный эффект (например, фиброз).
*   `ζ_{y,t} ~ N(0, σ²_ζ)` — стохастическая инновация, представляющая влияние неучтённых факторов.
*   `y_t` — наблюдаемое (или синтезированное) значение параметра.

### 3.2. Модель с кросс-доменным сопряжением (γ)
Для учета влияния параметра `x` (из другого домена) на изменение параметра `y`, Уравнение 1 расширяется:
```
Δy_t = α_y + β_y * y_{t-1} + γ_{y←x} * x_{t-1} + ζ_{y,t}   (Уравнение 3)
```
Коэффициент сопряжения `γ_{y←x}` количественно определяет, насколько предыдущее состояние параметра `x` влияет на последующее изменение параметра `y`. Например, `γ_{рост ← IGF-1}` ожидается положительным в Фазе I.

### 3.3. Векторная форма для системы доменов
Состояние системы из `D` параметров (объединённых по доменам) описывается векторным уравнением:
```
Δ𝐒_t = 𝛂 + 𝐁 * 𝐒_{t-1} + 𝛇_t
𝐒_t = 𝐒_{t-1} + Δ𝐒_t
```
где `𝐒_t` — вектор-столбец состояний, `𝛂` — вектор констант, `𝐁` — матрица коэффициентов, содержащая как автокорреляции (`β` на диагонали), так и кросс-доменные связи (`γ` вне диагонали). Матрица `𝐁` может быть блочно-разреженной, отражая гипотезы о силе связей между определёнными доменами в конкретных фазах.

## 4. Формальное определение возрастного метаморфоза

Пусть для домена `d` в момент времени `t` определён бинарный индикатор значимого перехода `T_d(t)`:
```
T_d(t) = 1, если |Δy_{d, t}| > κ_d * σ(Δy_d) для любого ключевого параметра y в домене d,
         0, иначе.
```
где `κ_d` — порог значимости (настраиваемый, по умолчанию 1.96), а `σ(Δy_d)` — стандартное отклонение изменений параметра.

**Определение 4.1 (Возрастной метаморфоз).** Метаморфоз регистрируется в момент времени `t`, если сумма переходов по всем доменам в окне `[t-δ, t+δ]` (где `δ = 0.5` года) превышает порог:
```
Metamorphosis(t) = 1, если Σ_{d ∈ {M,P,F,S}} [ max_{τ ∈ [t-δ, t+δ]} T_d(τ) ] ≥ 2,
                   0, иначе.
```
Проще: метаморфоз — это событие, когда значимые переходы происходят как минимум в двух разных доменах внутри 12-месячного окна.

**Гипотеза 4.2 (О числе метаморфозов).** Для полной траектории этагенеза (0–120 лет) ожидаемое число метаморфозов `N_meta` лежит в диапазоне `10 ≤ N_meta ≤ 14`. Их временное распределение не является равномерным и кластеризуется вокруг известных биосоциальных переходов.

## 5. Предсказания теории

1.  **Предсказание 1 (Фазовые переходы):** Статистический анализ синтезированных траекторий выявит значительное увеличение дисперсии скорости изменения (`Var(Δ𝐒_t)`) в окрестностях фазовых границ {9, 32, 66, 83 лет} по сравнению со средней дисперсией внутри фаз.
2.  **Предсказание 2 (Сеть сопряжений):** Матрица коэффициентов сопряжения `𝐁` будет иметь различную структуру для разных фаз. Например, в Фазе I-II доминируют связи `γ_{M←P}` и `γ_{P←M}` (рост ↔ гормоны), а в Фазе IV-V усиливаются связи `γ_{F←P}` и `γ_{S←F}` (когниции ← физиология, социум ← когниции).
3.  **Предсказание 3 (Метаморфозы и резильентность):** Индивиды с фенотипом "Delayed Aging" будут демонстрировать меньшее количество метаморфозов в Фазах IV и V, а также более низкие абсолютные значения коэффициентов самообратной связи `|β|` для параметров упадка (например, FibrosisTracker), что указывает на более устойчивый гомеостаз.
4.  **Предсказание 4 (Социум как модулятор):** Сила связи `γ_{F←S}` (влияние социальных параметров на психологические изменения) будет статистически значимым предиктором индивидуальной вариабельности в скорости когнитивного снижения в Фазе V, после контроля по базовым физиологическим параметрам.

## 6. Связь с MCOA формализмом

Ontogenesis устанавливает начальные условия для MCOA. По достижении `t = 25 лет` (конец модуля Ontogenesis, начало Mesogenesis) состояние индивида `𝐒(t=25)` преобразуется в начальные повреждения `D_i,0` для каждого из `N` клеточных/тканевых счётчиков в MCOA:
```
D_i,0 = f_i(𝐒(25))
```
где `f_i` — функция отображения, переводящая макроскопические параметры (например, объём органа, базовый уровень гормона) в проксимальный уровень начального клеточного стресса или повреждения для счётчика типа `i`. Это отображение является предметом текущих исследований и описано в [OPEN_PROBLEMS.md](./OPEN_PROBLEMS.md).

Теория Ontogenesis обеспечивает системный, количественный и проверяемый каркас для моделирования самой продолжительной и сложной "программы" в биологии — человеческой жизни.
