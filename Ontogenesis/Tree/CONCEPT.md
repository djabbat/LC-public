## Суть проекта

**Tree** — проект по построению **клеточного дерева дифференциации** (lineage tree) с использованием **транспланитрованных мутантных митохондрий** в качестве клонального маркера.

**Центральная идея:**
Мутантные митохондрии (с известными мутациями mtDNA), введённые в стволовую клетку, передаются дочерним клеткам асимметрично вместе с другими митохондриями — аналогично тому, как стареющие митохондрии сегрегируются по Katajisto 2015. Отслеживая распределение мутантного mtDNA по дочерним клеткам в режиме времени (scRNA-seq + mtDNA sequencing), можно **реконструировать дерево дифференциации** с клонально-митохондриальным разрешением.

## Биологические основания

### Почему митохондрии?
1. Митохондрии наследуются клонально (maternal inheritance)
2. Мутации mtDNA нейтральны в малых количествах и стабильны (drift-based lineage tracing)
3. Асимметричное наследование митохондрий при ACD (Katajisto et al., Science 2015):
   - Старые/повреждённые митохондрии (низкий ΔΨ) → дифференцирующаяся дочь
   - Молодые митохондрии (высокий ΔΨ) → стволовая дочь
4. Трансплантация: введение mitosomes с конкретной мутацией mtDNA позволяет создать **единственный в своём роде молекулярный штамп** клетки

### Связь с CDATA
Асимметричное наследование митохондрий при ACD тесно связано с асимметричным наследованием центриолей:
- Система I (структурная привязка): центриоль → апикально → стволовая дочь
- Система II (активный QC): молодые митохондрии → апикально → стволовая дочь; старые → базально → дифференцирующаяся
Обе системы работают на одной оси полярности. Мутантные митохондрии (как «чужеродные») могут нарушать нормальный QC-паттерн → создавать уникальные маркеры судьбы.

### Связь с Ontogenesis
**Особая связь:** Проект Tree предоставляет **экспериментальный инструмент** для валидации онтогенетических переходов, описанных в Ontogenesis:
- Дерево дифференциации Tree ↔ Emprical CV/Range transitions (Ontogenesis)
- Каждый узел дерева Tree соответствует эмпирическому переходу Ontogenesis
- Потенциально: 3D-визуализация дерева Tree в движке Ontogenesis

## Технический подход

### Метод трассировки
1. **Donorные митохондрии:** выделить митохондрии от донора с уникальной мутацией mtDNA (природная гетероплазмия или CRISPR-введённая точечная мутация)
2. **Трансплантация:** микроинъекция или MitoCapture-наночастицы в реципиентную стволовую клетку
3. **Отслеживание:** scRNA-seq + mitochondrial genome sequencing (mtscATAC-seq или similar)
4. **Реконструкция:** bioinformatic lineage tree reconstruction (LINEAGE/LARRY/Phyloseq)

### Ключевые вопросы
1. Как мутантные митохондрии распределяются при ACD относительно нормальных?
2. Можно ли реконструировать полное дерево дифференциации от единственной трансплантированной СК?
3. Как соотносится позиция клетки в дереве с её эпигенетическим статусом?
4. Какова роль центриолей в определении судьбы мутантных митохондрий?
