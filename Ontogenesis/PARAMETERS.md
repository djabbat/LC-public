# Параметры Ontogenesis: таблица, источники, статус

В таблице перечислены все параметры, отслеживаемые платформой, с указанием домена, единиц измерения, источника данных для диапазонов и статуса калибровки в LCS-модели.

**Легенда статуса калибровки:**
*   ✅ **Calibrated:** Параметры LCS (`α`, `β`, `σ_ζ`) оценены на реальных продольных данных с удовлетворительной точностью.
*   ⚠️ **Estimated:** Параметры оценены на поперечных данных, экстраполяции или экспертном мнении. Требует валидации.
*   ❌ **Theoretical:** Параметр введён теоретически, эмпирических данных для калибровки недостаточно. Используются значения по умолчанию.
*   **N/A:** Не применимо (например, для бинарных маркеров).
