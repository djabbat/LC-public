The LC ecosystem

Six core sub-projects, a supporting clinical pilot, and a Diamond Open Access journal — each autonomous, sharing one ecosystem. Click any card for the full project page.

MCAOA →

Multi-Counter Architecture of Organismal Aging — meta-framework of parallel damage-accumulation counters across tissues.

CDATA →

Centriolar Damage Accumulation Theory of Aging — hypothesis that mother-centriole damage drives stem-cell aging. Tqemaladze 2023.

Ze Theory →

Entropy-geometric formalism with a theoretical aging-time invariant. Open interactive simulator.

BioSense →

v* empirically verified

Wearable platform measuring the χ_Ze aging-activity biomarker from EEG, HRV, respiration, and sleep — privacy-first, on-device.

FCLC →

Federated Clinical Learning Cooperative — privacy-preserving infrastructure for cross-institutional aging research, GDPR-aligned.

Hive →

queen up

Federated AIM agent network — workers run AIM locally, queen distills eval-gated skill updates. Opt-in.

Activated

Anemia-management clinical pilot in Tbilisi — real-patient cohort for χ_Ze validation.

Annals of Rejuvenation Science →

e-ISSN 3088-439X

Diamond Open Access journal published by Georgia Longevity Alliance. OJS platform. DOAJ submission planned January 2027 (12-month wait period from first issue).
