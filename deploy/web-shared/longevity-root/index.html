Theoretical · meta-framework

MCAOA — Multi-Counter Architecture of Organismal Aging

submitted Nature Aging NOT peer-reviewed
Live at
mcaoa.longevity.ge · static landing with the full theory, axioms, and falsification protocol
For
biogerontologists evaluating multi-counter aging models; reviewers of NATAGING-P13741
Status
Theory complete · awaiting Nature Aging decision · counter-weight calibration pending FCLC v14
Stack
Rust simulator (MCAOA/crates), Python tooling, OMOP-compatible cohort schemas

Aging formalised as L_tissue = Σᵢ wᵢ(tissue) · fᵢ(Dᵢ(n,t)) — weighted sum of parallel damage-accumulation counters. Each counter has division-linked (αᵢ) and time-linked (βᵢ) kinetics. Tissue weights wᵢ fixed a priori from independent cell-biological knowledge (preserves falsifiability).

Axiom M4 (operational falsifiability): MCAOA is falsified if on a pre-registered cohort N ≥ 2000 at α = 0.001, partial r² for all-cause mortality (after age + sex) falls below 0.05 for every counter.

Manuscript ID: NATAGING-P13741 (under review).

Molecular-cellular · counter candidate

CDATA — Centriolar Damage Accumulation Theory of Aging

status: INCONCLUSIVE
Live at
cdata.longevity.ge · landing with hypothesis, evidence dossier, and Sobol-paradox notebook links
For
cell biologists working on centriole inheritance, asymmetric stem cell division, polyglutamylation
Status
Open hypothesis · Cell-DT v3.x sensitivity analysis inconclusive · awaiting GTEx N=948 decomposition (Cell-DT v4.0)
Stack
Rust simulator (Cell-DT), Python notebooks, GTEx tissue atlas

Hypothesis: progressive polyglutamylation of mother centriole in stem cells drives cellular aging. Tkemaladze 2023 PMID 36583780.

Sobol global sensitivity analysis on synthetic data showed an apparent paradox (R²no_α=0.833 > full R²=0.778), but the difference is NOT statistically significant after nested cross-validation (p=0.12 after correction). Full S1+S2+ST decomposition on real GTEx N=948 deferred to Cell-DT v4.0.

CDATA remains an open, falsifiable hypothesis; provisional Counter #1 candidate, not a confirmed counter.

Mathematical · ansatz

Ze Theory

internal manuscript, NOT peer-reviewed
Live at
ze.longevity.ge · interactive Phoenix simulator: drag k_λ, observe χ_Ze fixed point convergence in real time
For
theorists testing the entropic-geometric ansatz; physicists checking the CHSH deformation derivation
Status
Simulator working · v* = 0.45631 derived analytically · independent biological validation via BioSense pending
Stack
Elixir/Phoenix LiveView, Rust impedance-ODE solver, JS chart layer

Entropy-geometric formalism. Central law dτ_Ze/dt = −α·I(Z) where I(Z) is the Kullback–Leibler divergence between actual and modelled state. Postulated ansatz for biological systems (motivated by Burgholzer 2015 + Pearson 2021 — those concern PHYSICAL clocks; biological extension hypothesis-stage).

From the ansatz the simulator derives mathematically: a quadratic CHSH deformation E_Ze = −a·b + δ·[(a·b)² − 1/3], the LGI-QFI bound from Abboud 2026, and the χ_Ze fixed point v* = 0.45631 at k_λ = 1.

BioSense

v* empirically verified (consistent)
Live at
biosense.longevity.ge · Phoenix dashboard: upload sample EEG/HRV, see χ_Ze trajectory, compare to v* fixed point
For
wearable-device researchers, sleep scientists, exacerbation-prediction clinicians, AoU/UK Biobank reusers
Status
v* = 0.45631 confirmed on All-of-Us N=500 (95% CI 0.443–0.459) · pre-registered N≥2000 confirmatory trial pending EIC funding
Stack
Elixir/Phoenix Channels (real-time EEG stream), Python/NumPy χ_Ze estimator, mobile-friendly WebUI

Wearable platform measuring χ_Ze biomarker from EEG + HRV + respiration + sleep. Variational principle F = E − T·S − λ·I_pred; theoretical fixed point v* = 0.45631 (sensitivity range [0.32, 0.58] for k_λ ∈ [0.5, 2.0]).

Empirically tested via swept-v* search on All-of-Us N=500: v*_optimal = 0.451 (95% CI 0.443–0.459) — consistent with theory.

Honest disclosure: multimodal weights (0.30, 0.30, 0.20, 0.20) are post-hoc pilot fits, not theory-fixed. Confirmatory validation requires pre-registered cohort N ≥ 2000.

Stack

Privacy-preserving federated learning for cross-clinic χ_Ze model training without raw patient data transfer. Stack: differential privacy (Mironov 2017 RDP composition, ε ≤ 1.0/round, ε_total ≈ 0.43 at σ=1.5, q=0.013, T=5), k-anonymity (k ≥ 7), Krum Byzantine-robust aggregation (≤ 25% malicious clients), secure aggregation (Bonawitz 2017).

Threat model (explicit): secure ONLY against semi-honest server + Byzantine clients. NOT secure against active server collusion or malicious server. GDPR Article 9 blocker until FCLC v14 (active-adversary migration, planned Q1 2027).

Infrastructure · agent intelligence

Hive — federated AIM agent network

v1 production queen up
Live at
hive.longevity.ge · queen landing with OS-detected install for Linux/macOS/Windows, privacy contract, healthz
For
AIM users running local nodes who want their workers to share anonymized eval/skill signal and pull cross-worker improvements
Status
Queen live (FastAPI + SQLite) · workers (bees) opt in via AIM_HIVE_QUEEN_URL · DP-budget + Shapley scoring planned (see AIM/AI/FCLC_BORROW.md)
Stack
Python AIM core, FastAPI/uvicorn, SQLite WAL, systemd-hardened service, Cloudflare proxied

Federated collective intelligence for AIM agents, conceptually patterned on FCLC. Workers (bees) accumulate diagnostic runs, reflexion themes, skill usage, eval results locally. Periodically a worker packages an anonymized signal — L_PRIVACY filter strips PII (email/phone/path/name/PMID/DOI) — and posts to /v1/hive/contribute. The queen detects cross-worker patterns (compliance drift, theme convergence) and publishes eval-gated skill updates back to /v1/hive/updates. Each worker re-runs three local gates before installing: L_CONSENT (opt-out), signature integrity, L_VERIFIABILITY (re-eval).

Each bee is fully functional offline — Hive participation is opt-in and never required for AIM to work.
