## 0. Назначение

**Organismal Aging** — самообучающийся 4D-симулятор организма (3D + время), моделирующий развитие с зиготы через онтогенез к старению и смерти. Старение рассматривается как **основа всех заболеваний** (geroscience paradigm). Симулятор интегрирован с **физическим роботом ARGUS-LP** для автоматической валидации гипотез на физических симуляторах (INFOGEST, органоиды, клеточные культуры).

**Ключевые принципы:**
- **Старение — первопричина заболеваний.** Не «возраст-ассоциированные», а «возраст-обусловленные» (aging-driven).
- **Три уровня:** Центриоль (энтропия) → MCAOA-счётчики → Тканевые Ze-конфликты.
- **Самообучение:** Симулятор формулирует гипотезу → ARGUS-LP тестирует на физическом симуляторе → результат возвращается в модель → модель уточняется.
- **Универсальность видов:** Параметризуемая модель — от одноклеточных до млекопитающих.
- **Открытость:** Apache 2.0, открытые данные, открытое железо (ARGUS-LP).

## 1. Три уровня модели

### Уровень #1: ЦЕНТРИОЛЬ — Накопитель энтропии (Counter #1)

**Постулат:** Центриоль — единственная внутриклеточная структура, **не способная к самообновлению**. Все остальные структуры (белки, мембраны, митохондрии, ДНК) постоянно обновляются через синтез/деградацию. Центриоль передаётся как шаблон при делении — старая центриоль всегда попадает в одну из дочерних клеток.

| Свойство | Центриоль | Все остальные структуры |
|---|---|---|
| Обновление | Только репликация (1×/клеточный цикл) | Постоянный синтез/деградация |
| Накопление повреждений | **Да, необратимо** | Обратимо через turn-over |
| Энтропийная роль | **Накопитель энтропии** | Самоочищение |
| Связь со старением | **Первопричина** | Вторичные эффекты |

**Математически:**
```
S_centriole(t) = S₀ + ∫₀ᵗ η(τ) dτ

где:
  S_centriole(t) — энтропия центриоли [0,1]
  S₀ — начальная энтропия (при рождении/зиготе)
  η(τ) = η_div · n(τ)/n* + η_time · τ/τ* + η_ROS · ROS(τ) + η_rep · ε_rep(τ)
  n(τ) — число делений к моменту τ
  n* — референсное число делений (Хейфлик)
  τ* — референсное время (для постмитотических клеток)
```

### Уровень #2: 5 ТРЕКОВ MCAOA (Counters #2–#5)

> **Примечание:** Counter #6 (piRNA) исключён из v1.0. Оставлен как слот расширения.

| # | Счётчик | Механизм | Зависимость от #1 |
|---|---|---|---|
| **#2** | Теломерный | Укорочение теломер; лимит Хейфлика | Слабая (ориентация деления) |
| **#3** | Митохондриальный | АФК, дисфункция дыхательной цепи | Умеренная (неравное распределение) |
| **#4** | Эпигенетический | Метилирование ДНК, потеря полярности (Cdc42) | Сильная (центриоль ↔ полярность) |
| **#5** | Протеостатический | Агрегация белков, снижение шаперонов | Умеренная (организация синтеза) |

**Агрегатор бремени старения ткани:**
```
L_tissue(species, tissue, t) = Σᵢ₌₁⁵ wᵢ(species, tissue) · fᵢ(Dᵢ(t))

где:
  fᵢ(x) = 1 / (1 + exp(−kᵢ(x − x_crit)))  — сигмоида
  wᵢ — видоспецифичный и тканеспецифичный вес
```

### Уровень #3: ТКАНИ + Ze-КОНФЛИКТЫ + ОНТОГЕНЕЗ

**8 базовых типов тканей (с возможностью расширения):**

| # | Ткань | τ_renewal (человек) | Доминирующий счётчик | Уязвимость |
|---|---|---|---|---|
| 1 | Эпидермис | ~28 дней | #2 (теломеры) | Быстрое деление → истощение |
| 2 | Кишечный эпителий | ~5 дней | #5 (протеостаз) | Сверхбыстрое → агрегация |
| 3 | Гепатоциты | ~300 дней | #5 (протеостаз) | Медленное → накопление |
| 4 | Нейроны | Никогда (постмитотические) | #3 (митохондрии) | АФК, mtDNA мутации |
| 5 | Гемопоэтические стволовые | ~месяцы (покоящиеся) | #4 (эпигенетика) | Cdc42, дрейф |
| 6 | Кардиомиоциты | ~0.5%/год | #3 (митохондрии) | АФК, энергетика |
| 7 | Эндотелий | ~3 года | Все счётчики | Медленное → комплексное |
| 8 | Костная ткань | ~10 лет | #1 (центриоль) | Сверхмедленное → энтропия |

**Ze-конфликт между тканями:**
```
Z_conflict(i, j, t) = |τᵢ · dLᵢ/dt − τⱼ · dLⱼ/dt| · C_ij(species)

где:
  τᵢ — период самообновления ткани i
  C_ij — видоспецифичная матрица связности тканей
```

При Z_conflict > Z_crit → функциональный разрыв → недостаточность органа → disease onset.

## 2. 4D-Модель организма (3D + время)

### Развитие с зиготы

```
Zygote
  │ t=0: S_centriole = S₀ (минимальная энтропия центриолей)
  │ t=0→embryogenesis: быстрые деления, L_tissue ≈ 0
  ▼
Embryo
  │ Дифференцировка тканей, установка τ_renewal
  ▼
Neonate
  │ Все счётчики на стартовых значениях Dᵢ₀
  ▼
Adult (пик функции)
  │ Накопление Dᵢ(t), рост L_tissue(t)
  ▼
Aging (L_tissue > 0.30)
  │ Межтканевые конфликты, Z_conflict растёт
  ▼
Disease onset (L_tissue > 0.60 или Z_conflict > Z_crit)
  │
  ▼
Death (L_tissue → 1.0 или критический орган отказывает)
```

### Пространственная модель

Каждая ткань имеет:
- **3D-объём** (анатомическая позиция)
- **Клеточную плотность** (клеток/мм³)
- **Сосудистую сеть** (перфузия)
- **Иннервацию** (нейронные связи)
- **Микробиомный слой** (для пограничных тканей: кожа, кишечник, лёгкие)
- **Макробиомный контекст** (для кишечника — пищевой поток)

### Временная шкала

- **Δt_min = 1 час** (клеточный цикл быстрых тканей)
- **Δt_max = 1 год** (для медленных тканей — адаптивный шаг)
- **Total T_max = 120 лет** (человек) или видоспецифичный максимум

```

где:
  θ = {S₀, η_div, η_time, η_ROS, η_rep, wᵢ, kᵢ, x_crit, τ_renewal, C_ij, ...}
  data = результаты ARGUS-LP + INFOGEST + литература
```

Модель параметризуется под любой вид:

| Параметр | Человек | Мышь | C. elegans | E. coli |
|---|---|---|---|---|

## 5. Старение как основа всех заболеваний

Вместо «age-associated diseases» — **aging-driven diseases**:

```
L_tissue(t) → L_critical → disease onset

Примеры:
  L_vascular > 0.60 + Z_conflict(vascular, cardiac) > Z_crit → CVD
  L_neural > 0.60 + Z_conflict(neural, vascular) > Z_crit → нейродегенерация
  L_immune > 0.60 → иммуносенесценция → инфекции, рак
  L_bone > 0.60 + L_joint > 0.60 → остеоартрит, остеопороз
```

**Травмы и острые заболевания** — как внешние возмущения модели:
```
Dᵢ(t+t_trauma) = Dᵢ(t) + ΔD_trauma
L_tissue(t+t_trauma) = Σ wᵢ · fᵢ(Dᵢ + ΔD)
```

## 7. Принципы расширяемости

```rust
trait AgingCounter {
    fn name(&self) -> &str;
    fn damage(&self, n_divisions: f64, t: f64, species: &Species) -> f64;

```

```rust
struct SpeciesConfig {
    name: String,
```
