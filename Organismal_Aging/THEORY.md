## §0. Методологическое предуведомление

Настоящая теория различает **четыре** класса утверждений. Особое внимание — к границе между физическими теоремами и биологическими анзацами.

| Класс | Обозначение | Определение | Пример |
|---|---|---|---|
| **Теорема** (физика/математика) | 📐 | Строго доказанное в своей области | v* = 0.45631 (численный экстремум) |
| **Анзац** (перенос в биологию) | 📌 | Перенесён в биологию по аналогии, не доказан | dτ_Ze/dt = −α·I(Z) — по аналогии с физическими часами |
| **Гипотеза** (биология) | 🔬 | Проверяемое предположение | Z_conflict как предиктор болезней |
| **Модельный параметр** | ⚙️ | Значение для замыкания модели, не измерено | w_i, k_i, x_crit — калибровочные |

**Критическое правило:** Анзацы и гипотезы не должны повышаться в статусе без экспериментального подтверждения. Модельные параметры должны быть явно обозначены.

## §2. Уровень #1: Энтропия центриоли

### 2.1 Рабочая гипотеза (статус: 📌)

> **Гипотеза: центриоль имеет ограниченное самообновление (self-renewal). В отличие от большинства клеточных структур — белки (turnover часы–дни), мембраны (дни), митохондрии (дни–недели), ДНК (репарация постоянно) — центриоль в постмитотических клетках не обновляется, а в делящихся обновляется только через репликацию (1×/клеточный цикл). Предполагается, что это делает центриоль потенциальным накопителем энтропии в клетке. Статус: рабочая гипотеза, не установленный факт. Прямых измерений энтропии центриоли не существует.**

**Примечание о v*:** Два значения v* (0.45631 Python, −0.08738 Article) — ЭТО ОДНА И ТА ЖЕ ВЕЛИЧИНА в разных системах координат. Формула пересчёта: Article = 2 × Python − 1. Python ∈ [0,1]; Article ∈ [−1,+1]. Расхождение消除了.

### 2.2 Косвенные подтверждения энтропийного старения

| Система | PMID | Находка |
|---|---|---|
| Мышечные волокна | 41724675 | Энтропия гистологии мышц предсказывает мобильность у пожилых (Hong 2026, *Aging Cell*) |
| Метилирование ДНК | 40096548 | Энтропия метилирования — биомаркер старения (Chan 2025, *Aging*) |
| Метилирование ДНК | 36797759 | Энтропия метилирования как мера репликации стволовых клеток (Vaidya 2023, *Genome Biol*) |
| Хроматин | 23058597 | Хроматиновая энтропия эритроидных предшественников растёт с возрастом (Pantic 2012) |
| C. elegans | 19808878 | Текстурная энтропия изображений количественно определяет старение (Shamir 2009, *Bioinformatics*) |
| Термодинамика | 38367762 | Диссипативное масштабирование развития и старения многоклеточных (Kriete 2024, *Biosystems*) |

**Вывод:** Энтропия как маркер старения подтверждена на множестве систем. Специфическая гипотеза центриолярной энтропии остаётся постулатом до ARGUS Phase B.

### 2.3 Основное уравнение

```
S_centriole(t) = S₀ + ∫₀ᵗ η(τ) dτ                           [безразмерная, 0..1]

η(τ) = η_div·(n(τ)/n*) + η_time·(1/τ*) + η_ROS·ROS_norm(τ) + η_rep·ε_rep(τ)

где:
  S_centriole ∈ [0, 1]                    — энтропия центриоли
  η_div      = 0.02/n*          [1/деление] — [ОЦЕНКА] на основе PMID 36583780
  η_time     = 0.010            [1/год]     — [ОЦЕНКА] калибрована под смерть ~100 лет
  η_ROS      = 0.01             [1/год]     — [ОЦЕНКА] 
  η_rep      = 0.001            [1/деление] — [ОЦЕНКА]
  ROS_norm   ∈ [0, 1]                       — нормализованный уровень АФК
  n*         = 50 (человек)                 — [ФАКТ] лимит Хейфлика
  τ*         = 1 год                        — [КОНВЕНЦИЯ]
```

### 2.4 Фальсифицируемый тест

**Гипотеза H₀:** polyGlu на материнской центриоли НЕ коррелирует с числом клеточных делений.

**Тест:** ARGUS-LP, Phase B. BJ-hTERT + mEos3.2-CEP135, 9 блоков × 24-72 ч. N≥100 делений.

**Порог опровержения:** Коэффициент корреляции Пирсона r < 0.3 при p < 0.05.

| # | Счётчик | Confidence | Основание | План повышения |
|---|---|---|---|---|
| #1 | Центриолярный | 0.3 | Постулат, косвенные данные | ARGUS Phase B |
| #2 | Теломерный | 0.9 | Хейфлик 1965, сотни подтверждений | — |

| #5 | Протеостатический | 0.4 | VEXAS — патология; PPIA — одна работа | MCAOA Test 2 |

### 3.2 Кинетика счётчика

```
Dᵢ(n, t) = Dᵢ₀ + αᵢ·(n/nᵢ*) + βᵢ·(t/τᵢ) + γᵢ·I(другие счётчики)

где:
  Dᵢ ≥ 0                    — накопленное повреждение (безразмерное)
  αᵢ ≥ 0                    — интенсивность от делений
  βᵢ ≥ 0                    — интенсивность от времени
  γᵢ ∈ ℝ                    — сила связи (канонически: 0)
  nᵢ* — референсное число делений [ФАКТ: Хейфлик]
  τᵢ  — референсное время [с цитированием]
```

### 3.3 Референсные времена τᵢ (с обоснованием)

| Счётчик | τᵢ (лет) | Обоснование |
|---|---|---|
| #2 Теломеры | 0.019 (1 нед) | Turnover теломерных повторов; Blackburn 2015 |
| #3 Митохондрии | 0.082 (30 дн) | Turnover mtDNA через митофагию; Youle 2012 (PMID 22252130) |
| #4 Эпигенетика | 3.6 | DunedinPACE — темп старения; Belsky 2022 (PMID 35029144) |
| #5 Протеостаз | 0.027 (10 дн) | Turnover PPIA — доминантный шаперон HSC; Maneix 2024 |

### 3.4 Агрегатор бремени ткани

```
L_tissue = w_centriole·S_centriole + Σᵢ₌₂⁵ wᵢ·fᵢ(Dᵢ)

fᵢ(x) = 1 / (1 + exp(−kᵢ·(x − x_crit)))            — сигмоида (каноническая)

Условие нормировки: w_centriole + Σwᵢ = 1.0
w_centriole = 1.0 − Σwᵢ                              — вычисляется как остаток
                                                         (не из независимых данных — [ОЦЕНКА])
```

**Примечание рецензента:** w_centriole вычисляется как остаток, а не из независимых биологических данных. В будущих версиях — заменить на экспериментально измеренный вклад polyGlu в тканевую дисфункцию.

### 3.5 Калибровка через Frailty Index (НЕЗАВИСИМАЯ)

> **Важно:** Параметры модели (η, α, β, w_i) НЕ подгонялись под FI. Они взяты из независимых биологических источников: Хейфлик, DunedinPACE, turnover times. FI используется исключительно как операциональный мост для сравнения предсказаний модели с клиническими данными.

```
FI(t) = 0.7 × L_max(t)          — операционализация для сравнения

Пороги (Rockwood 2005, PMID 16129869):
  FI < 0.21 → здоровье (соответствует L_max < 0.30)
  FI > 0.42 → болезнь/госпитализация (L_max > 0.60) — Searle 2008, PMID 18671847
  FI > 0.70 → смерть (L_max > 1.00)

Связь FI со смертностью: Gompertz hazard doubles per 0.07 FI increment
  (Mitnitski 2002, PMID 12456714)
```

### 3.6 Анализ чувствительности сигмоиды

Параметр k определяет крутизну перехода повреждение→бремя. Проведён анализ:

| k | Форма | Влияние на возраст болезни |
|---|---|---|
| 3 | Плавная | Болезни рано, плавное нарастание |
| 5 | Умеренная | Среднее |
| **8** | **Крутая (каноническая)** | **Позднее начало, быстрый переход** |
| 12 | Очень крутая | Почти ступенчатая — болезни возникают внезапно |

**Канонический выбор k=8:** компромисс между постепенностью накопления повреждений и пороговым характером клинических проявлений. Требуется валидация на реальных данных.

## §4. Уровень #3: Ткани + Ze-конфликты

### 4.1 Тканевая модель

Каждая ткань характеризуется:
- τ_renewal — период самообновления (дни) [ФАКТ: стандартные данные клеточной биологии]
- w_i — веса счётчиков [ОЦЕНКА: требуют калибровки]
- L_crit — критическое бремя [ОЦЕНКА: предварительные]
- 3D-позиция, сосудистая плотность, иннервация [ФАКТ: анатомические данные]

### 4.2 Ze-конфликт (статус: 🔬 ГИПОТЕЗА)

> **Предупреждение рецензента:** Z_conflict — элегантная гипотеза без прямой эмпирической базы. C_ij полностью неизвестна. Требуется экспериментальная проверка.

```
Z_conflict(i, j, t) = |τᵢ·dLᵢ/dt − τⱼ·dLⱼ/dt| · C_ij

C_ij — сила анатомической/функциональной связи [НЕИЗВЕСТНА; по умолчанию 1.0]
```

**Косвенное подтверждение:**
- Сердечно-сосудистые заболевания как конфликт стареющего эндотелия и кардиомиоцитов: Lakatta & Levy (2003, PMID 12515756) — arterial aging and CVD
- Почечная недостаточность как конфликт клубочков (τ~200 дней) и канальцев (τ~2 года)

**Предлагаемый эксперимент:** Ко-культуры старых и молодых клеток разных тканей in vitro. Измерение апоптоза, секреции SASP, нарушения барьерной функции. Ожидается: Z_conflict > Z_crit → ускоренная дисфункция.

### 4.3 Онтогенез

```
Zygote: S_centriole = S₀ = 0.01, все Dᵢ = 0, все L_tissue = 0
Embryo: дифференцировка, установка τ_renewal и w_i
Neonate: стартовые значения, L ≈ 0
Adult: L растёт медленно (0 → 0.30 к 30 годам)
Aging: L > 0.30, Z_conflict растёт
Disease: L_k > L_crit_k ИЛИ Z_conflict(i,j) > Z_crit
Death: max(L_tissue) → 1.0 ИЛИ возраст > max_lifespan
```

## §5. Самообучение (Байесовский контур) — детальная методология

### 5.1 Постановка задачи

Цель: по данным экспериментов ARGUS-LP, INFOGEST и литературы обновлять параметры модели θ.

```
θ = {S₀, η_div, η_time, η_ROS, η_rep, αᵢ, βᵢ, kᵢ, x_critᵢ, wᵢⱼ, τ_renewalⱼ, C_ij}

Размерность θ: ~35 параметров (человек, 8 тканей)
```

### 5.2 Функция правдоподобия

```
P(data | θ) = Πₖ P(dataₖ | θ)

Где dataₖ — независимые эксперименты:
  data_ARGUS: {polyGlu_i, division_count_i} для N≥100 клеток
  data_INFOGEST: {digestion_rate, SCFA, ROS_impact}
  data_GTEx: {L_tissueⱼ} для 8 тканей × образцов
  data_UKBiobank: {age_death, FI, disease_onset}

Для непрерывных данных:
  P(dataₖ | θ) ~ Normal(предсказание(θ), σ²ₖ)
  σ²ₖ — дисперсия измерения, оценивается из данных
```

### 5.3 Априорные распределения

```
η_div  ~ LogNormal(ln(0.0004), 0.5)     // [ОЦЕНКА] на основе n*=50
η_time ~ LogNormal(ln(0.010), 0.5)      // [ОЦЕНКА] калибрована
αᵢ     ~ LogNormal(ln(0.001), 1.0)      // Слабое априорное
βᵢ     ~ LogNormal(ln(0.005), 1.0)      // Слабое априорное
kᵢ     ~ Gamma(8, 1)                     // Центрировано на 8
wᵢⱼ    ~ Dirichlet(1, 1, 1, 1)          // Равномерное по симплексу
```

### 5.4 Алгоритм обновления (MCMC)

```
1. Инициализация: θ₀ = default_params()
2. Для t = 1..T:
   a. Предложить θ' ~ q(θ' | θ_{t-1})  // No-U-Turn Sampler (NUTS)
   b. Вычислить P(data | θ') · P(θ')
   c. Принять/отклонить по Метрополису-Гастингсу
3. Возврат: posterior(θ) — MCMC-цепочка

ПО: PyMC (Python) или Stan (C++)
Сходимость: R-hat < 1.01, ESS > 400
```

### 5.5 Режим «симуляции эксперимента»

Для предотвращения остановки обучения в ожидании реальных данных ARGUS-LP:

```
Пока ARGUS-LP занят (24-72 ч на блок):
  data_synthetic = simulate_experiment(θ_current)
  θ' = bayesian_update(data_synthetic)
  // Поддержание цепи MCMC «горячей»

При получении реальных данных:
  θ' = bayesian_update(data_real)
  // Коррекция смещения синтетических данных
```

## §7. Нулевая модель и Decision Tree

### Нулевая модель (H₀)
Симулятор должен сравниваться с нулевой моделью: **стохастическое старение без центриоли.**
- L_tissue = random_walk(t) с дрейфом, оценённым по популяционным данным
- Если Organismal Aging НЕ outperforms H₀ (ΔAIC < 2) — модель отвергается

### Decision Tree при опровержении

```
ARGUS Phase B: r < 0.3 (N≥300)?
  ├── ДА → центриолярная гипотеза ОТВЕРГНУТА
  │         → Уровень #1 удаляется из v2.0
  │         → фокус на счётчиках #2-#5
  └── НЕТ → гипотеза сохраняется

MCAOA M4: partial r² < 0.05 для счётчика?
  ├── ДА → счётчик ОТВЕРГНУТ
  │         → удаляется из модели
  └── НЕТ → счётчик сохраняется

Z_conflict: AUC < 0.6?
  ├── ДА → гипотеза ОТВЕРГНУТА
  │         → Z_conflict удаляется, остаётся только L_tissue
  └── НЕТ → гипотеза сохраняется
```
