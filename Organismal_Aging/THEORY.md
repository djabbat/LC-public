## §1. Аксиоматика

### A1. Центриолярная энтропия (первопричина)
Центриоль — единственная внутриклеточная структура, не способная к самообновлению. Все остальные структуры обновляются через синтез/деградацию. Следовательно, центриоль является **первичным накопителем энтропии** в клетке.

### A2. Множественность счётчиков (MCAOA)
Старение организма управляется как минимум 5 параллельными механизмами накопления повреждений. Ни один счётчик не является достаточным для объяснения всех аспектов старения.

### A3. Тканевая специфичность
Каждая ткань имеет уникальный профиль весов w_i для счётчиков, определяемый её биологией (период самообновления, метаболическая активность, экспрессия генов).

### A4. Межтканевые конфликты (Ze)
Ткани с разными периодами самообновления вступают в конфликт при взаимодействии. Рассогласование скоростей старения создаёт функциональные разрывы.

### A5. Старение → Болезнь
Заболевания — не ассоциированные с возрастом, а **обусловленные** старением. При L_tissue > L_critical disease onset неизбежен.

### A6. Фальсифицируемость
Каждое утверждение должно сопровождаться операциональным тестом, результат которого может его опровергнуть.

## §2. Уровень #1: Энтропия центриоли

### 2.1 Основное уравнение

```
S_centriole(t) = S₀ + ∫₀ᵗ η(τ) dτ

η(τ) = η_div · ν_div(τ) + η_time · ν_time(τ) + η_ROS · ν_ROS(τ) + η_rep · ν_rep(τ)

где:
  S_centriole ∈ [0, 1]                            — безразмерная энтропия
  ν_div(τ) = n(τ) / n*                              — нормализованная частота делений
  ν_time(τ) = 1 / τ*                                — нормализованное хронологическое время
  ν_ROS(τ) = ROS(τ) / ROS_max                       — нормализованный окислительный стресс
  ν_rep(τ) = ε_rep(τ)                                — частота ошибок репликации центриоли
```

### 2.2 Параметры (человек)

| Параметр | Значение | Источник |
|---|---|---|
| S₀ | 0.01 (зигота) | Постулируется |
| η_div | 0.02 / n* | Tqemaladze 2023 (PMID 36583780) |
| η_time | 0.005 / год | Оценка |
| η_ROS | 0.01 | Оценка |
| η_rep | 0.001 / деление | Оценка |
| n* (фибробласты) | 50 (Хейфлик) | Hayflick 1965 |
| n* (HSC) | 20–40 | Pan et al. 2023 |
| τ* | 1 год | Конвенция |

### 2.3 Связь с polyGlu

```
S_centriole(t) ∝ polyGlu(t) / polyGlu_max

polyGlu(t) = polyGlu₀ + α_TTLL · [TTLL](t) − α_CCP · [CCP](t)
```

## §3. Уровень #2: Счётчики MCAOA

### 3.1 Кинетика счётчика i

```
Dᵢ(n, t) = Dᵢ₀ + αᵢ · (n / nᵢ*) + βᵢ · (t / τᵢ) + γᵢ · I(другие счётчики)

где:
  Dᵢ ≥ 0                    — накопленное повреждение
  αᵢ ≥ 0                    — интенсивность от делений
  βᵢ ≥ 0                    — интенсивность от времени
  γᵢ ∈ ℝ                    — сила связи (каноническое: 0)
  I = Σⱼ≠ᵢ Γᵢⱼ · Dⱼ/Dⱼ_crit — влияние других счётчиков
```

### 3.2 Агрегатор бремени ткани

```
L_tissue(species, tissue, t) = Σᵢ₌₁⁵ wᵢ · fᵢ(Dᵢ(t))

fᵢ(x) = 1 / (1 + exp(−kᵢ · (x − x_critᵢ)))   — каноническая сигмоида
```

**Условие нормировки:** Σᵢ wᵢ ≈ 1.0. Отклонение > 0.05 → неучтённый счётчик.

### 3.3 Калибровка через Frailty Index

```
L_tissue = FI / 0.7

Пороги (Rockwood 2005, PMID 16129869):
  L < 0.30  → FI < 0.21 → здоровье
  L = 0.30–0.60 → FI 0.21–0.42 → возрастной спад
  L > 0.60  → FI > 0.42 → L_critical: начало болезни
  L > 1.00  → FI > 0.70 → смерть
```

### 3.4 Γ-матрица связей

```
Γ ∈ ℝ^{5×5}

Известные не-нулевые связи:
  Γ₂₃ > 0: АФК ускоряет укорочение теломер (Parrinello 2003)
  Γ₄₃ > 0: NAD⁺/NADH влияет на эпигенетические модификаторы (Schultz & Sinclair 2019)

Гипотетические (канонически = 0):
  Γ₁₄ > 0: эпигенетика → TTLL/CCP (не подтверждено)
  Γ₃₅ > 0: АФК → агрегация белков (правдоподобно, не quantified)
```

## §4. Уровень #3: Ткани + Ze-конфликты + Онтогенез

### 4.1 Тканевая модель

```
Каждая ткань i характеризуется:
  τᵢ        — период самообновления (∞ для постмитотических)
  wᵢⱼ       — вес счётчика j в ткани i
  L_critᵢ  — критический порог бремени
  posᵢ      — 3D-координаты
  vascᵢ    — сосудистая плотность
  innᵢ     — плотность иннервации
```

### 4.2 Ze-конфликт

```
Z_conflict(i, j, t) = |τᵢ · dLᵢ/dt − τⱼ · dLⱼ/dt| · C_ij

где:
  τᵢ · dLᵢ/dt  — «Ze-скорость» старения ткани i
  C_ij         — сила анатомической/функциональной связи
```

**Интерпретация через Ze-Hierarchy:**
- Ткань с большей |τ · dL/dt| = intervention (возмущает среду)
- Ткань с меньшей |τ · dL/dt| = monitoring (адаптируется)
- Z_conflict > Z_crit → разрыв → болезнь

### 4.3 Онтогенез (развитие с зиготы)

```
Стадия 0: Zygote
  S_centriole = S₀
  Все Dᵢ = Dᵢ₀
  L_all_tissues = 0

Стадия 1: Embryogenesis
  Быстрые деления → S_centriole растёт слабо (η_div мало на ранних стадиях?)
  Дифференцировка → установка wᵢⱼ, τᵢ

Стадия 2: Neonate → Adult
  Рост тканей, пик функции
  L_tissue медленно растёт от 0 → 0.15

Стадия 3: Aging
  L_tissue > 0.30
  Z_conflict растёт

Стадия 4: Disease
  L_tissue > 0.60 ИЛИ Z_conflict > Z_crit

Стадия 5: Death
  max(L_tissue) → 1.0 ИЛИ критический орган отказывает
```

### 4.4 Травмы и острые заболевания

```
Dᵢ(t + t_trauma) = Dᵢ(t) + ΔD_trauma

L_tissue пересчитывается с новыми Dᵢ
Если L > L_crit → болезнь
Если травма затронула несколько тканей → рост Z_conflict
```

### 4.5 Микробиом

```
dL_intestinal/dt += f(microbiome_diversity, pathogens, SCFA)

Микробиом влияет на:
  L_intestinal (непосредственно)
  L_immune (через иммунную активацию)
  L_hepatic (через портальную вену)
```

### 4.6 Макробиом (питание + среда)

```
ΔDᵢ через диету:
  Калории → η_ROS (CR замедляет)
  Белки → протеостаз
  Жиры → митохондрии
  INFOGEST-модель: расщепление → всасывание → метаболизм

Токсины/радиация → ΔD_external → все счётчики
```

## §5. Самообучение

### 5.1 Байесовский контур

```
P(θ | data_ARGS) ∝ P(data_ARGS | θ) · P(θ)

θ = {S₀, η_div, η_time, η_ROS, η_rep, αᵢ, βᵢ, kᵢ, x_critᵢ, wᵢⱼ, τᵢ, C_ij, ...}

Prior P(θ): литература + экспертные оценки
Likelihood P(data | θ): ARGUS-LP + INFOGEST + публикации
Posterior P(θ | data): MCMC (PyMC/Stan)
```

### 5.2 Цикл гипотеза-эксперимент

```
1. Симулятор → H (гипотеза, напр.: «ингибирование TTLL6 → −30% polyGlu»)
2. ARGUS-LP → эксперимент (24/7, автономно)
3. Результат R(H) → обратная связь
4. Байесовское обновление P(θ | R)
5. Новая гипотеза H'
```

Для бактерий, архей:
- Уровень #1 отключён
- Аналоги счётчиков: мембранные повреждения, АФК, агрегация, мутации
- «Старение» = накопление повреждений → гибель клетки (не организма)
