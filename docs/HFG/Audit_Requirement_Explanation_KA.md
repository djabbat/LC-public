## 1. "წლიური ფინანსური ანგარიში" — 5 კომპონენტი

ყოველი წლისთვის სტანდარტული პაკეტი:

| # | დოკუმენტი | ქართული | რას გვიჩვენებს |
|---|-----------|---------|----------------|
| 1 | **Balance Sheet** | ბუღალტრული ბალანსი | აქტივები vs. ვალდებულებები წლის ბოლოს |
| 2 | **Income Statement (P&L)** | მოგება-ზარალის ანგარიში | წლის შემოსავლები − ხარჯები |
| 3 | **Cash Flow Statement** | ფულადი ნაკადების ანგარიში | ფაქტობრივი ფულის მოძრაობა |
| 4 | **Statement of Changes in Equity** | კაპიტალის ცვლილების ანგარიში | "net assets" ცვლილება |

**EIC მოითხოვს მინიმუმ #1, #2 და #5.** #3 და #4 — სასურველია.

## 2. "Audited" (აუდიტირებული) — ზუსტი მნიშვნელობა

"Audited" **არ ნიშნავს** უბრალოდ "ბუღალტრის მიერ შედგენილს". ეს მკაცრად ტექნიკური ცნებაა:

### რა არის რეალური აუდიტი

| კრიტერიუმი | მოთხოვნა |
|-----------|----------|
| **ვინ** | გარე, დამოუკიდებელი, **ლიცენზირებული აუდიტორული ფირმა** |
| **ლიცენზია** | საქართველოში — **SARAS** (საქართველოს აუდიტის, აღრიცხვისა და ფინანსური ანგარიშგების ზედამხედველობის სამსახური) |
| **სტანდარტი** | **International Standards on Auditing (ISA)** |
| **დოკუმენტი** | **Independent Auditor's Report** — ცალკე, ხელმოწერილი სერტიფიცირებული აუდიტორის მიერ |
| **დასკვნა** | 4 ვარიანტიდან: **Unqualified / Qualified / Adverse / Disclaimer** |

### რას აკეთებს აუდიტორი

1. ამოწმებს პირველად დოკუმენტაციას (ინვოისები, საბანკო ამონაწერები, ხელშეკრულებები)
2. ადარებს ანგარიშების ბალანსებს ფაქტობრივ დოკუმენტაციას
3. ამოწმებს internal controls
4. ადარებს ფინანსურ ანგარიშებს ქართულ კანონმდებლობასა და IFRS სტანდარტებს
5. წერს ფორმალურ დასკვნას (Independent Auditor's Report)

### რა **არ არის** აუდიტი

- ❌ ბუღალტრის შედგენილი ბალანსი
- ❌ RS.ge-ს საგადასახადო დეკლარაცია
- ❌ Global Fund-ის ფინანსური ანგარიში (sub-recipient report ≠ audit)
- ❌ NCDC-სთვის წარდგენილი ხარჯების ანგარიშგება

## 3. სამი ვარიანტი HFG-სთვის

### ვარიანტი A — **სრული external audit** (იდეალური)

ლიცენზირებული აუდიტორული ფირმა გამოცდის HFG-ის ბოლო 2-3 წელს:

**ფირმები:**
- Grant Thornton Georgia
- BDO Georgia
- EY Georgia
- Mazars Georgia
- Nexia TA Georgia

**რას მიიღებთ:**
- Independent Auditor's Report (თითოეული წლისთვის ცალკე ან კონსოლიდირებული)
- ხელმოწერილი წლიური ფინანსური ანგარიშები
- Management letter (რეკომენდაციებით)

| პარამეტრი | მნიშვნელობა |
|-----------|-------------|
| ღირებულება | ₾4,000-15,000 ერთ წელზე |
| დრო | 2-4 კვირა |
| EIC reviewer ნდობა | 🟢 მაქსიმალური |

### ვარიანტი B — **Global Fund / NCDC sub-recipient assessment** (შუალედური)

თუ HFG-მ მიიღო Global Fund დაფინანსება NCDC-ის მეშვეობით (ცნობილია, რომ 2021-2025 წლებში კი), Global Fund-ი ახდენს ავტომატურ ფინანსურ შემოწმებას.

**რა უნდა მოიძიოთ:**
- Global Fund-ის ოფიციალური **sub-recipient assessment report**
- NCDC-დან **clearance letter**
- ან Global Fund Country Coordinating Mechanism-იდან (CCM Georgia) დადასტურება

| პარამეტრი | მნიშვნელობა |
|-----------|-------------|
| ღირებულება | ₾0 (უკვე გაკეთებულია) |
| დრო | 1-3 დღე (ამონაწერის მიღება) |
| EIC reviewer ნდობა | 🟡 შუალედური (კარგი, მაგრამ არა იდეალური) |

### ვარიანტი C — **შიდა financial statements + accountant's statement** (მინიმუმი)

თუ A-ც და B-ც არ გამოვა დროის გამო:

**მინიმალური პაკეტი:**
- ბუღალტრის შედგენილი ყოველწლიური ბალანსი + P&L (2022, 2023, 2024)
- მეგის (Director) + ბუღალტრის ერთობლივი წერილობითი **დეკლარაცია** რომ ფინანსური ანგარიშები შეესაბამება ქართულ კანონმდებლობას
- RS.ge-დან **ცნობა არასამეწარმეო ფინანსური საქმიანობის შესახებ**

| პარამეტრი | მნიშვნელობა |
|-----------|-------------|
| ღირებულება | ₾0 |
| დრო | 3-5 დღე |
| EIC reviewer ნდობა | 🔴 დაბალი (შესაძლებელია, მაგრამ რისკი) |
