# HAP (HepatoEmotions) — Теория гепатогенного происхождения эмоций

> **MCOA Integration (2026-04-21 fix):** HAP (Hepato-Affective Primacy) is conceptually a candidate MCOA damage counter. Per [MCOA framework](../MCOA/THEORY.md), HAP dynamics could be formalized as `D_HAP(n, t) = D_0 + α·(n/n*) + β·(t/τ) + γ·I(other_counters)` where n = chronic stress episodes, t = time since initial hepatic dysregulation. Per LongevityCommon CORRECTIONS §1.3, `γ_HAP = 0` by default (independence from other counters) pending post-hoc statistical rejection.


**HAP (HepatoEmotions)** — это строгая эволюционно-онтогенетическая гипотеза, утверждающая, что для развития аффективных состояний (эмоций, чувств) у билатеральных животных необходим орган, выполняющий специфические гепатические функции. Проще говоря: **нет печени (или её функционального аналога) — нет эмоций.**

Теория не является мозгоцентричной. Она постулирует, что аффект — это свойство системы «мозг–гепатический орган», где критическую роль играет постоянный поток стероидных регуляторных молекул (жёлчных кислот, экдистероидов), синтезируемых исключительно в этом органе и способных преодолевать гематоэнцефалический барьер (BBB). Эта система формируется в эмбриогенезе, что объясняет, почему взрослые пациенты с ангепатией (отсутствием печени) на искусственной поддержке всё ещё могут испытывать эмоции — необходимые нейронные цепи были заложены в присутствии печени.

**Ключевое утверждение:** 100% корреляция в мета-анализе 56 таксонов: все животные с доказанным аффектом имеют гепатический орган, все животные без такого органа — аффекта не проявляют.

Этот проект структурирован вокруг проверки и уточнения данной гипотезы через комбинацию сравнительной биологии, молекулярных экспериментов и теоретического моделирования. Ниже — основные компоненты проекта.

## Основные разделы проекта

*   **[THEORY.md](THEORY.md) — Формальная теория.** Полное изложение аксиом, определений (аффекта, гепатического органа), механизмов действия (BBB-транспорт, рецепторы FXR/TGR5/EcR) и строгих предсказаний теории. Это канонический документ, основанный на CONCEPT v4.0.
*   **[EVIDENCE.md](EVIDENCE.md) — Доказательная база.** Таблицы с проверенными ссылками на научную литературу (PMID/DOI), данные внутреннего мета-анализа по таксонам, а также честное представление опровергающих данных и неудачных предварительно зарегистрированных тестов.
*   **[OPEN_PROBLEMS.md](OPEN_PROBLEMS.md) — Открытые проблемы и фальсификаторы.** Сердце научной строгости проекта. Содержит чётко сформулированные нерешённые вопросы, каждый с предлагаемым критическим экспериментом, четырьмя возможными исходами и интерпретацией каждого исхода для судьбы теории.
*   **[PARAMETERS.md](PARAMETERS.md) — Количественные параметры.** Сводная таблица всех измеряемых величин: константы связывания рецепторов, коэффициенты проницаемости BBB, концентрации стероидов в плазме. Указаны источник данных, единицы измерения и текущий статус (измерено/предсказано/неизвестно).
*   **[DESIGN.md](DESIGN.md) — Архитектура и дизайн.** Описание структуры репозитория, взаимосвязей между скриптами для анализа данных, моделирования и визуализации, а также API для доступа к ключевым функциям.
*   **[AGENTS.md](AGENTS.md) — Инструкции для ИИ-агентов.** Жёсткие правила для LLM (как Claude), работающих с материалами HAP. Запрещает спекуляции вне доказательной базы, требует указания на ограничения и соблюдения канонов CORRECTIONS_2026-04-22.
*   **[JOURNAL.md](JOURNAL.md) — Научный журнал.** Хронологическая лента всех значимых изменений в гипотезе, дизайне экспериментов и интерпретациях данных с указанием даты, решения и его обоснования.
*   **[ROADMAP.md](ROADMAP.md) — Дорожная карта.** План будущих работ: приоритетные эксперименты (например, условный нокаут гепатоцитов у zebrafish), цели по публикациям, зависимости от других проектов экосистемы LongevityCommon.

## Краткое резюме для нового участника

Если вы впервые сталкиваетесь с HAP:
1.  **Гипотеза:** Эмоции требуют печени (или аналога). Это неврологический, но не исключительно неврологический феномен.
2.  **Механизм:** Гепатический орган секретирует стероиды (жёлчные кислоты/экдистероиды), которые проходят через BBB и действуют на ядерные рецепторы в мозге, модулируя моноаминовые системы (серотонин, дофамин), необходимые для аффекта.
3.  **Доказательства:** Исчерпывающая корреляция у 56 видов. Механизмы подтверждены у позвоночных и насекомых.
4.  **Главная проблема:** Нельзя удалить печень у эмбриона позвоночного, не убив его. Это фундаментальный вызов для прямой проверки необходимости.
5.  **Путь вперёд:** Косвенные проверки через условные нокауты, сравнительный анализ новых видов и уточнение молекулярных механизмов.

Теория HAP представляет собой радикальный сдвиг от антропоцентричных и мозгоцентричных взглядов на эмоции к системному, метаболически встроенному пониманию, имеющему глубокие последствия для эволюционной биологии, нейронаук и психиатрии.
