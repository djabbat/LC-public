# HAP: Таблица количественных параметров

**Цель:** Каталогизировать все измеряемые величины, константы и пороги, критичные для теории HAP, с указанием источника, единиц и статуса достоверности.

**Условные обозначения статуса:**
*   **✅ Измерено:** Значение получено из проверенных экспериментальных данных (литература или наши данные).
*   **🟡 Предсказано/Модель:** Значение оценено по модели, требует экспериментальной проверки.
*   **❓ Неизвестно/Требует измерения:** Критический параметр, значение которого неизвестно или известно с очень низкой точностью.
*   **➖ Неприменимо:** Параметр не релевантен для данного таксона/контекста.
