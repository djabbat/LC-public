We submit our manuscript entitled **«Hepato-Affective Dynamics: A Nonlinear Dynamical Systems Framework for Bifurcation-Driven Affective Transitions»** for consideration in *Biomedical Signal Processing and Control*.

**Why Biomedical Signal Processing and Control?**

This manuscript presents a 6-variable nonlinear dynamical system that models the feedback loops between hepatic steroid output, brain steroid sensitivity, affective circuit integrity, inflammation, HPA axis activity, and metabolic state. The framework bridges **physiological signal processing** with **clinical monitoring** in three concrete ways:

1. **The model is directly interpretable as a physiological signal generator.** Each state variable corresponds to a measurable biomedical signal: hepatic steroid profiles (serum bile acids, cortisol), inflammatory markers (CRP, IL-6), HPA axis output (salivary cortisol rhythms), and metabolic indicators (HOMA-IR). The model provides the first dynamical systems framework linking these signals to affective state transitions.

2. **Bifurcation-driven transitions as clinical decision boundaries.** The saddle-node bifurcation at L_basal ≈ 0 defines a critical threshold — a signal-processing boundary between «affect possible» and «affect impossible» regimes. This maps directly onto clinical monitoring: crossing this boundary signals a transition from compensated to decompensated affective function, with implications for biomarker-based patient stratification.

3. **Validated against large-scale clinical signals.** The model's qualitative predictions align with NAFLD-depression comorbidity patterns (meta-analysis, N = 2,126,593), bile acid dysregulation in major depressive disorder, and Mendelian randomisation evidence. This is not a purely mathematical exercise — it is grounded in measurable, clinically relevant biomedical signals.

**Technical contributions:**

- **6-variable ODE model** coupling hepatic, neural, inflammatory, endocrine, and metabolic dynamics
- **Saddle-node bifurcation** at critical hepatic threshold — the boundary between affective regimes
- **Global sensitivity analysis** (Morris + Sobol) — hepatic threshold parameter θ_L dominates (Sobol ST = 0.75), quantitatively confirmed across all four sensitivity metrics
- **2D parameter scanning** (L_basal × θ_L, 900 simulations) mapping the affect/no-affect phase boundary
- **Stochastic robustness** — white noise CV = 0.78%, Ornstein-Uhlenbeck coloured noise CV = 1.59%, demonstrating structural stability suitable for noisy clinical measurements
- **In silico ablation experiments** reproducing the central biological prediction

**Interdisciplinary relevance to biomedical signal processing:**

The framework generates specific, falsifiable predictions for:
- **FXR/TGR5 agonist monitoring** — hepatic bile acid signalling as a modifiable control parameter
- **Conditional knockout experiments** (zebrafish) — ablation timing relative to critical window
- **Clinical biomarker panels** — liver function tests + affective scales as joint signals for patient monitoring
- **Wearable + lab integration** — continuous cortisol/HRV monitoring combined with periodic hepatic panels

**Novelty:** To our knowledge, this is the first application of bifurcation theory to the phylogenetic emergence of affective states, and the first mathematical model to position the liver — not the brain — as the primary dynamical driver of affect. For the biomedical signal processing community, it offers a new class of physiologically grounded, experimentally falsifiable dynamical models linking hepatic signals to affective outcomes.

**Prior publication:** The underlying theory was published as Tqemaladze, J. (2026) DOI: 10.65649/d76f6c48. An earlier preprint explored an initial formulation (El Fettahi & Tqemaladze, 2025, Preprints.org); the co-author withdrew in June 2026. The present manuscript is a sole-author, substantially revised version with new sensitivity analysis, stochastic robustness testing, 2D parameter scanning, and clinical validation not present in any prior version.
