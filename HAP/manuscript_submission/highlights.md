# Highlights

- The Hepato-Affective Primacy (HAP) theory is formalised as a 6-variable nonlinear dynamical system coupling hepatic steroid output, brain steroid sensitivity, affective circuit integrity, inflammation, HPA axis, and metabolism.

- Global sensitivity analysis (Morris + Sobol) identifies the hepatic threshold parameter θ_L as dominant (Sobol ST = 0.75), quantitatively confirming that hepatic — not neural — parameters drive the system.

- The model is structurally robust under both white Gaussian noise (CV = 0.78%) and Ornstein-Uhlenbeck coloured noise (CV = 1.59%).

- A 2D parameter scan (L_basal × θ_L, 900 simulations) maps the bifurcation boundary between A = 0 (affect impossible) and A > 0 (affect possible).

- Clinical validation against bile acid dysregulation in MDD, NAFLD-depression comorbidity (meta-analysis, N = 2,126,593), and Mendelian randomisation supports the model's qualitative predictions.
