# Highlights

- A 6-variable nonlinear dynamical system coupling hepatic steroid output, brain steroid sensitivity, affective circuit integrity, inflammation, HPA axis, and metabolism is formulated and analysed.

- A saddle-node bifurcation at L_basal ≈ 0 provides the first mathematical expression of the necessary condition for the phylogenetic emergence of affective states.

- Global sensitivity analysis (Morris + Sobol) identifies the hepatic threshold parameter θ_L as dominant (Sobol ST = 0.75), confirming that hepatic — not neural — parameters drive the system.

- The model exhibits structural stability under both white Gaussian noise (CV = 0.78%) and Ornstein-Uhlenbeck coloured noise (CV = 1.59%).

- A 2D parameter scan (L_basal × θ_L, 900 simulations) maps the bifurcation boundary between A = 0 (affect impossible) and A > 0 (affect possible), generating testable predictions for developmental biology.
